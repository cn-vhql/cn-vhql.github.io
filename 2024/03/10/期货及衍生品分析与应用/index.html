期货及衍生品定价

  1. 远期与期货定价

    1. 基本原理：无风险套利，有效的金融市场不存在无风险套利机会；

      远期和期货的无套利定价公式：\(F_0=S_0e^{rT}\)

      如果\(F_0>S_0e^{rT}\)，做空期货，借入\(S_0\)现金买入现货，到期实现套利；

    2. 定价分析：\(e=2.718\)，\(T-t在实际计算中，使用\frac{t}{T}\)；

      ​

  2. 期权定价

    1. 期权平价公式与无套利区间

      1. 上限：标的资产价格是看涨期权上限

      2. 无孳息标的资产的欧式看涨期权下限：\(c \geq max(S_0 - Ke^{-rT} ,0)\)

      3. 无孳息标的资产的欧式看跌期权下限：\(p \geq max(Ke^{-rT} - S_0 ,0)\)

      4. 看涨-看跌期权平价公式（欧式）：在理想情况下，购买一个看涨期权和卖出一个看跌期权（或者相反）与直接购买标的资产并持有到执行价格到期的等价性。

        \(看涨期权价格+执行价格的现值=看跌期权价格+标的资产的价格\)

        \(C + K e^{-rT} = P + S_0\)​

    2. 二叉树模型：可以对欧式，美式，奇异期权，结构化产品进行定价；

      看涨期权的当前价格等于其未来两种可能价值的加权平均值的无风险贴现值。

      \(p\)也被称为风险中性概率，即在此概率下，所有资产的期望收益率均为无风险收益率\(r\)。

      由约翰·考克斯，斯蒂芬·罗斯，马克·鲁宾斯坦等人提出。

    3. \(BSM\)期权定价模型：推导需要几何布朗运动和伊藤引理相关知识。

      \(BSM\)定价模型是多期二叉树期权定价模型在极限条件下收敛而成。

      在\(BSM\)模型中，资产的价格波动率\(\Theta\)经常采用历史数据，隐含波动率来估计。

      6个基本假设：

      1. 标的资产符合几何布朗运动
      2. 标的资产可以被自由买卖，无交易成本，允许卖空；
      3. 期权有效期内，无风险利率r和标的预期收益率是常数，投资者可以以无风险利率无限制借入或者贷出资金；
      4. 标的资产价格是变动的，不存在价格的跳跃；
      5. 标的资产的价格波动率是常数；
      6. 无套利市场；

  3. 短期国债期货标的的资产价格通常是零息债券，没有持有收益， 持有成本包括购买国债所需资金的利息成本，因此其期货定价公式和不支付红利的标的资产定价公式一致。

  4. 期权中常见的希腊字母和波动率

    1. 希腊字母：常见的希腊字母

      1. Delta：\(\Delta = \frac{\Delta C}{\Delta S}\)​

        1. 看涨期权Delta在【0,1】，看跌期权Delta在【-1,0】

        2. 标的价格大于行权价时（虚值期权），标的资产价格上升，看涨期权的delta变大，看跌期权delta趋近0；

          标的价格小于行权价时（实值期权），标的资产价格下跌，看涨期权的delta趋于0，看跌期权delta趋近-1；

          平价看涨期权Delta收敛于0.5，平价看跌期权Delta收敛于-0.5；

        3. Delta对冲，投资者为了规避资产组合的价格变动风险，赚取波动率涨跌带来的收益；

      2. Gamma：\(\Gamma = \frac{\partial^2 C}{\partial S^2}\)

        1. 看涨期权和看跌期权的Gamma均为正值；
        2. 平值期权的Gamma最大。因为Gamma大导致Delta策略需要频繁调整期权关系，所以平值期权交易活跃；
        3. 期权临近到期，平值期权的Gamma值趋近无穷大，实值期权和虚值期权Gamma先增大后减小，随着期权到期收敛于0；
        4. 波动率和\(Gamma\)最大值成反比，波动率增加使行权价格附近的Gamma减小，远离行权价的Gamma增加；

      3. Vaga：\(\nu = \frac{\partial C}{\partial \sigma}\)，\(=\frac {期权价格的变化量}{标的资产波动率的变化量}\)

        1. 期权价格是波动率的递增函数
        2. 平值期权对波动率变化最敏感
        3. 期权到期日的临近，标的资产波动率对期权价格影响变小

      4. Theta：\(\Theta = \frac{\partial C}{\partial T}\)，\(= \frac{期权价格的变化量}{到期时间的变化量}\)；

        1. 通常是负的，期权价值随到期日来临而降低；
        2. 随着期权到期，平值期权Theta单调递减至负无穷大，非平值期权Theta先变小后变大，收敛到0。

      5. Rho：\(\rho = \frac{\partial C}{\partial r}\)，\(=\frac{期权价格的变化量}{市场利率的变化量}\)；

        1. 看涨期权Rho是正的，看跌期权Rho是负的；
        2. 对看涨期权，标的资产价格越高，利率对期权价值影响越大。对看跌期权，标的资产价格越低，利率对期权价值影响越大。
        3. 期权实值程度越高，利率变化对期权价值影响越大。期权虚值程度越高，利率变化对期权价值影响越小；
        4. Rho随着时间到期单调收敛于0。

    2. 波动率：在期权交易的地位——波动率和期权价格正相关

      1. 三组波动率

        1. 历史波动率：持有期权之前的标的价格数据计算出来的；
        2. 已实现波动率：持有期间真实存在的波动率；
        3. 隐含波动率：通过已知的期权价格用BSM期权定价公式倒推出来的波动率，是隐藏在期权价格里面的，反映市场参与者的认知；

      2. Theta的大小与波动率正相关；

    3. 风险中性概率公式：\(p = \frac{e^{rT} - d}{u-d}\)，\(u = \frac{\text{未来最大价格}}{\text{当前价格}}\)，\(d = \frac{\text{未来最低价格}}{\text{当前价格}}\)

    4. 持有成本理论以商品持有（仓储）为中心，分析期货市场的机制，论证期货交易对供求关系产生的积极影响，并逐渐应用到金融期货产品定价上来。完全市场假设下提出的。

      持有成本理论认为，现货价格和期货价格的差由融资利息，仓储费用，持有收益组成

      持有成本的基本原则

      1. 借贷利率（无风险利率）相同且维持不变
      2. 无信用风险，无远期合约违约风险和保证金结算风险
      3. 无税收和交易成本
      4. 基础资产可以无限分割
      5. 基础资产卖空无限制
      6. 期货和现货头寸均持有至期货合约到期日

宏观经济分析与指标

  1. 总需求函数：以产量所表示的总需求和价格水平之间的关系（反向关系），反映的是在某个特定价格水平下经济社会所需要的产品和劳务水平。价格为纵坐标，需求为横坐标。

    总需求函数是指在一个经济体中，所有消费者愿意购买的货物和服务的总量，这个总量与价格和收入有关。简单来说，总需求函数就是告诉我们，当价格降低或消费者收入增加时，消费者会愿意购买更多的货物和服务。

  2. 总供给函数：总产量与一般价格水平的关系（正向关系）。价格为纵坐标，产量为横坐标。

    总供给函数是指在一个经济体中，所有企业和生产者愿意提供的货物和服务的总量，这个总量与价格和生产成本有关。简单来说，总供给函数就是告诉我们，当价格提高或生产成本降低时，企业和生产者会愿意提供更多的货物和服务。

  3. 供需均衡：总供给函数和总需求函数的交点就是市场均衡点。在市场均衡点上，供给的总量等于需求的总量，这意味着市场上所有的货物和服务都被售出了，价格也在最合适的水平上。如果市场上的价格高于均衡价格，那么会出现供过于求的局面，这时企业和生产者会减少供给，消费者会减少购买；反之，如果价格低于均衡价格，就会出现需求过于旺盛的情况，这时企业和生产者会增加供给，而消费者会增加购买。

  4. 根据货币工资的价格和水平进行调整所要求的时间长短，总供给曲线分为极端的古典总供给曲线，凯恩斯总供给曲线，常规的短期总供给曲线。

  5. 哈罗德-多马模型强调资本形成在经济增长中的作用，经济增长率与储蓄率成正比，与资本-产出成反比。该模型忽视劳动投入，技术进步，制度因素的作用。

  6. 凯恩斯经济周期理论从心理因素角度阐述经济周期，是就业通论在经济周期方面的应用。

  7. 新古典增长模型也称为索洛增长模型，外生技术进步增长模型；

  8. 新增长模型也称为内生技术进步增长模型。

  25. 货币供应

    1. M0：流通中现金。包括居民手中的现金和单位备用金，不包含银行库存现金，购买力最强。
    2. M1：狭义货币，活期存款。国家调控的主要对象。M1=M0+活期存款
    3. M2：广义货币，银行存款中的定期存款，储蓄存款和各种短期信用工具。M2=M1+准货币+可转让存单
    4. M3：大额定期存款，金融债券，商业票据。M3=M2+大额定期存款+金融债券+商业票据

  30. 常设借贷便利（SLF）：满足金融机构期限较长的大额流动性需求。期限1~3个月，以高信用评级债券，优质信贷资产抵押发放。

  32. 中央银行货币工具中：

    回购操作调节银根不仅机动灵活，而且较为中性；

    短期流动性调节工具（SLO）是公开市场常规操作的必要补充，一般用来调控银行体系的流动性，在银行体系流动性出现临时性波动时相机使用，并非常规操作使用。

    MLF能够发挥利率走廊上限的作用，有利于维护货币市场利率平稳运行。

统计分析

  1. 描述型统计

    1. 数据集中趋势的度量：平均数，中位数，众数；
    2. 数据离散程度的度量：极差，方差，标准差，离散系数（变异系数）
    3. 数据分布形状的度量：偏度（偏斜方向），峰度（陡峭或平坦程度）
    4. 数据的位次指标：四分位数，四分位差；四分位差反映一组数据中间50%数值的离散情况，数据越大表明部分数据分布越不集中，离散程度越深。
    5. 箱线图

  2. 偏度大于0时，数据右偏分布；偏度小于0时，数据左偏分布；

  3. 当数据呈现明显的偏态分布时，中位数具有较强的代表性。

  4. 抽样分布

    1. 正态分布（高斯分布）应用最广的连续性分布

    2. 卡方分布（X2）：k个独立的标准正态分布变量的平方和服从自由度为k的卡方分布。卡方分布常用于假设检验和置信区间的计算。

      假设你有一个标准的骰子，每次掷骰子得到的点数是随机的，就像标准正态分布变量一样（取值是随机且有一定规律的）。现在你掷k次骰子，把每次掷出的点数减去平均值（骰子点数的平均值是\((1 + 2 + 3 + 4 + 5 + 6)÷6 = 3.5\)），然后将结果平方，最后把这k个平方后的结果加起来。这个总和的分布就是自由度为k的卡方分布。这里的 “自由度k” 就相当于你掷骰子的次数，次数越多，可变化的情况就越多，自由度就越大。

      比如你想检验一批产品的实际合格率是否和理论合格率相符，就可以用卡方分布来进行假设检验。把实际观察到的合格产品数和不合格产品数，与理论上应该有的数量进行比较，计算出一个卡方值，再根据卡方分布来判断实际情况和理论情况之间的差异是否显著。

    3. \(t\)分布（学生分布）：\(t\)分布是统计学中的一种连续概率分布，由奥西普·威廉姆·学生于1908年提出。

      • \(t\)分布的形状取决于自由度，自由度是样本容量减去1。当自由度较大时，t分布接近正态分布；当自由度较小时，\(t\)分布的尾部更厚，更分散。
      • \(t\)分布的位置参数决定了分布的中心位置，也就是分布的平均值。这个参数可以通过样本均值来估计。

      t分布在实际应用中有着广泛的应用，可以用于估计未知总体均值，进行参数估计和假设检验等。

      t分布也是一种连续概率分布。它的形状和自由度有关。当你抽取的样本数量比较小的时候（自由度就小），t分布的曲线和正态分布曲线相比，两边的 “尾巴” 会更厚一些，也就是说出现极端值的概率会比正态分布大。随着你抽取的样本数量越来越多（自由度增大，比如样本容量减去1后的值变大），t分布就会越来越接近正态分布。一般来说，当自由度大于30时，t分布曲线就很接近标准正态分布曲线了。

      应用场景：假如你想知道某所学校学生的平均成绩，但你没办法调查全校所有学生，只能抽取一部分学生作为样本。由于样本量有限，而且总体的标准差（衡量成绩分散程度的一个指标）也不知道，这时候就可以用t分布来估计总体的平均成绩，进行参数估计和假设检验。比如检验这个学校学生的平均成绩是否和全市的平均成绩有显著差异。

    4. \(F\)分布：一种非对称分布，由英国统计学家Ronald.A.Fisher于1924年提出，并以其姓氏的首字母命名。在方差分析，回归性方程的显著性检验中应用广泛；

      \(F\)分布和\(t\)分布之间有一个确定的等量关系。

      产生原理：想象你有两个装满球的盒子，每个盒子里球的编号都有一定的分布规律（就像两个卡方分布）。从第一个盒子里随机拿出一个球，从第二个盒子里也随机拿出一个球，然后用第一个球的编号除以第二个球的编号（或者反过来），把这个比值记录下来。重复很多次这样的操作，这些比值的分布就是F分布。F分布是一种非对称分布，它和两个卡方分布的比值有关，并且有两个自由度，分别对应两个卡方分布的自由度。

      应用场景：在分析不同班级学生成绩的方差（衡量成绩波动大小的指标）是否有显著差异时，就可以用F分布。通过计算不同班级成绩的F值，再根据F分布来判断这些班级成绩的波动情况是否不同。在回归分析中，也会用F分布来检验整个回归方程的显著性，看自变量是否能很好地解释因变量的变化。

  5. 参数估计：假设总体分布已知的前提下，从总体中按照随机原则抽取样本，并利用样本统计量估计总体的参数。

    1. 点估计：利用样本统计量的某个取值直接作为总体参数估计值的方法称为点估计，常用的点估计方法：矩估计法，极大似然法，最小二乘法
    2. 关于一元线性回归模型的参数估计方法常采用的是普通最小二乘法，回归估计值与实际观测值的偏差尽可能小。
    3. 区间估计：在点估计的基础上，构造置信区间对参数进行估计，由点估计值加减抽样误差得到的区间估计，点估计值减去抽样误差构成区间估计的下限，点估计值加上抽样误差构成区间估计的上限。
    4. 样本容量的确定：正态总体均值，正态总体比例
    5. 抽样误差由两部分构成，标准正态分布的\(\alpha/2\)分位数\(Z_{\alpha/2}\)或t分布的\(α/2\)分位数\(t_{\alpha/2}\)，以及点估计的标准误差。

  6. 假设检验

    1. 假设检验：一种基于小概率原理结合反证判断给出的命题在统计意义下是否成立的统计推断方法。
    2. 小概率原理：小概率事件在一次实验中几乎不可能发生，在假定原假设正确的情况下，如果一个小概率事件发生了，则可以认为原假设是错误的，进而拒绝原假设，接受备择假设。

  7. 线性回归分析

    1. 相关性：变量的三类关系（确定性函数关系，相关关系，没有关系）

    2. 分析相关性的方法：观察变量的散点图，求解相关系数的大小来度量变量间的线性相关程度。

    3. 线性回归模型：一元线性回归分析，多元线性回归分析，多重共线性；

    4. 多元线性回归模型满足基本假定之一为：无多重共线性假定，即解释变量之间不存在线性关系。

    5. 检验回归方程是否显著就是要检验回归方程中所有系数是否同时为0，也就是原假设H0：β1=β2=0，备择假设H1：β1，β2至少有一个不为0。如果检验结果表明，至少有一个系数显著不为0，我们就可以说回归方程是显著的，即模型揭示的关系是可信的，并且模型具有解释力。

    6. 变量X的回归系数检验的\(P\)值为0，表明回归系数显著；可决系数\(R^2\)处于【0,1】和可调整的\(R^2\)越接近1，拟合效果越好，越接近0，拟合效果越差。说明所建立的一元线性回归模型整体上对样本的拟合效果较好；对整个模型的F检验的P值为0，说明回归模型线性关系显著。

    7. \(y\)平均值的区间预测和个别值的区间预测具有如下特点：

      1. 对于相同的置信度下，\(y\)的个别值的预测区间宽一些，说明比\(y\)平均值区间预测的误差更大一些。
      2. 样本容量\(n\)越大，预测精度越高，预测越准确；
      3. \(x_f\)距离\(x\)的均值越近，预测精度越高；
      4. \(\sum(x_i-\overline{x})^2\)越大，反映了抽样范围越宽，预测精度越高。

    8. 总离差平方和（TSS）Total：\(TSS = \sum_{i=1}^{n} (y_i - \bar{y})^2\)

      实际意义：比如在研究居民消费支出的问题中，\(y_i\)是各个居民的消费支出，\(\bar{y}\)是平均消费支出，TSS 就反映了居民消费支出之间的总体差异大小。

    9. 残差平方和（RSS）Residual：\(RSS = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2\)

      在上述居民消费支出的例子中，如果建立了一个消费支出与收入等因素的回归模型，\(\hat{y}_i\)是根据模型预测的消费支出，RSS 就表示实际消费支出与预测消费支出之间的误差大小，RSS 越小，说明模型对数据的拟合效果越好。

    10. 回归平方和（ESS）Explained：\(ESS = \sum_{i=1}^{n} (\hat{y}_i - \bar{y})^2\)

      还是以居民消费支出模型为例，ESS 反映了收入等自变量能够解释的消费支出的波动部分，ESS 越大，说明模型能够解释的消费支出的变异越多，模型的解释能力越强。

    11. \(TSS = ESS + RSS\)，\(R^2 = \frac{ESS}{TSS} = 1 - \frac{RSS}{TSS}\)​

      这个等式表明了被解释变量\(y_i\)的总变异可以分解为两部分，一部分是可以由回归模型解释的 ESS，另一部分是模型无法解释的 RSS。

      \(R^2\)被称为判定系数或决定系数。它表示回归平方和占总离差平方和的比例，取值范围在0到1之间。\(R^2\)越接近1，说明 ESS 在 TSS 中所占的比重越大，即模型能够解释的变异部分越多，模型对数据的拟合程度越高；\(R^2\)越接近0，说明模型能够解释的部分越少，模型拟合效果越差。

    12. 当\(R^2=0\) 时，\(ESS=0\)，\(F=0\)

      这意味着回归模型完全不能解释被解释变量\(y_i\)的变异，即所有预测值\(\hat{y}_i\)都等于均值\(\bar{y}\)，模型没有任何解释能力。

  8. 时间序列分析

    1. 时间序列基本概念：

      1. 随机过程：随机变量按照时间的先后排列顺序的集合叫随机过程；随机误差项是随机变量，与解释变量不相关；

      2. 白噪声过程：若一个随机过程的均值为0，方差不变的常数，且序列不存在自相关性，这样的随机过程称为白噪声过程。

      3. 自相关函数和偏自相关函数；

        • 自相关函数（\(ACF\)）。自相关是指时间序列数据与其自身在不同时间点的相关性，它衡量了一个时间序列与其滞后版本之间的线性关系。自相关函数的值范围在-1到1之间，其中-1表示完全负相关，1表示完全正相关，0表示无相关。
        • 偏自相关函数（\(PACF\)）。偏自相关是在控制其他滞后项的影响下，衡量一个时间序列与其滞后版本之间的相关性。偏自相关函数的计算通常基于递归方法，如\(Yule-Walker\)方程。（排除了其他因素后，分析不同时间点该因素是否还具有相关性）

      4. 平稳和非平稳序列；平稳序列就是基本不存在趋势的序列，非平稳序列就是包含趋势、季节性或周期性的序列；

        • 平稳序列。平稳序列是指在时间上统计特性保持不变的序列。均值不随时间变化；方差不随时间变化；自协方差不随时间变化。
        • 若一个随机过程的均值和方差不随时间的改变而改变，且在任何两期之间的协方差值仅依赖于两期的距离或滞后的长度，而不依赖于时间，这样的随机过程称为平稳性随机过程；反之，称为非平稳随机过程。
        • 非平稳序列。非平稳序列则不满足平稳序列的条件之一或多个条件，即其均值、方差或自协方差会随时间的推移而发生变化。这些特征导致序列的统计特性在时间上发生变化，使得非平稳序列更难进行建模和预测。

      5. 单整；指一个非平稳的时间序列的差分序列是平稳的。

        如果一个时间序列经过一次差分变成平稳的，则称原序列是1阶单整的，记为\(I（1）\)。一般地，如果时间序列经过\(d\)次差分后变成平稳序列，则称原序列是\(d\)阶单整序列，记为\(I（d）\)。

        差分是时间序列分析中的一种常见操作，用于减少或消除数据的趋势或季节性。差分通常通过从当前观察值中减去前一个观察值来计算。一次差分就是简单地计算相邻时间点的差异。

      6. 时间序列分析中常用的检验有单位根检验，自回归移动平均（\(ARMA\)），格兰杰因果分析，协整检验和误差修正（\(ECM\)）模型。

  9. 非平稳序列的单位根检验：所检验序列存在单位根，为非平稳序列；所检验序列不存在单位根，为平稳序列；

    想象你有一个时间序列，比如每天的股票价格数据。单位根有点像一个 “特殊的魔法”。如果这个时间序列存在单位根，就好比你在推一个没有摩擦力的小车，一旦你推它一下，它就会一直按照原来的速度跑下去，不会停下来或者回到某个固定的位置。也就是说，今天的股票价格和昨天的价格有很强的联系，而且这种联系不会随着时间减弱，价格会一直变动，没有稳定的趋势，这就是一个非平稳序列。相反，如果不存在单位根，就好像小车有摩擦力，推一下后它慢慢就会停下来，也就是时间序列会趋向于一个稳定的状态，这就是平稳序列。

    1. \(DF\)检验。即分布自由检验，它通过比较样本数据和理论分布来检验数据的假设，通常用于检验样本数据的正态性、方差齐性和独立性等。\(DF\)检验的优点是简单易用，适用于多种分布的检验，但对于非标准分布的数据不太适用，且检验的前提假设较为严格。

    2. \(ADF\)检验。即单位根检验，它通过检验时间序列数据的平稳性来判断数据是否存在单位根，即是否存在非平稳性。\(ADF\)检验适用于时间序列数据的检验，可以判断数据的平稳性，但同样需要假设数据符合一定的时间序列模型，且检验的前提假设较为严格。

      \(DF\)检验实际上是\(ADF\)检验的一种特殊情况，\(ADF\)检验是对\(DF\)检验的扩展和改进。

      \(DF\)检验只适用于存在1阶滞后相关的时间序列，高阶滞后需要用\(ADF\)检验。

  10. \(ARMA\)模型：

    1. 自回归移动平均模型（\(ARMA\)）由因变量对它的滞后值以及随机误差项的现值和滞后值回归得到。

      \(ARMA\)模型的诊断和检验主要包括：（1）检验模型参数的估计值是否具有统计显著性（t检验）；（2）检验残差序列的随机性（Q检验）。

      \(t\)检验：用于检验模型中每个参数估计值的统计显著性，判断自变量（滞后值和误差项的滞后值等）对因变量是否有显著影响。

      \(Q\)检验：主要用于检验残差序列是否为白噪声，即是否具有随机性。如果残差序列是随机的，说明模型已经充分提取了时间序列中的信息，不存在未被解释的相关性；如果残差序列不具有随机性，则表明模型可能存在缺陷，还需要进一步改进。

    2. 分类：移动平均模型（\(MA\)），自回归模型（\(AR\)），自回归移动平均（\(ARMA\)）

  11. 格兰杰因果关系：两个经济变量\(X，Y\)为例，若在变量\(Y\)​历史信息基础上，变量\(X\)​历史信息的加入有利于更好的预测变量\(Y\)​未来的变化，则认为变量\(X\)是变量\(Y\)的格兰杰原因。

  12. 协整分析和误差修正模型：

    1. 一组时间序列的数据是平稳的，那就意味着它的均值和方差保持不变，成为零阶单整序列。

    2. 协整：由多个非平稳性时间序列的某种线性组合是平稳的；

      在经济学中，协整用来检测两个或多个经济时间序列（比如股票价格、利率、汇率等）是否具有长期的均衡关系。即使这些序列短期内会有各自的波动，它们之间可能存在一个长期稳定的比率或关系。这种关系意味着，即使一个序列因为某些短期因素而偏离了这个比率，最终它还是会回归到这个长期均衡状态。

    3. 协整检验常用方法是\(E-G\)两步法：

      协整是指多个非平稳性时间序列的某种线性组合是平稳的。从协整的角度来看，因变量能被自变量的线性组合所解释，说明二者之间具有稳定的均衡关系；因变量不能被自变量解释的部分就构成了一个残差序列，这个残差序列不应该是序列相关的，即残差应该是平稳的。因此，协整检验常用方法是\(E-G\)两步法，即通过检验两个一阶单整时间序列构成的模型的残差序列的平稳性，来判定两个时间序列是否存在协整关系，若残差序列是平稳的，则表明两序列存在协整关系。

      第一步：建立回归模型，第二步：检验残差序列的平稳性​

    4. 传统的经济模型通常表述的是变量之间的一种长期均衡关系，而实际经济数据却是由非均衡过程产生的，因此，建模时需要用数据的动态非均衡过程来逼进经济理论的长期均衡过程，于是产生了误差修正模型；

    5. 误差修正模型的基本思想：若变量之间存在协整关系，则表明这些变量之间存在着长期均衡关系，而这种长期均衡关系是在短期波动中不断调整下得以实现的。

  13. \(GARCH\)模型：

    1. 波动率的基本概念：波动率是金融资产价格的波动程度，是对资产收益率不确定性的衡量，用于反映金融资产的风险水平。

    2. \(BSM\)期权定价公式中的波动率：在\(BSM\)期权定价公式中，波动率的大小会直接影响期权的价格。一般来说，波动率越大，期权价格也会越高，因为投资者需要支付更多的溢价来弥补标的资产价格的不确定性和风险。相反，如果波动率越小，期权价格也会相应降低。

    3. \(ARCH\)模型，全称“自回归条件异方差模型”（Autoregressive Conditional Heteroskedasticity Model），解决了传统的计量经济学对时间序列变量的第二个假设（方差恒定）所引起的问题。在现代高频金融时间序列中，数据经常出现波动性聚集的特点，即长期来看数据是平稳的，但短期来看方差是不稳定的，这种异方差被称为条件异方差。\(ARCH\)模型就是在这种背景下提出的，用于捕捉这种条件异方差性。

      想象你有一个弹簧玩具，当你拉动弹簧一端的时候，它会根据你的力度产生不同程度的伸展或压缩。如果你轻轻地拉，弹簧的移动就会比较小；如果你用力拉，弹簧的移动就会比较大。每次弹簧的伸展或压缩都会影响下一次的移动，这就是ARCH模型的基本思想。

      1. 过去的波动性（过去的弹簧位置）：如果弹簧之前伸展得很大，那么它可能还会继续伸展；如果之前压缩得很小，那么它可能还会继续压缩。这就像是ARCH模型中的自回归部分，它考虑了过去的波动性对当前波动性的影响。
      2. 随机冲击（你施加的力度）：每次你对弹簧施加的力度都是随机的，可能大可能小，这种随机性也会影响弹簧的当前位置。在ARCH模型中，这种随机冲击被称为“条件异方差”，意味着波动性的大小是变化的，并且这种变化是有条件的，依赖于过去的信息。

    4. \(GARCH\)模型，即广义\(ARCH\)模型（Generalized ARCH Model），是\(ARCH\)模型的拓展，由Bollerslev在1986年提出。\(GARCH\)模型允许条件方差依赖于过去误差项的平方和过去条件方差的线性组合，从而更有效地捕捉时间序列数据的波动性集群现象。

      这是一个用来分析和预测金融时间序列数据波动性（风险）的统计模型。为了通俗易懂地解释\(GARCH\)模型，我们可以把它比作一个天气预报系统。

      想象一下，你是一个气象预报员，你的工作是预测未来几天的天气情况。在某些日子里，天气可能非常稳定，没有太大的变化；而在其他日子里，天气可能会突然变得非常动荡，比如出现暴风雨。\(GARCH\)模型就像是你的预报工具，它不仅关注天气的平均温度（这相当于时间序列的均值），而且还特别关注天气的波动性（这相当于时间序列的方差或风险）。

      \(GARCH\)模型的核心思想是，时间序列的波动性（就像天气的不稳定性）会随时间变化，并且这种变化是有规律的。它认为，未来的波动性可以通过过去的波动性和过去的信息（比如过去的均值和波动性）来预测。这就像是，如果今天的天气非常动荡，那么明天的天气可能也会很动荡；如果今天的天气相对平静，那么明天的天气可能也会比较平静。

    5. \(ARCH\)模型和\(GARCH\)模型的区别：

      1. \(ARCH\)模型只考虑了最近的一次波动性冲击。
      2. \(GARCH\)模型考虑了多个时期的波动性冲击。

    原假设通常表示两组数据之间没有显著差异。如果我们的统计检验结果（如t值）超过了临界值，我们就会拒绝原假设，认为两组数据之间存在显著差异。

    \(β_1\)是回归分析中的一个参数，表示自变量对因变量的影响程度。在t检验的上下文中，\(β_1\)显著不为0意味着自变量对因变量有显著影响。

股指期货衍生品应用

  1. 股票阿尔法对冲产品：

    1. 相关概念：

      1. 投资组合的总体收益=贝塔收益（来自与市场系统性风险相匹配的市场收益）+阿尔法收益（来自与投资者水平相关的超额收益）
      2. 阿尔法策略：通过股指期货，期权等衍生品，投资者可以将投资组合的市场收益和超额收益分离开来，在获取超额收益的同时规避系统性风险。

    2. 新兴市场更可能获取到超额收益，但系统风险比成熟市场大。

    3. 股票阿尔法策略（股票中性策略原理）：

      1. 寻找一个具有高额，稳定，积极性收益的投资组合；
      2. 通过卖出相对应的股指期货合约来对冲该投资组合的市场风险（系统性风险），使组合的β值在投资全程始终保持为0，从而获得与市场相关性较低的积极风险收益阿尔法。

    4. 阿尔法策略的优点：

      1. 扩大了投资的可选择范围，提供了广阔的投资领域；
      2. 优化资产配置
      3. 有效构建组合
      4. 节约管理费用

  2. 股票指数增强基金

    1. 指数化投资策略的特点：

      1. 投资策略：主动管理型投资策略和被动型的投资策略
      2. 指数化投资可以降低非系统性风险，持有周期较长，进出市场频率和换手率较低，节约大量的交易成本和管理运营费用；

    2. 指数化投资策略原理：

      1. 合成指数基金：现金储备+股指期货合约；
      2. 合成增强指数基金：积极管理现金资产+股指期货合约；
      3. 几种常见的指数化投资策略：期货加固定收益债券增值策略，期现互转套利策略；

    3. 期货加现金增值策略中期货头寸和现金头寸的比例一般是1:9。

  3. 现金化资产债券

    1. 应用背景：随着机构投资者所投资证券的利息，股息，红利的发放以及新的投资资金的增加，资产组合中的现金会不断增加，现金的增加使机构投资者的投资收益有可能低于基准收益；
    2. 原理：现货资产证券化：投资者先将流入的资金投资于短期政府债券，并购买相当于新流入资金的股指期货，待流入的资金积累到一定程度后，再直接投资于一系列股票，同时将期货头寸平仓；
    3. 特点：现金资产证券化对开放式基金的管理尤其有效，基金经理利用现金证券化机制快速再现金和证券之间进行低成本的转换；

  4. 股指期权策略指数：

    1. 备兑看涨期权策略：又称为抛补式看涨期权，是指持有股票或者股票组合的同时卖出对应的看涨期权，适用于认为股票价格保持平稳波动幅度较小的投资者；
    2. 保护性看跌期权策略：属于期权的套保策略，即在持有股票和股票组合的同时，买入该资产为标的的看跌期权，从而在一定期限内保障资产的价值不会低于某一限定值，而市场价格上升式，组合价值随之上升。
    3. 双限期权策略：双限期权策略即在持有现货头寸的同时，卖出等头寸的看涨期权，买入等头寸的看跌期权，其中看涨期权的行权价格要高于看跌期权价格，成本很低，甚至达到零成本。

  5. 期现互转套利策略是利用期货相对于现货出现一定程度的价差时，期现进行相互转换。这种策略的目的是使总收益除了原有复制指数的收益以外，也可以套取期货被低估的收益。这种策略仍随时持有多头头寸，只是持有的可能是期货，也可能是股票现货。期限互转套利策略在操作上的限制是股票现货头寸的买卖，大量卖出股票组合时，会对股票现货市场产生冲击，会产生冲击成本。

  6. 沪深300指数仅在每年的6月和12月进行指数成分调整，每次调整数量占总数的5%~10%。

  7. 中证500指数行业分布更为分散，前五大行业分别为医药生物，化工，电子，电气设备，有色金属，合计占比40%。

  8. 若公司在股票组合上拥有较好的业绩，相比债券组合而言更易产生阿尔法收益，通过使用股票的阿尔法收益替代债券的阿尔法收益，使得企业的被动型债券资产组合获得了超额收益。实际上，上述策略转移了不同资产的阿尔法收益，被称为“转移阿尔法策略”。

  9. 期货加固定收益债券的增值策略首先保证了可以很好的跟踪指数，当寻找到价格低估的固定收益品种时，可以获取超额收益。

国债期货衍生品分析与应用

  1. 国债相关利差分析：

    1. 期限利差：

      1. \(期限利差=长期利率-短期利率\)
      2. 功能：长短端利差反映债券市场对短端货币政策和长端基本面的预期；
      3. 经济走弱，利差回落；经济走强，利差回升；
      4. 货币宽松，利差回升；货币紧缩，利差回落；

    2. 信用利差（质量利差）：

      1. 定义：相同期限的信用债和利率债到期收益率之间的差值；

        信用利差（Credit Spread）是金融市场中一个重要的概念，它指的是不同信用等级的债券之间的收益率差异。简单来说，信用利差反映了投资者对于持有风险较高债券所要求的额外回报。这个概念通常用于比较两种债券：一种是风险较低的债券（利率债，如政府债券），另一种是风险较高的债券（信用债，如企业债券）。

      2. 信用利差：包含信用风险溢价和流动性风险溢价；

      3. 功能：利用利差预测经济周期；

        在市场对信用风险的担忧增加时，信用利差可能会扩大，反映出投资者要求更高的风险溢价。相反，在市场对信用风险的担忧减少时，信用利差可能会缩小。

    3. 境内外利差：

      1. 定义：境外其他国家的国债到期收益率与我国境内国债到期收益率之间的差值；

        常用的是\(中美国债利差=10年期中国国债收益率-10年期美国国债收益率\)，反映了两国货币环境的差异，包含对未来两国经济基本面，政策环境以及各种不确定的预期因素；

      2. 主要影响因素：中美两国经济增长，通货膨胀和货币政策的差异；中美利差与两国经济增长的差异呈现负相关关系；中美CPI之差与中美长期利差呈现显著正相关关系；

        中美CPI之差与中美长期利差呈现显著正相关关系，意味着中国和美国的消费者价格指数（CPI）之间的差异与两国长期债券收益率之间的差异在变动趋势上呈现出一致性，即当中国CPI高于美国CPI时，中国长期债券收益率也倾向于高于美国长期债券收益率，反之亦然。

  2. 国债套期保值策略

    1. 套保比例的计算：\(\text{套保比例} = \frac{\Delta \text{国债现货价格}}{\Delta \text{国债期货合约价格}}\)；

      $套保比例 = $

      利用利率敏感度对冲

      \(套保数量=\frac{现券价格\times现券久期}{国债期货净价\times国债期货久期}\)

    2. 确定套保比例的方法：久期中性法和基点价值法；

      1. 久期中性法：久期本质上是衡量债券价格相对利率变动的敏感程度，数值上近似等于利率每变动 1% 时债券价格变动的百分比。例如，久期为 5 的债券，当利率上升 1% 时，债券价格大约下降 5%。
      2. 基点价值法：基点价值是衡量债券价格对利率微小变动的具体数值反应，即利率变动 0.01% 时债券价格的变动金额。它直接量化了债券价格对利率变动的敏感度。基点价值法是通过计算债券的基点价值，并确定需要多少这样的债券来对冲利率风险，以达到套保的目的。
        通过计算现货组合和期货合约的基点价值，确定两者之间的比例关系，从而得出需要多少期货合约来对冲现货组合的利率风险。例如，现货组合的基点价值为 1000 元，期货合约的基点价值为 500 元，那么套保比例就是 2，即需要卖出 2 份期货合约来对冲现货组合的利率风险。

    3. 久期管理：

      1. 久期也称持续期。它是以未来时间发生的现金流，按照收益率折现成现值，再用每笔现值乘以距离该笔现金流发生时间点的时间年限，然后进行求和，以这个总和除以债券价格得到的数值就是久期。概括来说，就是债券各期现金流支付所需时间的加权平均值。
      2. 组合久期：\(\text{组合久期} = \frac{(\text{初始组合久期} \times \text{初始组合市值} + \text{期货久期} \times \text{期货市值})}{\text{初始组合市值}}\)。
      3. 运用国债期货进行久期管理：\(\text{国债期货数量} = \frac{(\text{目标久期} - \text{组合久期}) \times \text{组合价值}}{\text{国债期货久期} \times \text{国债期货价格}}\)

    4. 资产配置：

      1. 单一资产配置应用：

        1. 水平移动：水平移动指的是利率曲线上所有期限的收益率点同时上升或下降，且变动的幅度相同。这种变动意味着短期、中期和长期利率都以相同的基点数增加或减少，从而保持了曲线的形状不变，只是整体上移或下移。适合梯式策略。

          • 实际案例：假设市场利率整体上升 100 个基点，1 年期、5 年期、10 年期国债收益率都从原来的 3% 分别上升到 4%。采用梯式策略配置了等额的 1 年期、5 年期、10 年期国债的投资组合，各期限债券价格都会下跌，但由于分散投资，组合的整体损失相对单个期限债券投资要小。

        2. 斜率变动：斜率变动则是指利率曲线的形状发生变化，通常是曲线的陡峭程度发生变化。这可能是由于短期、中期和长期利率的变动幅度不一致造成的。例如，如果短期利率上升幅度大于长期利率，或者长期利率下降幅度大于短期利率，那么利率曲线将变得更加平缓；相反，如果短期利率上升幅度小于长期利率，或者长期利率下降幅度小于短期利率，利率曲线将变得陡峭。

          • 实际案例：经济复苏初期，通常短期利率上升较快，长期利率上升相对缓慢，利率曲线趋于平缓。例如，短期利率从 2% 上升到 3%，长期利率从 3% 上升到 3.5%。此时持有大量短期债券的组合价值下降幅度会大于持有较多长期债券的组合，投资者可根据这一趋势调整资产配置比例

        3. 曲度变动：利率曲线的曲度变化是指利率曲线不仅斜率发生变化，而且曲线的弯曲程度也发生改变。这意味着短期、中期和长期利率之间的关系变得更加复杂，不再仅仅是简单的平行移动或斜率的单调变化。可能出现短期和长期利率变动幅度较大，而中期利率变动幅度较小，使得利率曲线呈现出更为明显的凹凸形状变化；或者相反，中期利率变动幅度相对较大，而短期和长期利率变动相对较小。

          收益率曲线变凸后，中期期限的利率相对向下移动，短期和长期的利率相对向上移动。

        2. 通过对不同国债期货间的操作迅速完成转换，此外，不同国别间的债券转换，也可以通过国债期货来完成；

  3. 国债期货的套利：

    1. 基差交易：利润来源来自：基差变化和持有收益

      1. 相关概念和原理：\(国债期货基差=现货价格-经转换因子调整后的期货价格\)

      2. 任何一只可交割券：基差 \(B = P - (F \times C)\)​

        \(P\)：面值为100元国债的即期价格；\(F\)：面值为100元国债期货的价格；\(C\)：对应可交割国债的转换因子

    2. 买入基差交易：预期基差扩大，买入现货国债，卖出相当于转换因子数量的期货合约，国内较容易操作；

      卖出基差交易更易获利（因为基差收敛），风险更小。

    3. 基差交易的特点：

      1. 基差交易可以使用非CTD券进行交易，交易机会更多，但同时带来更多风险；

      2. 单边基差交易：当基差较大时，买入国债期货替换手中的现货，基差收敛后平仓国债期货买回现货，以实现增强收益；

      3. 国债基差收益与股指期现套利之间的区别

        1. 国债基差交易期货头寸要用到转换因子调整，同样市值的不同债券卖空手数也不一样；
        2. 国债期货为实物交割，最终的CTD券（最便宜可交割债券）可能发生变动，也为国债基差交易带来更多的变数；

      4. 基差影响因素

        1. 持有收益：持有国债现货获得的持有收益；
        2. 交割期权价格
        3. 不同久期的可交割债券基差有着不同的期权属性；

        2. 我国可交易的国债期货品种：2年期（TS）；5年期（TF），10年期（T）

          跨品种交易的实质是交易期限利差；

      6. 基差交易的利润来源于基差变化和持有收益；

  4. 利率期权

    1. 利率上限期权（利率封顶）防范利率上升的风险，期权买方支付权利金，并与期权的卖方达成一个协议，该协议中指定某一种市场参考利率，同时确定一个利率上限水平；也是最早的场外利率期权，1983年推出。主要适用于有浮动利率债务的借款方，担心市场利率上升导致借款成本大幅增加，通过购买利率上限期权来锁定最高利率成本，稳定财务支出。

      比如，企业 A 与银行 B 达成一笔利率上限期权协议，参考利率为 LIBOR，利率上限为 5%。若在结算日 LIBOR 升至 6%，则银行 B 需向企业 A 支付超出 5% 部分的利息差额，从而使企业 A 的实际融资成本锁定在 5% 以内。

    2. 利率下限期权（利率封底）买方在市场参考利率<下限利率，取得卖方支付的低于下限利率的差额；对于一些持有浮动利率资产的投资者，如银行的浮动利率贷款业务，为防止市场利率下降导致收益减少，会购买利率下限期权来保障最低收益水平。

      例如，投资者 C 购买了一份利率下限期权，参考利率为 Shibor，下限利率为 3%。若到期时 Shibor 为 2.5%，则期权卖方要向投资者 C 支付 3% 与 2.5% 之间的利息差额。

    3. 利率双限期权（利率上下限期权，利率领圈期权）确保支付与指定的市场利率介于两个上下限水平之间，即利率双限期权的买入方相当于买入一个利率上限的同时卖出一个利率下限，该利率上限期权和利率下限期权具有相同的到期日，但上限的执行利率比下限的执行利率高。对于那些希望将利率波动范围控制在一定区间内的投资者或企业来说，利率双限期权是一种较为合适的风险管理工具。它在一定程度上降低了购买单一利率上限期权的成本，同时也通过卖出利率下限期权承担了一定的利率下行风险，以换取部分权利金收入。

      比如，投资者 D 买入一份利率双限期权，利率上限为 6%，利率下限为 4%。若市场利率高于 6%，可获得利率上限期权的补偿；若市场利率低于 4%，则需向期权卖方支付差额，而当利率在 4% - 6% 之间时，双方无需进行额外支付。

  9. 利率和国债收益率是同向变动的，和国债价格是反向变动的。

    当利率上升时，国债收益率会上升，国债价格会下降；利率下降时，国债收益率会下降，国债价格会上升。

    利率是资金的价格，当市场利率上升时，新发行的债券会提供更高的收益率来吸引投资者，那么已有的国债相对吸引力下降，其价格就会下降，以使其到期收益率上升到与市场利率相匹配的水平；反之，当利率下降时，国债价格上升，收益率下降。

    当国债收益率曲线向下变平时，远期国债收益率下降幅度大于短期国债收益率下降幅度，也就是远期国债价格上涨幅度大于短期国债价格上涨幅度，此时适合购买远期国债期货，卖出短期国债期货套利。

    国债收益率曲线变平，意思代表长短期国债收益率的差会变小。

  12. \(久期缺口 = \text{资产久期} - \text{负债久期} \times \frac{\text{负债}}{\text{资产}}\)

    久期缺口反映了资产和负债对利率变动的敏感度差异。当久期缺口为正时，资产的久期大于负债的久期乘以负债与资产的比例，意味着资产价值对利率变动的敏感度高于负债，在利率上升时，资产价值下降的幅度会大于负债价值下降的幅度，可能会导致企业或机构的净值下降；当利率下降时，资产价值上升的幅度大于负债，净值会增加。反之，当久期缺口为负时，情况则相反。久期缺口是金融机构进行利率风险管理的重要工具，通过调整资产和负债的久期结构，使久期缺口保持在合理范围内，以降低利率波动对机构净值的影响。

外汇期货

  1. 影响因素

    1. 国际收支

      1. 国际收支统计（国际收支账户）：反映国际收支方式之一，对国际收支量的统计和描述；
      2. 国际收支平衡表：国际货币基金组织制定的统一的平衡表，将国际收支按照复式记账原理和特定账户分类编织出来的一种统计报表，反映了国际收支的结构，总体状况和储备资产的增减变动，为一国制定经济政策，分析影响国际收支平衡的基本经济因素，采取相应的政策提供了依据，包括经常账户，资本和金融账户，以及净误差和遗漏三个一级账户。
      3. 国际收支平衡：国际收支平衡分为静态平衡和动态平衡。静态平衡是指在一个特定时期内国际收支的总量平衡，而动态平衡则更注重在一定经济发展阶段内，国际收支在结构、趋势等方面的平衡，考虑了经济的长期发展和国际收支的调节机制等因素。当国际收支处于平衡状态时，表明该国的对外经济交往相对稳定，经济运行较为健康。
      4. 国际收支与汇率：当国际收支为顺差时，大量外国资金流入本国，在外汇市场上表现为对本国货币的需求增加，而本国货币的供给相对稳定，根据供求关系原理，本国货币的价格（即汇率）就会有上升的动力。相反，当国际收支为逆差时，资金流出大于流入，本国货币供给增加而需求减少，本国货币则面临贬值压力。
      5. 国际收支与经济：经常账户顺差意味着本国出口大于进口，对外贸易处于有利地位，这可以带动相关产业的发展，增加就业机会，促进经济增长。同时，经济增长也会提升本国的生产能力和技术水平，进一步增强本国产品和服务在国际市场上的竞争力，推动对外贸易的发展，二者相互促进、相互影响；
      6. 外汇管理和资本项目，经常项目开放：外汇管理的目的在于保持本国的国际收支平衡，限制资本外流，防止外汇投机，促进本国经济的健康发展；
      7. 次级账户是在经常账户这个一级账户下进行细分的账户，对经常账户所包含的内容进一步分类，以便更详细地记录和分析国际收支的具体项目。包括：收益，经常转移，货物和服务

    2. 通货膨胀

      1. 定义：在信用货币制度下，流通的货币数量超过经济实际需要而引起的货币贬值和物价水平全面持续的增长。
      2. 通货膨胀影响汇率的传导机制：贸易机制，利率传导，预期传导，通货膨胀导致实际购买力下跌，对外比价下跌；

    3. 利差

      1. 一国利率发生变化会影响该国跨境资本的流动，其主要是通过资产价格来影响跨境资本流动；
      2. 利率水平高低直接反映了某国资产的无风险利率，当一国的利率相对较高，该国资产吸引力水平相对增强；

    4. \(财政赤字=财政支出−财政收入\)，被作为一国货币汇率的主要指标；

  2. 外汇远期和掉期交易

    1. 外汇远期交易（期汇交易）：交易双方在成交后并不立即办理交割，而是事先约定币种，汇率，金额，交割时间等交易条件，到期才进行实际交割的外汇交易。

    2. 外汇远期交易特点：

      1. 交割日期在未来某一个时刻，交易地点不固定；
      2. 外汇远期合约是双方经协商后达成的协议；
      3. 外汇远期合约双方当事人都要承担风险；

    3. 外汇掉期交易：可以解决本币或者外币资金短缺的问题

      1. 外汇掉期：交易双方约定在一前一后两个不同的起息日进行方向相反的两次货币互换。

        第一次交换：按约定的汇率用货币A交换货币B，第二次交换：按另一约定的货币B交换货币A；

      2. 起息日：近端起息日：第一次货币交割的日期；远端起息日：第二次货币交割的日期；

      假设一家美国公司在欧洲有业务，预计6个月后需要支付100万欧元的货款。为了避免未来欧元对美元汇率上涨导致成本增加，该公司可以选择进行外汇掉期。首先，公司现在以当前的即期汇率将美元换成100万欧元。然后，在6个月后的约定日期，公司再以约定的远端汇率将这100万欧元换回美元。这样，公司就锁定了汇率，规避了未来汇率变动的风险。

      1. 按照起息日类型不同，掉期交易分为远期对远期，即期对远期，隔夜掉期交易。

        远期对远期：假设一家企业预计在 3 个月后有一笔欧元收入，同时预计 6 个月后需要支付一笔美元货款。为了规避汇率风险，企业可以与银行进行远期对远期的掉期交易，约定 3 个月后将欧元按约定汇率换成美元，6 个月后再将美元换回欧元。

        即期对远期：例如，某金融机构发现市场上即期日元相对便宜，而预期未来 3 个月日元会升值，同时又有美元资金需求。于是该机构可以进行即期对远期的掉期交易，即期用美元买入日元，3 个月后再按照约定的远期汇率用日元卖出换回美元。

      2. 隔夜掉期：比如银行在进行外汇交易时，当天收盘前发现某种货币的头寸过多或过少，为了平衡头寸和规避隔夜汇率风险，就可以通过隔夜掉期交易来进行调整。

  3. 外汇期货和期权的应用策略

    1. 外汇期货策略：

      1. 外汇期货套利交易：交易者根据不同市场，期限或币种间的理论性价差的异常波动，同时买进或卖出两种相关的外汇期货合约，以期价差朝着有利方向变化后将手中合约同时对冲平仓而获利的行为。

        分类：跨期套利，跨市套利，跨品种套利，期现套利；

      2. 外汇跨期套利：牛市套利（买近卖远），熊市套利（卖近买远），蝶式套利（由两个方向相反，共享居中交割月份外汇期货合约的跨期套利组成）；期现套利（正套/反套）

      3. 外汇跨市套利；外汇跨币种套利（两个币种具有一定关联性和相互替代性，交易受到同一因素制约，买入或卖出合约在相同的交割月份）

    2. 外汇期权策略：

      1. 单一期权（期权和现汇的组合）：

        1. 卖出看涨期权与现汇多头的配置——有保障的看涨期权策略
        2. 买入看跌期权与现汇多头的配置——买进有保护的看跌期权策略
        3. 买入看涨期权与现货空头的配置
        4. 卖出看跌期权与现货空头的配置

      2. 期权组合（期权和期权的组合配置）

        1. 垂直价差策略：同时买入同一类型的两个期权

          牛市看涨垂直价差策略：投资者买入一份执行价格较低的看涨期权，同时卖出一份到期日相同但执行价格较高的看涨期权。当标的资产价格上涨时，只要价格高于较低的执行价格，就开始获利，最大收益为较高执行价格与较低执行价格之差减去净权利金（买入期权支付的权利金减去卖出期权收到的权利金），最大损失为净权利金。

          熊市看跌垂直价差策略：买入一份执行价格较高的看跌期权，同时卖出一份到期日相同但执行价格较低的看跌期权。当标的资产价格下跌时，只要价格低于较高的执行价格，就开始获利，最大收益为较高执行价格与较低执行价格之差减去净权利金，最大损失为净权利金。

        2. 日历价差策略：投资者卖出一份距离到期日较近期权的同时，买入一份距离到期日较远的期权策略，两个期权的协定汇率相同；

          由于短期期权的时间价值衰减速度比长期期权快，在日历价差策略中，随着时间推移，卖出的短期期权价值会迅速下降，而买入的长期期权虽然时间价值也在减少，但相对较慢。如果在持有期间标的资产价格波动较小，接近协定汇率，那么短期期权到期时可能变得毫无价值，投资者就可以赚取卖出期权的权利金，同时长期期权仍保留一定价值。

          该策略的风险相对有限，最大损失通常为构建策略时支付的净权利金（买入长期期权支付的权利金减去卖出短期期权收到的权利金）。潜在收益则取决于标的资产价格的波动情况以及期权到期时的时间价值变化。如果标的资产价格在到期时恰好等于协定汇率，投资者可能获得较好的收益。

        3. 对敲策略：价差策略皆为同类期权间的组合运用，对敲策略是将不同期权进行混合组合，即同时买进和卖出看涨期权和看跌期权的策略；

          多头对敲：同时买入一份看涨期权和一份看跌期权，且这两份期权的标的资产、到期日和执行价格都相同。当标的资产价格大幅上涨或大幅下跌时，多头对敲策略都能获利。如果价格上涨超过执行价格加上净权利金（买入看涨期权和看跌期权支付的权利金之和），则看涨期权获利；如果价格下跌低于执行价格减去净权利金，则看跌期权获利。

          空头对敲：同时卖出一份看涨期权和一份看跌期权，且这两份期权的标的资产、到期日和执行价格都相同。空头对敲策略在标的资产价格波动较小，处于执行价格附近时获利，最大收益为卖出期权收到的净权利金。但如果标的资产价格大幅波动，空头对敲策略可能会遭受巨大损失，因为其潜在损失是无限的。

          风险与收益特点：多头对敲的风险是有限的，即支付的净权利金，潜在收益是无限的，只要标的资产价格波动足够大。空头对敲的收益是有限的，为收到的净权利金，风险是无限的，一旦标的资产价格大幅波动，损失可能会非常巨大。

  4. 风险管理中的应用：

    1. 进出口贸易

      1. 出口业务：

        1. 出口业务的汇率风险：应收账款是外币，面临本币升值（外币贬值）风险，以外币计价的应收账款的本币价值下降，相当于资产下跌。
        2. 管理汇率风险的策略：本币多头套期保值，在外汇市场上买入本币远期合约，本币外汇期货合约，或买入本币外汇看涨期权及卖出本币看跌期权等交易；

    2. 境外投融资

      1. 境外投资：公司选择美元兑人民币的外汇期权来对项目不确定性和汇率风险进行规避；
      2. 境外融资：涉及时间跨度较长的融资时，境外融资企业多适合以场外的外汇衍生品作为风险管理的工具；

  6. 国内银行外汇牌价：（把人民币理解为商品）

    1. 投资者以人民币换外币，用现钞卖出价计算；

      当投资者想要用人民币兑换外币时，银行会按照现钞卖出价来进行计算。这是因为银行在卖出外币现钞时，需要考虑到自身获取外币现钞的成本以及相关的运营成本等因素，所以投资者需要以这个价格来购买外币现钞。例如，投资者想去国外旅游，需要用人民币兑换一些美元现金，此时银行就会根据现钞卖出价来收取人民币并提供相应金额的美元现钞。

    2. 投资者以外币换人民币，用现钞买入价计算；

      当投资者持有外币想要换成人民币时，银行会使用现钞买入价。银行在买入外币现钞后，需要对其进行保管、运输等操作，这些都需要成本，所以现钞买入价相对较低。比如，投资者从国外旅游回来，手中还有一些剩余的美元现钞，到银行去兑换成人民币，银行就会按照现钞买入价来计算能兑换的人民币金额。

    3. 企业结汇，需看银行现汇买入价。

      企业在进行国际贸易等业务过程中，收到的外汇通常是通过银行转账等非现钞形式的外汇，即现汇。企业将现汇卖给银行，也就是结汇时，银行会根据现汇买入价来计算支付给企业的人民币金额。因为现汇不存在现钞的保管、运输等额外成本，所以现汇买入价一般高于现钞买入价。例如，一家出口企业收到了国外客户支付的美元货款，以现汇形式存在银行账户中，企业要将这些美元兑换成人民币，银行就会按照现汇买入价来进行操作。

  7. 银行买卖现钞涉及现钞的保管，转移等各费用，所以现钞买入价要低于现汇的买入价，现钞的卖出价有时也高于现汇的卖出价，从而导致现钞买入卖出差价要大于现汇买入卖出差价。

  8. 远期汇率定价公式：\(F = S \cdot \left( \frac{1 + r_b \cdot \frac{d}{t_b}}{1 + r_t \cdot \frac{d}{t_f}} \right)\)，\(S\)是即期汇率，\(D\)是即期日到远期开始日的天数（天）；\(t_b\)是本币的计算惯例天数（360），\(t_f\)是外币的计算惯例天数（360），\(r_b\)是本币的无风险利率；\(r_f\)为外部的无风险利率。

  9. 跨境资本流动依据时间长短可以分为两类：长期（1年以上）资本流动，短期资本流动。

场外衍生品

  1. 远期：

    1. 金融类远期协议

      1. 远期利率协议：交易双方约定以未来既定时刻开始，在既定的时期内按照协议利率借贷一笔数额确定，以特定货币定价的协议；

        主要用于规避利率波动风险，对于那些未来有借款或贷款需求的企业或金融机构来说非常重要。例如，企业预计在 3 个月后有一笔大额贷款需求，担心届时市场利率上升会增加借款成本，就可以通过签订远期利率协议，锁定 3 个月后的借款利率。

      2. 远期汇率协议：远期汇率协议是指按照约定的汇率，交易双方在约定的未来日期买卖约定数量的某种外币的远期协议。

        常用于跨国贸易和投资中。比如，国内企业与国外客户签订了一份 3 个月后交货的出口合同，货款以美元结算。企业担心 3 个月后人民币升值导致收到的美元货款兑换成人民币后减少，就可以通过签订远期汇率协议，锁定 3 个月后的人民币兑美元汇率，确保收益稳定。

    2. 商品类远期协议：在现货贸易合同中，约定在未来特定时间按既定价格交易一批大宗商品，同时还约定了现货交割方式；

    3. 铁矿石掉期：

      1. 德意志银行基于铁矿石普氏指数推出了世界上第一个以现金结算的铁矿石掉期合约，而瑞士信贷，必和必拓以及嘉吉等公司也积极参与了进来。
      2. 铁矿石掉期是以一定规格和品质的铁矿石在特定区域以现货价格为标的，以特定货币价计价，结算和清算的铁矿石价格指数衍生品合约。

    4. 从市场实践来看，商品远期的作用在于价格发现和风险管理；

  2. 互换：互换协议是线性的，结构更简单，更容易定价，更容易受到客户的欢迎。

    1. 利率互换概念：指交易双方约定在未来一定期限内对约定的名义本金按照不同的计息方法定义交换利息的互换协议，通常合约的一方（互换卖方空头）支付的利息是基于浮动利率进行计算的，而另一方（互换买方多头）支付的利息则是基于固定利率的计算，也可以两边的利息计算都基于不同的浮动利率。

      互换市场中的主要金融机构参与者通常处于做市商地位；

      利率互换交易中存在信用风险，所以企业结清现有互换合约头寸时，实际和效果具有差异。

      对互换交易商而言，商品互换是场外衍生品业务特别是大宗商品类业务的主要产品。

    2. 货币互换概念：交易双方在约定期限内交换既定数量的两种货币本金，同时定期交换两种货币利息的互换协议，一般在期初按约定汇率交换两种货币本金，并在期货以相同的汇率，相同金额进行一次本金反向互换。（可能存在一定的违约风险）

      1. 货币互换的基本形式：固定对固定（无利率风险，仅有汇率风险），固定对浮动，浮动对浮动；
      2. 一般进行货币互换的目的就是规避汇率风险；交叉型的货币互换可以管理利率风险和汇率风险；

    3. 股票收益互换：股票收益互换所交换的两个现金流系列，其中一系列挂钩于某个股票的价格或者某个股票价格指数，另一系列则挂钩于某个固定或浮动的利率，也可以挂钩于另外一个股票或股指的价格。有明确的名义本金和结算日期，通常约定无本金交割，即期初和期末的股票或股指都不会发生交割。

      股票收益互换可以实现杠杆交易，股票套期保值或创建结构化产品等目的；

      举个例子，假设投资者持有某公司的股票，并且预计这些股票未来会上涨。投资者可能会与金融机构达成一个股票收益互换协议，根据协议，投资者将支付固定利率，而金融机构则支付等同于股票涨幅的收益。如果股票确实上涨，投资者就能从互换协议中获得额外的收益；如果股票下跌，投资者则只需支付固定利率，而不会承担股票下跌的损失。

    4. 大宗商品互换

      想象一下，你经营一家糖果店，而制作糖果的主要原料是可可豆。可可豆的价格会随着市场供需变化而波动，这对你的生意有很大的影响。如果可可豆价格上涨，你的成本就会增加，利润就会减少；如果价格下跌，你就可以以更低的价格购买，从而节省成本。

      现在，假设你听说未来几个月可可豆的价格可能会上涨。为了保护你的生意不受价格上涨的影响，你找到了一个供应商，你们之间达成了一个特别的协议，这就是大宗商品互换（Commodity Swap）。

      在这个协议中，你同意在未来的一年里，每个月以一个固定的价格从供应商那里购买一定量的可可豆。作为交换，供应商同意如果可可豆的市场价格下跌，他们将以更低的市场价格购买一定量的可可豆。

      这样，无论市场价格如何变化，你都能确保以一个固定的价格获得可可豆，从而避免了价格上涨带来的额外成本。而供应商则通过这个互换获得了价格下跌时的保护。

      总结一下，大宗商品互换就像是你和供应商之间的一个保险政策。通过这个协议，你们双方都能在一定程度上规避价格波动带来的风险。

      1. 商品互换：以大宗商品为标的资产的互换合约；

      2. 交易双方在合约中确定互换标的资产的名义规模数量，同时约定在未来一个或多个特定时刻，双方按照既定价格互相交换现金流；

      3. 买方：支付以固定价格计算的现金流；卖方：支付以浮动价格计算的现金流。买卖双方基于不同的两个浮动价格计算现金流并进行互换的情况。双方通常约定对现金流进行轧差支付，只划拨净额。

      4. 特点：

        1. 互换合约品种更加丰富，满足不同类型投资者的个性化需求；
        2. 基差与价差类互换可以作为基差风险和月间升贴水风险管理的工具；
        3. 商品指数互换解决了中长期套保中的连续性问题，企业不需要投入过多的精力研究月间价差；
        4. 互换合约在交割时间，期限，保证金管理等方面更加灵活。

  3. 场外期权

    1. 路径依赖期权：

      1. 亚式期权：也称为平均价格期权或者均价期权，其价值取决于期权存续期内标的资产在特定时期的平均价格；

        相比传统期权，亚式期权受标的资产短期波动的影响较小，因为它基于一段时间内的平均价格，能在一定程度上平滑价格波动，降低了因价格瞬间大幅波动对期权价值的影响，常被用于对价格波动较为敏感的资产的套期保值或投资策略中。

      2. 远期起点期权：是指期权并非签订就生效，而是在签订之后的未来某个事先约定的时刻生效，且其行权价格等于期权生效时的标的资产价格，即该期权在生效时是平值期权；

        为投资者提供了一种对未来特定时间点资产价格进行锁定或投机的工具。适用于那些预期在未来某个特定时期市场会出现特定变化，希望提前布局但又不想立即承担风险的投资者。

      3. 回望期权：回望期权的价值取决于标的在期权存续期间价格的最高值或者最低值，回望期权在到期时要回望过去标的资产的表现，其收益支付结构与合约期间标的资产价格所达到的最大值或者最小值相关。可进一步细分为浮动行权价格回望期权，固定行权价格回望期权，部分回望期权，美式回望期权。

        给予投资者在期权有效期内根据标的资产价格的极端值来获取收益的机会，相比普通期权，它赋予了投资者更多观察和选择的空间，收益可能更高，但通常价格也比普通期权贵。

      4. 障碍期权：也被称为触发期权，其价值取决于期权存续期内标的资产价格是否触及事先设定的障碍水平或者触发价格，分为敲入障碍期权和敲出障碍期权。

        敲入期权（Knock-In Option） ： 敲入期权在合约开始时并不立即生效。只有当标的资产的价格达到预设的障碍水平时，期权才会“激活”并开始像普通期权那样运作。例如，一个敲入看涨期权可能规定，只有当股票价格下跌至某一特定水平以下时，期权才有效，此时期权持有者可以以约定的执行价格购买股票。

        敲出期权（Knock-Out Option） ： 与敲入期权相反，敲出期权在合约开始时是有效的，但如果标的资产的价格在有效期内达到或超过预设的障碍水平，期权将“失效”。这意味着，一旦触发障碍水平，期权持有者将失去其期权权利。例如，一个敲出看跌期权可能规定，如果股票价格上涨至某一特定水平以上，期权即失效，无论之后股票价格如何变动。

    2. 相关性期权：也称为多资产期权，其收益受到至少两个标的资产的影响；

      1. 交换期权：赋予持有人用一种资产按既定比率交换另外一种资产的权利，是相关性期权的基本形式，众多其他种类的相关性期权都可以转换为交换期权；

        在资产配置和风险管理方面有重要作用。当投资者预测到不同资产之间的相对价值会发生变化时，交换期权可以帮助他们提前布局，实现资产的平滑转换，避免直接买卖资产可能带来的高额成本和市场冲击。同时，它也是众多复杂相关性期权的基础形式，很多其他相关性期权都可以通过一定的方式转换为交换期权来进行分析和定价。

      2. 彩虹期权：用两个或多个标的资产的最大值或最小值作为以一个虚拟资产或者指数充当标的资产的期权；

        由于其基于多个标的资产的最值来确定收益，彩虹期权的风险和收益特征与单一资产期权有很大不同。它既可能因为多个标的资产中出现个别表现特别突出的资产而获得高额收益，也可能因为各资产表现相对均衡，最值没有明显优势而导致收益较低甚至期权价值为零。这种期权适合那些对多资产市场有深入研究，并且敢于承担一定风险，追求高收益的投资者。

      3. 价差期权：以两个价格的差作为标的资产的期权，可以是两个不同资产间价差，也可以是两个不同的资产价格指数或者利率，汇率之间的差。

        投资者可以利用价差期权来对不同资产之间的相对价格变化进行投机或套期保值。例如，当投资者预期两个行业的发展趋势会出现差异，导致它们的股票价格指数价差会扩大时，就可以买入相应的价差看涨期权。如果实际情况符合预期，价差扩大，投资者就可以获得收益。同样，对于那些持有多种资产组合，担心资产之间价差波动带来风险的投资者，也可以通过合理配置价差期权来对冲这种风险。

    3. 场外期权的设计：识别约束条件，明确客户需求，设计具体条款；

    4. 交易成本的高低会通过场外期权的复制对冲策略而反映在期权价格中，市场流动性在较大程度上限制了特定场外期权的交易规模，做空交易相关限制政策会影响到场外期权的类型；

  4. 信用衍生品：

    1. 信用违约互换（Credit Default Swap，简称CDS）：CDS的结构类似于保险合约，其中买方支付一系列定期费用（类似保险费），以换取卖方在特定信用事件发生时（如违约）的赔偿承诺。

      1. 信用事件：债务人无法按时且足额的偿还债务的本金或利息；
      2. 信用衍生品：用以管理信用风险的衍生品合约，当信用事件发生时，信用衍生品的卖方将按照合约要求向买方提供合约规定损失补偿，从而在一定程度上降低或者规避了信用事件给合约买方带来的损失；
      3. 信用违约互换（CDS）：通过CDS，买方向卖方转移参考实体的信用风险。当投资者（买方）认为参考实体的信用情况将变差，购买一定名义金额的CDS，并支付一定的费用，CDS的卖方在获得费用的同时承担了参考实体的信用风险，并在参考实体发生违约时向买方支付相当于名义金额减去回收金额之外的损失。
      4. 信用违约互换（CDS）是一种在交易双方之间建立的信用衍生品合同，信用保护买方向信用保护卖方定期支付费用，如果合同中指定的第三方参考实体在合同有效期内发生信用事件时，信用保护买方将获得来自信用保护卖方的赔付。根据ISDA发布的2009年场外衍生品交易主协议和定义文件，信用衍生品合约中的信用事件可以包括：①债务人破产或倒闭；②在破产保护条件下进行债务重组；③技术违约；④债券的信用利差超过既定水平；⑤信用评级下调超过既定的水平。其中，技术违约是指债务人无法按时足额地偿还债务的利息。

    2. 担保债务凭证（Collateralized Debt Obligation，简称CDO）是一种资产证券化产品，它通过将一组固定收益资产（如公司债券、抵押贷款、贷款等）打包成一个资产池，然后基于这个资产池产生的现金流向投资者发行不同级别的证券。这些证券通常分为几个层次，包括优先级（senior tranches）、中间级（mezzanine tranches）和股权级（equity tranches），不同级别的证券承担不同的风险和收益。

      CDO的利息来源：从发起人处定期获得CDS费用；CDO发行所融资金的投资收益；

      信用资产池的所有者把该资产池打包转让给一个特殊目的机构（SPV），然后该SPV再以该资产池未来的收益为基础向投资者发行票据进行融资，该票据就是所谓的CDO。

      SPV 以资产池产生的现金流为基础，向投资者发行不同级别的证券，包括优先级、中间级和股权级。优先级证券通常具有最高的信用评级，最先获得资产池的现金流分配，在风险发生时也具有最优先的受偿权，所以其收益率相对较低；中间级证券的风险和收益水平介于优先级和股权级之间；股权级证券则承担着最高的风险，通常在资产池的现金流满足了优先级和中间级证券的偿付要求后，才会获得剩余的现金流分配，但如果资产池表现良好，股权级证券也可能获得最高的收益。

    3. 合成担保债务凭证（Synthetic Collateralized Debt Obligation，简称合成CDO）是一种特殊类型的担保债务凭证，其基础资产不是实际的债务资产池，而是信用违约互换（Credit Default Swap，简称CDS）合约。这意味着合成CDO的结构并不依赖于实际拥有债务资产，而是依赖于与这些资产相关的信用保护的衍生品。

      从投资人角度来看，合成CDO和CDO最大的差异在于信用风险转移给SPV的方式，在常规CDO是通过转移信用资产（应收债务）实现的，在合成CDO是通过信用衍生品（CDS）的交易来实现的。

  5. 牛市套利一般用于预测未来近月期货合约涨势大于远月合约，或者近月跌势小于远月合约。

  6. 央行货币互换通常用于国际金融市场的小币种。

  7. 有的CDO按照优先，中间和权益级来分，优先级的CDO信用评级是AAA，风险最低；中间级的评级可能是B或者BB，权益级没有评级，风险最高。

  8. CDS 与保险很类似，投资者买入 CDS 相当于投了保险，当风险事件发生时，投资者就获得赔偿，CDS 的卖方相当于在获得费用的同时承担了参考实体的信用风险，并在参考实体发生违约时向买方支付相当于名义金额，减去回收金额之外的损失。

结构化产品

  1. 权益类结构化产品

    1. 保本结构：

      1. 含义：由固定收益证券与金融衍生品结构组成；

      2. 特征：

        1. 产品所配置的固定收益证券在产品到期时的价值等于产品本身的初值；
        2. 产品所配置的金融衍生品在产品到期时的最大损失不超过所配置的资金，不会给产品带来额外的亏损。也称为完全保本结构，或者100%保本结构；

      3. 构造：构造最简单的保本结构由无风险的零息债券和普通期权多头构建。

      4. 鲨鱼鳍结构产品是保本结构的一种。

        假设一款鲨鱼鳍结构的理财产品，本金为 10 万元，期限为 1 年，挂钩某股票价格。产品设定观察区间为初始股价的 ±10%，如果在这 1 年中，该股票价格始终在这个区间内波动，产品到期时投资者可获得 5% 的年化收益率，即收益为 5000 元；若股票价格超出了这个区间，投资者只能获得 1% 的保底年化收益率，即收益为 1000 元。

      5. 保本型股指联结票据产品设计需要考虑的因素包含客户需求，期权的价格，发行产品的成本，收益。

    2. 收益增强结构：

      1. 定义：产品在获得内嵌的固定收益证券所带来的利息收益之外，还通过期权净空头或者其他结构（价值为负的远期合约）来获得收入，二者叠加使得产品的固定收入部分明显高于市场同期和同信用级别固定收益证券的利息收入。对投资者最大的优势是可以产生较高的现金流入，产品支付的利息收入明显高于同期限，同信用等级固定收益证券的利息。可能会存在无法保本的情况。

      2. 收益构成：

        1. 固定收益：固定收益证券所带来的利息和金融衍生品结构所带来的固定收入；
        2. 浮动收益：即金融衍生品结构所带来的潜在收益或亏损。

      3. 构造：最简单的收益增强结构：持有无风险的固定收益证券的同时卖出一定量的普通欧式期权；

    3. 自动赎回结构：

      1. 定义：产品在特定事件发生时终止，投资者不必等到产品到期即可收到产品的最终收益，而发行者则必须按照既定的收益算法计算产品价值并赎回该产品。

      2. 原理：

        1. 典型的自动赎回条件：当标的资产在既定日期的价格上涨达到或者超过初始价格的100%或者105%；
        2. 当自动赎回条件被触发，产品提前终止，投资者将获得全额的本金并获得一定的利息收入，利率水平通常高于同期同信用级别债券的到期收益率。
        3. 若自动赎回未被触发，则产品到期时投资者仅能获得本金，不包含利息收入；

    4. 双赢结构：金融衍生品由向下敲出的跨式期权多头和下跌敲入的看跌期权空头组合构成，其中向下敲出的障碍水平和向下敲入的障碍水平相同。

      • 向下敲出的跨式期权多头：跨式期权多头是指同时买入具有相同行权价格和到期日的看涨期权和看跌期权。而向下敲出的跨式期权多头意味着当标的资产价格下跌到预先设定的向下敲出障碍水平时，该跨式期权多头就会失效。这种期权组合给予投资者在标的资产价格大幅波动时（无论上涨还是下跌）获利的机会，但当价格下跌到特定水平时，期权的权利会消失。
      • 下跌敲入的看跌期权空头：看跌期权空头是指卖出看跌期权，即承担在期权买方行权时，以行权价格买入标的资产的义务。下跌敲入的看跌期权空头表示只有当标的资产价格下跌到预先设定的向下敲入障碍水平时，该看跌期权空头才会生效。如果价格未下跌到该水平，期权卖方无需承担任何义务。

      设计双赢结构的关键点在于设定敲入/敲出的障碍水平，该水平的设置必须满足：

      \(固定收益证券利息+向下敲入看跌期权价格=向下敲出跨式组合的价格\)

      收益情况：当标的资产价格波动适中，没有触及向下敲出障碍水平时，向下敲出的跨式期权多头可以为投资者带来收益，因为无论价格上涨还是下跌，只要波动幅度足够大，看涨期权或看跌期权就可能产生价值。同时，卖出下跌敲入看跌期权所获得的期权费也增加了收益。如果标的资产价格下跌并触及向下敲入障碍水平，下跌敲入看跌期权空头生效，但由于之前的平衡等式，整体结构仍然可能保持盈利或至少保本。

    5. 雪球类产品既有收益增强，也有自动赎回。

  2. 利率类结构化产品（利率联结结构）：根据结构化产品内嵌的金融衍生品类型不同，利率结构化产品包括内嵌利率远期的结构，或内嵌利率期权的结构。前者包括正向/逆向浮动利率结构，超级浮动利率结构。后者包括利率封顶浮动利率结构以及区间浮动利率结构。

    1. 逆向浮动结构：类似于浮动利率债券。相当于一份内嵌了利率互换和利率上限期权的固定利率债券，\(产品票息=某个固定利率-某个浮动利率\)，每3个月或6个月调整一次。当利率持续下跌，或者正收益曲线时，逆向浮动结构将为投资者带来更高的收益；

    2. 区间浮动结构：

      1. 为普通债券和利率期权的结合，是利息支付联结于某个市场基准利率的浮动利率债券，而且，产品的息票率具有上下浮动界限。
      2. 投资适用场景：金融市场的名义利率比较低，且收益率曲线呈现较为陡峭的正向形态，追求高投资收益的投资者通常会选择；
      3. 相当于普通的浮动利率产品以及利率上限期权空头和利率下限期权多头的组合，在正收益曲线陡峭时能产生比较高的投资收益。
      4. 区间浮动利率结构化产品的投资者大多是货币市场的投资者，投资目的是确保投资的最低收益，当市场利率走势下跌时，这类产品对投资者而言有吸引力。

  3. 汇率类结构化产品

    1. 双货币结构：双货币结构本金是外币，利息是本币，因此利息没有汇率风险，本金有汇率风险，风险敞口较大。

      1. 产品特征及结构：双货币结构化产品与普通的债券类似，定期支付利息并在期末偿还本金，本金和利息均以相同的货币（本币）计价，期末收回的本金以另一种货币（外币）计价。由于以外币作为本金偿还的计价货币，本金收回将承担外汇风险，风险敞口等于投资本金。

      2. 使用场景：双货币结构化产品使得投资者可以从人民币贬值过程中获得相对较高的收益，对于那些持有人民币资产，追求稳健投资但又不愿意提早将人民币卖出的投资者而言，是有较大吸引力的。

      3. 双货币债券可以分解为两种结构：一份以本币计价的固定利率债券，加一份或多份外汇远期合约。

      4. 反向双货币结构化产品：

        1. 反向双货币结构化产品中利息收入是以外币计价并结算的，而本金则是以本币计价并结算的；
        2. 投资者的风险敞口只体现在利息收入上，风险敞口是比较小的。

      5. 发行者的风险对冲：对冲汇率风险，通常是在二级市场中建立起来相应的衍生品头寸

    2. 汇率联动期权结构

      1. 定义：汇率联动期权中的金融衍生品为双货币期权，即产品收益结算所用货币不同于标的资产计价所用货币的期权，包括数量调整期权和复合期权两类；

      2. 原理：

        1. 持有本币资金的投资者投资于外币计价的权益类资产或者大宗商品资产，不仅标的资产的价格变动影响投资者的收益，汇率的变动也是影响收益的重要因素；
        2. 汇率联动期权结构化产品的主要功能是用结构化产品实现上述投资目的；

  4. 金融机构作为发行人在设计权益类结构产品时，需要在客户需求，业务经营，外部市场环境达成平衡。

  5. 牛市价差期权（买低卖高）是指买入一个低行权价的看涨期权，同时卖出一个高行权的看涨期权；

    买入低行权价看涨期权 + 卖出高行权价看涨期权：投资者预期标的资产价格会上涨，所以买入低行权价的看涨期权，以便在价格上涨时能以较低价格买入标的资产从而获利。同时，卖出高行权价的看涨期权，虽然限制了潜在的最大收益，但可以获取期权费收入来降低成本。当标的资产价格上涨到高行权价以上时，买入的看涨期权带来的收益与卖出的看涨期权带来的损失相互抵消，组合的收益达到上限。

    或者买入一个低行权价的看跌期权，同时卖出一个高行权价看跌期权。

    买入低行权价看跌期权 + 卖出高行权价看跌期权：这种情况相对少见，一般是在预期市场虽处于牛市，但可能存在一定回调风险时使用。买入低行权价看跌期权是为了在价格意外下跌时提供一定的保护，而卖出高行权价看跌期权则是为了获取期权费来对冲部分成本。如果标的资产价格上涨，看跌期权价值下降，两个期权的组合会有一定亏损，但亏损有限。

  6. 熊市价差期权（买高卖低）是指买入一个高行权价的看涨期权，同时卖出一个低行权价的看涨期权；

    或者买入一个高行权价的看跌期权，同时卖出一个低行权价的看跌期权。

  7. 比率期权组合

    看涨比率期权组合：投资者买入一定数量的较低执行价格的看涨期权，同时卖出较多数量的较高执行价格的看涨期权。

    看跌比率期权组合：投资者买入一定数量的较高执行价格的看跌期权，同时卖出较多数量的较低执行价格的看跌期权。

  8. 当金融市场的名义利率比较低，而且收益率曲线呈现较为陡峭的正向形态时，追求高投资收益的投资者通常会选择区间浮动利率票据，区间浮动利率票据实际上相当于普通的浮动利率票据以及利率上限期权空头和利率下限期权的多头的组合，这样的结构在正向收益率曲线较陡峭时能产生比较高的投资收益。

衍生品业务风险管理

  1. 风险识别：

    1. 市场风险；信用风险；流动性风险；操作风险；模型风险
    2. 产品层面的风险识别是识别单个衍生品所存在的风险，核心内容是识别出影响产品价值变动的主要风险因子。部门层面的风险识别是在产品层面风险识别的基础上，充分考虑一个交易部门所持有的所有衍生品的风险，核心内容是识别出各类风险因子的相关性和整个部门的衍生品持仓对各个风险因子是否有显著的敞口。公司层面的风险识别是在部门层面风险识别的基础上，更多的关注业务开展流程中出现的一些非市场行情所导致的不确定性，包括授权，流程，合规，结算等活动中的风险。

  2. 风险度量：

    1. 敏感性分析：在保持其它条件不变的情况下，研究单个风险因子的变化对金融产品或资产组合的收益或经济价值产生的可能影响；

      敏感性指标：β系数（股票系统性风险），久期和凸性（利率风险），期权希腊字母（期权风险）等

    2. 情景分析：假设多种风险因子同时发生特定变化不同的情景，金融机构所承担的市场风险，为一种多因素同时作用的综合性影响分析。

    3. 压力测试：一些风险因子发生极端变化情况下的极端情景分析，极端情景下计算金融产品的损失。

    4. 在险价值VaR：在一定概率水平\(α\)%（置信水平）下某金融资产或资产组合的价值在未来特定时期（N天）可能得最大损失；\(α\)越大，风险厌恶程度越高，能够得到把握较大的预测结果。

      1. 计算VaR所需要的信息，时间长度N，置信水平，资产组合未来价值变动的分布特征；
      2. 计算方法：历史模拟法和蒙特卡洛模拟法；使用历史模拟法计算VaR，无需假设条件。
      3. 在置信水平的选择上，一定程度反映了金融机构对于风险承担的态度或偏好。
      4. \(VaR_N = \sqrt{N} \cdot VaR_1\)

  3. 风险对冲：

    1. 场内工具对冲：

      1. 识别现有风险暴露，风险因子，并进行风险度量；
      2. 据现有的风险管理政策来决定需要对冲的风险的数量；
      3. 据风险因子的属性选择合适的场内金融工具，并根据所需要对冲的风险的量来确定所需要建立的场内金融工具的持仓量；
      4. 形成风险对冲计划，执行计划并及时监控和调整；

    2. 场外工具对冲

    3. 创设新产品进行对冲：金融机构为客户提供一揽子金融服务后，将风险打包并分切为多个小型金融产品，并将其销售给众多的投资者，实现风险转移。如资产证券化，信托产品，商业银行理财产品等。

  4. 牵制风险：当平值期权在临近到期时，期权的\(delta\)会由于资产价格的小幅变化而发生较大幅度的变动。

  5. 买入套期保值，基差走弱，盈利；卖出套期保值，基差走强，盈利；

  6. 期货套期保值交易过程中存在的价差，手续费和资金占用费用可能成为影响升贴水的主要因素；

  7. 持有成本假说认为现货价格和期货价格的差由融资利息，仓储费用，持有收益组成。

  10. 在大豆国际贸易中，美国是主要的出口国，而中国是美国大豆的最大客户。中美之间的大豆贸易主要是在中国油厂,美国贸易商和美国豆农三方之间进行。美国豆农的卖方叫价交易，是指农民把大豆卖给国际贸易商之后，并不马上确定大豆的卖出价格，而是对应于CBOT大豆某月期货合约，双方协商确定一个升贴水，美国豆农在一定期限内可以自行选择“点取”规定合约的期货价格，所点取的期货合约价格再加上事先商定的升贴水，即为美国豆农最终得到的大豆销售价格。

  11. 实时监控量化交易和程序的最低要求：（1）量化交易必须连续实时监测，以识别潜在的量化交易风险事件；（2）当量化交易系统自动化交易订单消息行为违反设计参数，网络连接或行情数据线，以及市场条件接近量化交易系统设计的工作的边界时，可适用范围内自动警报；（3）系统或市场条件需要时，量化交易者的监控人员应具备断开量化交易系统，取消现存订单，联络交易所和结算公司的相关人员（如适用 ），寻求信息和取消订单的能力和权限；（4）在交易时间内，量化交易者可以跟踪特定监测工作人员负责的特定量化交易系统的规程。

  13. 被动型算法交易除利用历史数据估计交易模型的关键参数外，不会根据市场的状况主动选择交易的时机和交易数量，而是按照一个既定的交易方针进行交易。其核心是减少滑价（目标价与实际成交均价的差）。被动型算法交易最成熟，应用也最广泛，在市场上使用最多的成交量加权平均价格（VWAP）时间加权平均价格（TWAP）等都是被动型算法交易的例子。
