工作，消费主义和新穷人

发表于 2022-03-13 更新于 2024-03-17

工作，消费主义和新穷人

作者：齐格蒙特·鲍曼

‍

工作的意义：创造工作伦理——生产者社会

工作伦理的起源：吸引穷人到工厂工作，消灭贫困，保证社会安宁；作用是训练和约束人们，向他们灌输新的工厂制度发挥作用所必须的服从性；

  1. 工作伦理本质是一条戒律，即时你看不到任何收益，也应该继续工作，工作是正义的，不工作是罪恶的；

    1. 两个前提：人们想维持生活并获得快乐，需要做一些别人认同的有价值的事情；安于现状且不思进取是可耻的；
    2. 两个假设：所有人都有能力工作，通过工作获得回报和维持生活；只有公认有价值的工作，可以要求薪水回报的，才会被工作伦理认同；

  2. 驱使人们去工作；

    1. 工作伦理改革运动是一场关于控制和服从的战争，是权力斗争，以崇高道德为名，迫使劳动者接受既不高尚，也不符合他们道德标准的生活；

      1. 通过训练出机械化的工人，成为复杂机器中如同没有灵魂的小齿轮一样工作的部分；工作伦理的本质是对自由的摈弃；
      2. 工作伦理把生产活动和人们的需求分离，人们优先考虑能做什么，而不是需要做什么；形成了新时代悖论，为增长而增长；
      3. 工厂体系的诞生瓦解了工匠与他们事业间的亲密关系，后面演变的绩效管理等方式只是试图让工人再度全身心进入工作；

    2. 历史不同流派的思想家，都提到人类的终极目标是征服自然；任何有利这个目标的，都成了合乎道德的，穷人对工厂化的抵制，都成了进步的阻碍，需要被消灭；

      1. 怜悯，同情和关怀被摒弃，它们会拖累人们变革的步伐；
      2. 工作伦理否定伦理改革运动目标群体的生活习惯，喜好和欲望，质疑他们的秉性，认为他们病态的，不受控制的冲动都是新兴工业必须抵御，对抗并消灭的一部分；

  3. 要么工作，要么死亡

    1. 把工作定义为是唯一体面的，道德的，可行的生存方式，让不适合工作的人生活的很痛苦，以减少不工作的人口；
    2. 工作的道德优越性，可以让政府毫无心理负担的向不工作的人以最小化原则援助；
    3. 救济院环境极差，令工厂工作与救济院相比也就显得不那么糟糕了，甚至显得像一种幸福，让工人缺少选择，只能工作；
    4. 引入“仅供生存”概念，把薪水保持在一个够低的水平，工人只能靠它活到第二天的黎明，这样除了继续辛苦工作，他们别无选择；

  4. 制造生产者，将工作穿插人们的一生；

    1. 给所有人工作，让所有人都去工作，通常被认为是治愈因社会暂时的不完美，不成熟导致的一切弊病和麻烦的良方；
    2. 工作类型决定了个人在生活，社会里的合理地位，决定了人们应该匹配的生活标准，标记了人生旅程，是回溯人生成败的重要记录；
    3. 工作成为了人们的主要基准，一旦确定了职业规划，其它方方面面都水到渠成，需要做什么也基本确定了下来；
    4. 工作使人们接受训练，培养服从规范，遵守纪律，形成自己的社会性格，成为顺从现代国家的公民，是现代社会的“圆形监狱”；
    5. 让大部分男性进入工厂工作是作为生成和维持社会秩序的主要手段，以男性为绝对权威建立父权家庭，使秩序生成和服务网络的规训压力传导到工厂无法触及的人群；
    6. 工作伦理要求人们选择一种献身于劳动的生活，但这也意味着没有选择，无法选择和禁止选择；

  5. 从“更好”到“更多”，工作伦理进化至消费主义

    1. 在美国，工人们认为工作是一种手段而不是价值，是获取更多财富，从而更加独立自由的手段，无需任何道德包装；
    2. 美国梦劝导人们，所有工厂遭受的苦难只是暂时的烦恼，屈从老板的虚妄只是成为老板的必经之路；
    3. 通过物质激励而不是道德宣教获得服从纪律的工人，告诉大家工作是赚取更多金钱的手段，人们认为，从工匠变成工人时失去人的尊严，需要通过赚取更多盈余才可以恢复，衡量人们声望和社会地位的是工资的差别，而不再是勤于道德或者惰于工作的罪恶；
    4. 社会生存质量的权力斗争转变为获取更多金钱的斗争，经济收益成为了自治自主的唯一体现，这也把人的动机和对自由的渴望牢牢的，不可逆转的转向消费领域；

‍

从工作伦理到消费美学——消费者社会

现代社会渐进到消费主义社会，大规模生产不再需要大规模的劳动力，曾经作为劳动者储备军的穷人被重铸为有缺陷的消费者，使他们失去了有用的社会功能；

  1. 技术进步在就业萎缩的前提下确保了生产力的增长，工厂员工越来越少，生产者社会进化至消费者社会；

    1. 工作伦理与消费者角色要求背道而驰，消费者应该不固守任何一样东西，没有永久的承诺，没有可以被完全满足的需求，也没有终极欲望；

    2. 理想情况下，消费者得到立刻满足，没有时延，不需要持久的准备和技能学习；如果消费者无法对任何目标保持长期关注和欲望，如果他们没有耐心，焦躁和冲动，尤其是易于激动，又同样容易失去兴趣，“即时满足”就达到了最佳效果，类似短半衰期收益的事；

      1. 如果欲望不用等待就可以充分满足，消费者的消费能力可能会远远超越所有先天，后天的需求和限制，超越消费品物理耐久性的限制，一直以来的消费与满足的关系或许会反转，对满足感的期待或期望会先于需求，并总是大于现存需求，但又不至于大到让人失去欲望，事实上，越是陌生的需求，越具有吸引力，能获取未知生活体验将会带来大量的乐趣；前所未有的新鲜的兴奋感，就是消费者游戏的秘密；
      2. 要想提高消费者的消费能力就不能让他们休息，他们需要不断接受新的诱惑，持续处于永不枯竭的兴奋之中，持续处于怀疑和不满之中，诱使他们转移注意力的诱饵需要肯定这种怀疑，同时提供一个宣泄的出口：“你以为这是全部？好戏”
      3. 在正常运转的消费社会中，消费者会主动寻求被诱惑，每一个新的诱惑都不尽相同，似乎比之前的更诱人，消费者生活在这个轮回中；
      4. 本来是市场选择了消费者，剥夺了他们不受诱惑的自由，但每次来市场，消费者都觉得自己掌控了一切。寻求自我认同，获取社会地位，以他人认为有意义的方式活着，这些都需要日复一日消费；

    3. 在工业阶段，每个人首先是生产者，然后才拥有其它身份；在消费时代，每个人首先是消费者，然后才拥有其它特别身份；

      1. 消费逻辑通过意象性的符码关系让人们进入一种他们欲望深处企盼的消费游戏，这种游戏通过一种“你追我赶”的竞争性购买，自发的生成一体化的无意识的“纪律”，由此，消费主义在阴暗处形成了自己的统治，进入消费社会；

  2. 制造消费者

    1. 不同政治派别都开始鼓吹通过消费主导经济复苏；消费取代了工作，把个人动机，社会整合和系统再生产链接在一起；
    2. 现代终身工作基本被灵活工作取代，寄希望通过工作构建终身身份对大部分人而言不可能；人们的身份变得短暂又灵活，在不同的身份中转换变化；
    3. 目前全球的趋势是通过大幅减少产品和服务的寿命，提供不稳定的工作，将经济导向短周期和不确定的生产；

  3. 由美学评判的工作

    1. 消费是彻头彻尾的一种个人的，孤独的活动；这种活动通过不断满足和唤醒，缓释和激发某种欲望实现；聚集只是凸显了消费行为的私密性，增强其乐趣；
    2. 选择的自由设定了消费者社会的阶层，设定了消费者社会成员绘制生活理想的框架，为他们定义了自我改善的方向，一个人选择的自由度越大，自由行使的权力越多，他在社会阶层中的地位就越高，获得的尊重和自尊就越多，距离美好生活的理想就越近；
    3. 这是信用卡社会，而不是存折社会，信贷的膨胀成了事情朝正确方向发展的可靠信号，受到欢迎；
    4. 消费愈发多样化，对消费者是一种享受，消费者以审美趣味为导向，美学把崇高的体验放在首位；
    5. 对于合格的消费者而言，世界是一个充满了可能性的巨型矩阵，包含着强烈的感受和更深刻的体验，评判世间万物的标准是激发人们的感受和体验的能力——唤起欲望的能力，这是消费过程中最愉悦的阶段，比欲望得到满足更令人陶醉；
    6. 工作也受到了美学的审视，工作的价值取决于产生愉悦体验的能力，不能使人获得内在满足的工作没有价值；

  4. 使命是一种特权

    1. 工作伦理传达出一种工作平等的信息，淡化工作间的差异；但引入美学审视后，工作被强调了区别，放大了差异，某些职业被提升到引人入胜程度，被赋予精致的美学内涵和艺术性体验，其它那些为生计获取报酬的工作则没有任何价值；
    2. 消费时代下，理想的工作必须是有趣的，多样化的，令人兴奋的，具有挑战，存在适度的风险，并不断带来崭新的体验；
    3. 工作的美学价值成为消费者社会有力的层级化因素，把工作本身变成最令人愉悦的体验，具有使命感，娱乐式的工作成为了一种最令人羡慕的特权；

  5. 消费社会的穷人

    1. 消费者社会中，正常生活是作为消费者的生活，他们专注从各种公开的机会中选择，以获取愉悦的感受和生动的体验；一个人如果无法正常生活，就意味着他是失败的消费者，或者是有缺陷的消费者；
    2. 在消费者社会中，作为消费者的不合格是被个人社会降级和内部流放的首要原因，正式这种不合格，无法履行消费者义务的无能，转化为痛苦，他们被抛弃在正常人共同享用的社会盛宴之外；
    3. 有一种境况对失业者特别痛苦，他们似乎永无止境的空闲时间，无法有效利用这些时间，无聊不知如何打发时间；消费主义不允许无聊存在，在消费社会语境下，幸福的生活是绝缘于无聊的，而金钱是进入治疗无聊场所的通行证；
    4. 无聊是消费者社会特有的社会分层因素产生的心理学结果，可能也是社会底层人民最痛苦的感受，也是他们最恼怒，最抗拒的感受；
    5. 消费社会通过大众传媒设定了得体的标准，使未达标的不合格的消费者自卑，制造人为的，主观的不满足感；
    6. 人们满足于自己拥有的东西被消费社会认为是最可怕的威胁，富人被人崇拜因为财富本身，而非工作伦理；富人拥有选择自己生活的能力，随心所欲的改变生活，对他们而言，未来总比过去要精彩，诱人；富人对消费美学的掌握，也让他们高人一等，令他们获得普遍的崇拜；

工作伦理和新穷人

技术进步取代工人，生产力的发展推动失业率的提升，消费社会使每个人都有消费欲望，但并不是所有人都有能力满足，无力满足之人成为新穷人；

  1. 现代无论穷人是否真的不愿意参加生产，都不会影响生产力的增长；人力成本的提升使得资本对失业人口数量的增长无动于衷，甚至还会感到振奋；

  2. 市场上的四类职业：

    1. 引导者：发明家，广告商，推广者和商人；
    2. 教育工作者：积极从事于可雇佣劳动力的再生产，把劳动力塑造成可供购买和消费的商品；
    3. 从事消费市场服务的人：工作目的是激活他人的消费能力；
    4. 常规劳动者：被配置于传统流水线；

    第四类劳动者（常规劳动者）在市场上缺少核心竞争力，很容易被弃用和替换；

  3. 在由工厂，办公室，工作间等构成的世界崇尚灵活性，工作伦理失去的市场，不需要那么多劳动力，在这样的世界存活需要寻找其它的意义，消费主义适时而出，并愿意提供这种意义；

  4. 全球化资本流动使得工人面对资本失去议价权，劳动者的生计取决于老板的仁慈；

  5. 从失业到过剩

    1. 失业者虽然暂时没有工作，但环境一旦好转，他们就有望回到生产者行列。一切也将回到正轨。过剩的人不同，他们是多余的，编外的，不被需要的，如果他们不存在，经济将会更好；
    2. 在如今精简型，小型化，资本和知识密集化产业，劳动力成为提升生产力的制约要素；
    3. 经济增长和就业率提升成了相互矛盾的，技术进步和利润增长是以劳动力的替代和淘汰为标准的，这种背景下，工作伦理显得愈发空洞；
    4. 工作伦理在新时代下的作用是调和社会众生和永恒存在的穷人之间的关系，将穷人贫困的原因归结于他们自身；

  6. 底层阶级的浮现

    1. 底层阶级被认为是那些被排除在阶级体系之外的人，他们没有任何机会，也无法被重新接纳，他们没有容身之所，对社会没有任何有益的贡献，原则上不值得救赎；他们的共同点是没有存在的必要，特征是危险和无用，如果他们消失，所有人会获益；
    2. 底层阶级也是经济逻辑的产物，因社会发展而失去了工作，他们也被认为是消费社会下错误选择者的集合，证明其成员的选择无能；
    3. 工作伦理在消费社会下另一作用，洗涤社会中正常人的双手和良知，帮他们在抛弃同胞致使其永久的失业的罪责中摆脱出来，在对穷人的道德进行谴责的同时，对非穷人的道德实施赦免，通过这种双管齐下的方法，达到双手和良心的洁净；

  7. 贫穷即犯罪

    1. 消费者社会的逻辑是把穷人塑造成未被满足的消费者，消费者的生活方式对于那些低收入者而言越来越触不可及，这也是最令人痛苦的剥夺；
    2. 一个社会如果对自己的生存方式缺乏自信，就会产生围城心态，塑造出一个攻墙的恶魔，一个有形的敌人，人们可以与之不断斗争并有望征服它；
    3. 在消费社会中，消费需求越大，这个社会就越安全，越繁荣；但是，不是所有人都有资源去消费，没有消费能力的人在这个社会中成为了废弃品，只要这个消费游戏不停止，参与者源源不断，废弃品也会源源不断的产生，他们有消费欲望却没有消费能力，当他们屈服于欲望行动时，他们被定义为犯罪；
    4. 在消费时代下，依赖救济的人是犯罪集团天然的温床，他们有着消费意愿却又无力满足；

  2. 美学：应该指愉悦的感受和体验；

农民与帝国

帝国是暴力竞争的产物

  1. 当掠夺性活动收益高于生产性活动，并且可以保持长期稳定时，人类社会就出现了以暴力掠夺为专业的群体，这种分工的社会表现形态是——暴力财政实体；

  2. 暴力财政实体内部有暴力赋敛集团和福利生产集团；

    1. 暴力财政实体内部的各种正式关系，总是由最强伤害能力的拥有者规定或认可的；
    2. 生产能力拥有者，只能通过对暴力主体的得失损益影响或控制，间接决定各种政治经济和社会关系；
    3. 人类社会中的种种权利安排，对这种权利体系的美化和神化，都是暴力保护下某种安排的称谓；

  3. 分封制度呈现为巨石金字塔结构，构成每一块巨石的诸侯大夫领地都是一个相对独立的暴力财政实体。“王”则是居于顶端的最大巨石，控制着权利逐层递减的下层巨石。

  4. 帝国制度是分封制度的进化，是复杂形式的单一暴力财政实体，各种资源集中在顶端，中层则由官僚代理人构成的支架代替了贵族领地的巨石，基层是一盘散沙般的小农。动力在顶端，资源在基层，两端之间的钢管架构就是负责上传下达的各级官僚代理人。

    1. 由秦国发挥到极致的国君集权制度，下层有一个能“尽其民力”和“地力”的自耕农制度，中层有一个由号令赏罚驱动的官僚代理制度，上层有一个控制一切资源的独裁者。凭借这个体制和奖励耕战的政策，秦国将作为主要物质生产者的农民的生产潜力激发出来，将各种人力资源的体力智力和勇气激发出来，并将激发出来的强大力量聚集在国君手中，形成了集中使用的巨大优势。依靠这种优势，秦国在暴力竞争中淘汰列强，创建了中国历史上第一个大一统帝国，礼乐征伐自天子出，立法权与合法暴力集中在皇帝一人手里。

    2. 帝国制度造成了新问题：

      1. 官僚集团瞒上欺下追求代理人利益；
      2. 大一统后统治阶级不受制约的自我膨胀，导致了对被统治者的过度侵害，导致社会基本关系的失衡，即暴力赋敛集团对福利生产集团的过度侵害，也是此后多数帝国灭亡的主要原因；

帝国的均衡与失衡

  1. 统治阶级利用儒家学说描述和论证了帝国内部暴力赋敛集团与福利生产集团的均衡关系；

    1. 自耕农向国家缴纳赋税，国君通过多层次官僚代理网络和基层里甲组织征收赋税，征集兵员，保护帝国及其臣民的安全，维持君君臣臣父父子子的等级秩序，维护国君恩赐给各层臣民的相对权利，并向社会提供福利。
    2. 皇室、贵族及其官僚代理人构成的统治集团，拥有强大的暴力威慑和意识形态劝说能力，因而在双方关系中占据了主导地位。
    3. 农业生产者通过怠工、避税、逃亡以及走投无路时揭竿造反等对策，决定着统治集团在选择不同的土地制度、人身权利、赋税形式和赋税比例时的风险和利益，决定着不同统治方式的成本和收益。
    4. 为了双方长期利益的最大化，牧人必须约束自己，必须付出努力，提供并维护羊群的生长条件。

  2. 在实际生活中，在统治集团与被统治集团的基本关系方面，现实关系总是顽强地偏离儒家的理想和规定，偏离所谓的王道和天道，呈现出日渐堕落的总体趋势，并形成意料之中又摆脱不掉的王朝循环。

    1. 这种偏离均衡点的趋势，发源于官僚代理集团对代理人私利的不懈追求。最高统治者无力约束这种庞大的私下追求，弱小分散的小农阶级又无力抵抗各级权势集团整体或个体的巧取豪夺，于是就有了潜规则体系对儒家宣扬的均衡体系的替代，就有了王朝更替和治乱循环。王朝更替是帝国制度对过度失衡的自我校正机制。

    2. 皇帝希望维持各集团关系的稳定和均衡，保证帝国的长治久安。但这种愿望受到了自身利益和客观能力的双重限制。

      1. 帝王个人利益的最大化与帝国利益的最大化并不完全一致，天对皇帝的约束程度完全取决于皇帝自己愿意接受的程度。
      2. 官僚代理人的个人利益与帝国和部门的利益也远非一致。他们扩张自身特殊利益的形式，体现为税外加税、费外加费、层层加码的一套潜规则体系，一套通过自身的膨胀而架空了统治集团对被统治集团正式承诺的体系，一套并不明说的、由各种利害主体在实际生活的冲突中挤压出来的、勾勒出真实的人际关系和集团关系边界的规则体系。
      3. 帝国官僚体系中的每个活动主体都处于徇私卖法的诱惑和激励格局之中。抑制这种对帝国整体利益有害的激励，从技术上说是困难的，是帝国财政难以承担的；从利害关系上说则是难以指望的，因为徇私卖法的受益者正是各级监督者，受害的平民百姓不仅无权监督，还被阻挡在官方的信息通道之外。

  3. 对国君来说，帝国制度是一个比较容易控制的高效而稳定的制度。

    1. 官吏代理人对他治下的各种资源的支配和控制能力要比分封的诸侯大夫弱得多，短暂得多，与中央对抗的能力也就弱得多。
    2. 新的问题也随之产生了：官吏与其治下民众的利害关系更加短暂脆弱，就好像牧人受雇放牧别人的羊群一样，官吏代理集团比分封的贵族集团更不关心百姓的死活。

  4. 帝国制度下的农户是以一盘散沙的状态存在的。

    1. 农户群体力量微弱，反抗官府压榨的收益很微薄，却面临着杀鸡吓猴的巨大风险。在这种个体反抗风险远大于收益的利害格局之下，只要能凑合活下去，退缩忍让通常是农户的最佳生存策略。因此小农对公正稳定的社会秩序的要求并不苛刻，也不愿意为此付出较高的代价。
    2. 小农经济对贪官污吏的耐受性很强，对帝国官僚制度的适应能力很高，直接结果便是支持了帝国粗放迟钝的统治方式，并且间接抑制对帝国统治方式的耐受性较弱的工商集团的发育。
    3. 小农经济对贪官污吏的承受能力又孕育了导致帝国崩溃的隐患。由于贪官污吏的敲诈勒索比较容易得逞，这个集团的收入水平较高，这就激励更高比例的人口加入贪官污吏的行列，激励压榨程度逐步升级，直到十羊九牧的生矗机出现，引发大规模的暴烈反抗。

  5. 小农经济的分散自给特性决定了农民阶级以两种形态交替存在：和平生产时期的一盘散沙状态与造反时期的集团状态。

    1. 对造反集团状态的敬畏决定了统治集团对儒家规定的正式秩序的敬重，对一盘散沙状态的轻视又导致了鱼肉百姓的潜规则的盛行。
    2. 王朝建立初期，农民造反集团的余威尚存，儒家正式秩序实现的程度偏高。承平日久，潜规则在软弱可欺的零散小农的土壤上成长发育，儒家秩序的基本均衡被逐渐破坏。
    3. 在争夺天下的混战中，最终获得竞争优势的体系，又势必属于驾轻就熟，无须冒险试验的帝国制度。这是战争效率最高，社会认同最广的制度。

  6. 在两千多年的历史上，帝国制度对自身弱点的修补一直没有停止。

    1. 汉朝有削藩，有独尊儒术；隋唐发明了选拔人才的科举制度；宋朝完善了抑制军阀藩镇的代理军官制度；明朝发明了代替相权的内阁制度和提高皇帝个人监控能力厂卫制度；清朝的皇位传贤原则又建立了激励皇子进行素质竞争的新机制。此外还有不断改进但成效不佳的宦官控制制度、后戚控制制度等等。
    2. 在最根本的关系上，在农业生产者与暴力赋敛集团的关系方面，一直不能出现有效的权力制衡。因为最有制衡欲望的受害者，始终是毫无权力的平民百姓。

  7. 在两千多年的历史上，帝国农业的生产方式也经历了许多进步，但这种生产方式对自然资源的利用效率终究有自己的极限。

    1. 在帝国承平日久，人口增加之时，农业依赖的土地资源便日渐紧张，帝国各阶层对土地资源的争夺也日趋激烈。资源竞争导致了严重的两极分化，在竞争中失败的大量人口造反的机会成本很低，帝国崩溃的风险也因此加大了。

  8. 帝国无法吸纳自身创造的过剩人口，在走向崩溃中同时解决人口过剩问题。

    1. 大量人口被逼入了在生存竞争中占据优势地位的官吏集团，加剧了帝国官吏集团膨胀和腐败的原有趋势。制造出更多的流民和造反者，迫使帝国通过更过分的搜刮聚集力量进行镇压，因而更大幅度地偏离儒家对基本秩序的规定，直到每次搜刮所激起的反叛暴力超过新聚集的帝国镇压能力。秩序崩溃，天下大乱，生产性活动大面积停止，人民在战乱和饥荒中大批死亡，土地荒芜，人口锐减，自然资源相对宽裕。此时劫掠的利益低于维护秩序并从事生产的利益，生产开始恢复，各暴力集团小范围的割据局面形成，各暴力集团相互攻杀吞并，最后建立新的帝国。
    2. 这是儒家视野之外的问题，是儒家认知框架无力分析也未曾认真看待的更深层次的危机，帝国秩序的破坏与人口压力增加的同步交织确实容易搅乱对这个问题的分析。。

  9. 帝国制度轮回十余次而基本结构不改，根本的原因，是不能形成冲出农业文明的力量。

    1. 因此既不能解决人口与资源关系的长期性问题，也不能形成构造新型政治均衡的社会力量，从而解决统治集团堕落的周期性问题。
    2. 小农经济的基础不变，诱导或胁迫帝国制度发生根本变迁的利害格局就不能形成，王朝循环就不会终止。

官营工商业与民营工商业

  1. 欧洲形成了冲出农业文明的力量。

    1. 工商业吸纳了大量人口，工业的发展又为农业提供了化肥农药等新投入的生产要素，从而提高了农业产出水平；发达的商业也保证了外来的食品供应，支撑着工商业分工的深化和繁荣。
    2. 分工与专业化交互促进又相互支持，逐步改变了经济结构和劳动力结构。这种文明体系及其市场色彩浓厚的激励机制，改造了人们的生活方式和生育意愿，最后达到了人口增长率和生产方式的承受能力的基本平衡。

  2. 在中国历史上，工商业的形成和发展与官府的关系极其密切。

    1. 在早期阶段，较大规模的工商业分工本身就产生于暴力赋敛集团及其支配的行政权力的需求和指令，这就是周朝的“工商食官”。
    2. 帝国的代理官员是帝国官营工业的直接管理者。但是，无论官营工商业如何发达，它终究是帝国的附庸，是小农经济的剩余产品所支撑的政治军事组织的附庸。它没有自身的生命和发展动力。
    3. 官营工业生产体系还面临着一道难以突破的边界：在自身的分工与专业化的发展中，分工越细，体系越复杂，代理链越长，管理成本就越高，分工带来的利益也就越低。当管理成本高过分工所带来的收益的时候，分工发展的进程就会终止。

  3. 与官营工业不同，民营工业在市场体系中的分工和发展，是一种利益主体不断生成的过程。

    1. 只要分工带来的收益大于交易成本，分工和发展的进程就会无止境地持续下去，永远不存在管理效率和行政需要的限制边界。
    2. 专业化分工导致的专业知识积累和生产力水平的提高是没有止境的，这种提高又进一步提高了对交易成本的支付能力，这个自发演化过程持续下去，便有可能创造出一个资源利用效率更高、力量更强大的新文明。

  4. 帝国制度下的民营工商业也在缓慢发展，统治集团对民间工商业集团的看法和态度也渐趋缓和。但是民间工商业的生存和发展总要受到许多外部限制。

    1. 帝国的权力太大了，有利可图的领域一定会被它霸占和垄断，可是行政管理的效率逐层递减特性又注定了它经营不善。经营不善的恶果又要以成本摊派和无偿征调的方式转嫁给民营工商业集团，转嫁给大大小小的工匠、商人和企业主。官营工商业既侵占了民营工商业的发展空间，又削弱了他们的发展能力。
    2. 为了发展和自卫，民营工商业集团一直在收买和巴结帝国官员甚至皇帝本人，他们被迫在政治领域投入巨大的资金和精力，以行贿送礼、捐钱买官和培养子弟参加科举的方式，为本人和后代争取社会地位和政治保护。
    3. 在这种环境中生存的民营工商业，并不能发展为赋税的主要承担者，他们的盛衰对帝国便不那么重要。不仅如此。民间工商业者的牟利动机，他们收买官员的行为，在官方意识形态看来还是对仁义道德和帝国秩序的破坏性力量。

  5. 作为独大的暴力财政实体，大一统帝国可以自己说了算，根据自身的利益和偏好调整与其他社会集团的关系，并由此决定这些社会集团的生存发展条件。而欧洲那些互相征战的国君却要拿出垄断权甚至征税权与纳税人做交易，把自己的权力让给纳税人选举出来的议会，以此换取他们对加税的支持，降低了暴力赋敛集团为所欲为的能力，同时还为资本抽逃提供了去处。

    1. 欧洲统治者的额外索取不能超过资本抽逃的费用，超出的部分，不得不以权力交换，否则就要破坏自身的财政基础和政治稳定。
    2. 中国的大一统帝国却不怕资本飞走，普天之下莫非王土，民间资本根本没有讨价还价不成时退出的空间，只能被帝国按住脑袋萎缩在角落里。
    3. 在欧洲复杂的格局中，资本有机会宣扬自己的意识形态，也有机会取得局部突破，建立一套资本控制暴力的体制，从而为资本的运行和积累提供更适宜的政治条件，尽可能低的交易费用。这种制度又促进更专业更高效的分工体系的演进，形成强大的工业文明，在世界范围内为自己开辟发展空间。

  6. 所谓资本主义制度，就是这样一个资本控制了暴力和劝说力的制度。

    1. 十九世纪中叶，在欧洲千百个主权国的竞争环境中脱颖而出的胜利者，在比春秋战国更丰富多彩的环境中经过数十代人的试错淘汰脱颖而出的资本主义制度，在开辟自身发展空间的征途中闯入了清帝国的家门。
    2. 帝国的战败标志着一个历史性的转折：暴力赋敛集团直接控制下的暴力，敌不过福利生产集团控制下的暴力。暴力赋敛集团支配一切的社会形态，在生存竞争中丧失了最拿手的优势。

备注

  1. 暴力赋敛集团：社会的统治阶级，控制暴力资源的集团，拥有着强大的伤害能力，是生产关系的直接选择者和维护者，为了自身利益最大化，他们可以占有生产资料和劳动者人身，或者通过赋敛方式替代占有；
  2. 福利生产集团：社会生产者群体；
  3. 帝国制度是在多种暴力财政实体并存、优胜劣汰的环境中逐步建立和完善的组织形式。这套制度调动资源的能力、战争能力和稳定程度接近了当时的生产和技术条件所允许的最大化。这是一套经过上百个国家二十多代人断断续续的积累和摸索，将不同领域和不同层次的制度组合匹配而成的高效率的体系。这套体系高度适应草原地带游牧民族不断入侵的地理环境，高度顺应众多暴力财政实体争霸中原的历史演化路径，同时又密切对应着作为自身基础的小农经济。这套高效率的综合性适应体系，依仗着最适者生存的强大生命力，成为称雄天下两千余年的具有独立生命的历史活动主体，占据了历史舞台的中心；
  4. 生长条件：这种功能，儒家称之为天恩或德政，西方现代政治经济理论称之为“公共产品”，由公民以付税的形式购买。在帝国制度中，它终究是一个暴力财政实体稳定存在所必须提供的。
  5. 儒家学说：将儒家学说确立为官方意识形态，并不意味着改变了统治集团的暴力赋敛性质。增强统治策略中劝导说服和人心控制的成分，减少赤裸裸的暴力威慑的成分，同时对自身的行为有所约束，对物质生产者的意愿有所尊重，这是统治集团降低统治风险的需要，也是暴力统治的合乎逻辑的发展。
  6. 潜规则体系：潜规则体系对正规道德法令体系的偏离，源于从皇帝到官吏的真实行为对正式角色规定的偏离。各级代理人追逐代理人的私利，损害受托维护的公共秩序，这种倾向构成了贯穿帝国两千多年历史的醒目的公私矛盾。只有在官僚代理人的行为与公共利益的要求完全一致的苛刻条件下，帝国的正式规则才是真实的。官吏的私心对仁义道德的取代程度，决定了潜规则体系对正式规则体系的取代程度。

资产负债表

  1. 资产负债表怎么看？
  2. 负债表的警惕点？
  3. 企业真实的负债情况怎么看？
  4. 怎么规避激进型企业？
  5. 资产表要点有哪些？
  6. 怎么通过资产表选股？
  7. 怎么通过应收款变化判断企业的销售策略？
  8. 怎么判断企业是否会出现债务危机？
  9. 怎么判断这个行业里企业的看法？

  1. 优先看负债和所有者权益，知道公司有多少钱是借的，有多少钱是自己的；

  2. 负债表：为何借，向谁借，借多久，利息几何，了解企业借款的原因和迫切程度；

    需特别注意：

    1. 企业借款的代价远高于市场平均借贷成本；
    2. 企业自有货币资金较多，但仍以较高利率借款；

  3. 企业负债率：通常情况下无息负债对企业压力不大，利用有息负债/总资产有息负债/总资产有息负债/总资产计算更可以体现企业负债情况；

    有息负债超过60%的企业比较激进，投资者需注意；

  4. 资产表的要点：钱花哪里去了，有什么变化，同行业什么水平，结构组成怎么样；

    1. 重点指标：

      1. 生产资产/总资产生产资产/总资产生产资产/总资产，判断企业是否是重资产公司；==结构上轻资产公司优于重资产公司；==重资产公司需要不断投入大量资金维护，更新或者升级，产生大量折旧，边际成本高，当缺少新的需求时，重资产投入的产能无法快速退出；
      2. 当年税前总利润/生产资产当年税前总利润/生产资产当年税前总利润/生产资产，数值显著高于社会平均资本回报率，属于轻公司；投资上选择轻公司，赚取超额收益；
      3. 应收款/总资产应收款/总资产应收款/总资产，应收款用资产负债表里带“应收”的科目数字汇总，减去银行承兑汇票金额；计算应收款相当于几个月的营业收入，对比同行业判断公司财务是否稳健，通过观察应收账款和营业收入的增幅对比，判断公司销售策略是否改变；
      4. 货币资金/有息负债货币资金/有息负债货币资金/有息负债，观察企业是否会出现债务危机；
      5. 非主业资产/总资产非主业资产/总资产非主业资产/总资产，判断企业是否把重心放在擅长的领域，如果非主营资产过大，可能企业管理层已经对本行业丧失信心；

  5. 生产类资产：与企业日常生产直接相关的资产科目，包括合并资产负债表从“固定资产”到“长期待摊费用”之间的全部科目；

    1. 固定资产必须折旧，优秀的公司倾向于通过加速折旧以增加前期费用，降低利润以延期纳税；有些固定资产随时间推移实际价值并未下降多少，甚至还是升值，这也是财报里隐藏的公司价值，可能成为股价的催化剂；
    2. 需要警惕企业利用主观低估固定资产可变现净值，计提大额减值，降低该固定资产的账面价值，将损失记录在当年，第二年出售资产的方式实现报表利润的同比增长或扭亏为盈；
    3. 在建工程不需要计提折旧，如果一家企业在建工程数目巨大，持续多年不转入固定资产科目，需要对此报以怀疑；
    4. 研发支出可以计入资产，也可以计入费用；一般情况下，利润高的企业通过把研发支出计入费用，可以减少税收支出，利润数据更可靠；经营困难或者创业阶段公司，通过把费用资本化，增加当期资产，让报表好看点；
    5. 生物资产由于难以审计，是造假的高发区；税收优惠多，产品不易核查，产品差异大，不易找到价格参照物，产品交货量不好监控；警惕农林牧渔行业；
    6. 商誉减值可以作为费用从当年利润扣除，即便未来资产获利能力提升也不可转回，商誉减值容易成为造假高发区；商誉不用摊销，买家们会通过低估可辨认资产的价值，做高商誉价值，降低后期折旧费用，让财报更好看；投资者应该将商誉从净资产去除后计算负债率评估企业安全边际；
    7. 长期待摊费用是一笔已经花掉的费用，但在表中被假装为资产，可以通过这样做减少当期费用，增加当期利润，数字越大，企业资产质量越大，需要警惕；

  6. 经营类资产：与企业销售商品或提供劳务相关的资产；

    1. 货币资金注意点：货币资金需要和短期债务和经营需要相匹配；特别注意：

      1. 存贷双高，货币资金充裕，却借了很多有息甚至高息负债，可能是账目货币资金虚构，挪用或者被担保冻结的征兆；
      2. 定期存款和其它货币资金很多，却缺少短期可动用流动资产，可能资金已经被冻结；
      3. 其它货币资金数额巨大，缺少合理解释，需要结合是否存在公司对外大额支付投资预付款，意向金等，导致货币资金被股东或关联人占用；
      4. 货币资金的利息收入，显著低于常见理财或存款利率，表明货币只是在银行待了短暂的时间，审查后又转移；

    2. 通过应收票据的组成，当应收票据占比营业收入低，且是银票，说明企业地位优势，当企业手中有大量商票，说明企业的营业收入靠宽松的销售策略换来的；

    3. 如果在一家非金融企业报表里，存在与正常经营规模不匹配的应收票据，那就是需要警惕的信号，很可能这家公司不务正业，在票据市场当炒家；

    4. 当一家企业应收账款大幅上涨，增长幅度超过同期收入增长幅度，应收账款周转率显著低于行业平均水平，或呈现下降趋势时，预示着两种可能，公司临时放宽销售策略，加大赊销力度，或公司提前确认收入甚至虚构收入；

    5. 当一家企业长期需要存在大量的预收款项，说明企业产品优势地位，在商业链地位高，值得选择；

    6. 预付款是企业财报造假的常用资金出口，当企业预付工程款，预付专利费或非技术的采购款大幅上涨时，需要注意公司是否是通过预付款流出资金，然后购买公司的商品和服务名义回流，以虚增营业收入和净利润；

    7. 如果企业账上没有多少钱，但是有着大量的应收账款和其它应收款，这个企业肯定有问题；

    8. ==优秀的企业的“其他XX”科目涉及金额较小，甚至为零，==当投资者见到“其他XX”较大的企业，可以认定该企业经营不够规范；

    9. 企业利用存货造假的手段：

      1. 存货应该计提跌价准备却不做计提，需要结合行业来看，尤其高科技或者生鲜产品需要警惕；
      2. 通过虚构采购商品或服务将资金流出，再将资金购买本公司产品和服务回流回企业虚增利润；这种造假往往伴随着存货增长大幅超过同期营业成本的增长，且期末存货账目价值与当期营业成本相比，显著高于同行业水平；
      3. 通过加大生产，降低单位产品的成本，从而欺诈性的提升毛利率，虚增成本；

利润表

  1. 怎么通过利润表选股？
  2. 怎么看待营业收入的变化？
  3. 怎么通过毛利率评估企业？
  4. 怎么通过费用率找到优秀企业？
  5. 费用项需要的警惕点？
  6. 怎么通过营业利润率选股？
  7. 怎么看待营业利润率的变化？
  8. 如何通过净利润和现金流量表判断企业的质量？

  1. 营业收入：投资者首选企业是营业收入过去，现在，可见的未来增长的企业；企业收入正常增长的三种途径：

    1. 潜在需求的增长：可靠性最强，行业向好，欣欣向荣；同时要结合行业对手看，判断企业是否是行业强者；
    2. 市场份额的扩大：抢占竞争对手地盘，会遭遇反击，需要评估竞争对手的反击力度和增长的可持续性；
    3. 产品价格的提升：客户需付出更大代价，可能迫使客户减少消费或者选择可替代品，需要评估产品的用户价值；

  2. 毛利率：企业利润的源头，是营业收入−营业成本营业收入-营业成本营业收入−营业成本，高毛利率意味着企业的产品竞争优势更强，替代代价高；投资者尽量选择高毛利率的企业，通常而言，毛利率能保持在40%的企业，通常具有某种持续竞争优势； 需要警惕通过存货造假虚增毛利润；

  3. 费用率：指销售费用，管理费用和财务费用占营收比例。需要警惕费用率过高和剧烈变化的公司；费用能够控制在毛利润30%以内的企业属于优秀企业，高于70%的企业关注价值不大；

    1. 销售费用高说明企业产品吸引力弱，需要营销才能完成销售，当企业扩张时，需要相关市场营销配套，所需成本和对管理能力要求高；
    2. 管理费用正常增幅等于或小于营收增幅，出现管理费用大于增幅时，需要查出明细，尤其注意连续几年出现小额净利润的公司，利用管理费用调控报表；
    3. 研发费用需要结合行业看，保守的投资者，倾向于放弃研发费用高的企业，风险较高；

  4. 营业利润率：等于营业利润/营业收入营业利润/营业收入营业利润/营业收入，体现企业盈利能力，越大越好；需要考虑营业利润率增长的原因，是产品涨价，成本下降还是费用降低带来的。

    1. 产品涨价是否会导致市场份额下降；
    2. 成本降低是全行业一起降了还是公司独降，原因是什么；
    3. 费用控制有没有伤及公司战斗力，是一次性的还是永久的，对手是否会跟进；

  5. 最重要的一步，确定净利润是否变成现金回到了企业的账户，用现金流量表里的“经营现金流净额”除以利润表的“净利润”，比值越大越好，持续大于一是优秀企业的重要特征，代表企业净利润大部分变成真实现金回到公司；

  6. 利润表也称作“损益表”，由于权责发生制，也是最容易造假的表；

现金流量表

  1. 如何通过现金流量表造假？
  2. 现金流量表里的异常信息？
  3. 通过现金流量表选取优质股票？

  1. 在现金流量表造假的三种情况：

    1. 控股股东或者管理层期间挪用上市公司现金，报表日前归还；
    2. 借用一次性行为美化当期现金流；
    3. 将投资活动和筹资活动产生的现金流伪装成经营活动现金流入，或者将经营活动现金流出化装成投资和筹资活动现金流出；

  2. 投资者需要关注合并现金流量表，和其中表达出来的异常信息；

    1. 经营活动现金流量中的异常现象：

      1. 持续的经营活动现金流净额为负；
      2. 经营活动现金流净额为正，但主要是来源于应付账款和应付票据的增加，应付账款和应付票据的大量增加，可能意味着上市公司大量拖欠供应商货款，是资金链断裂前的一种异常征兆；
      3. 经营活动现金流净额远低于净利润，需要警惕企业利润的真实性；

    2. 投资活动中现金流量中的异常现象：

      1. 购买固定资产，无形资产支出，持续高于经营活动现金流量净额，说明企业持续借钱维持投资行为，可能是某种特殊原因造成企业必须流出现金，通常也是企业营收和利润造假的源头；
      2. 投资活动现金流入里，有大量现金是因出售固定资产或其它长期资产获得的，可能是企业经营能力衰败的标志；

    3. 筹资活动现金流量中的异常现象

      1. 企业明明需要大量资金时，却出现“取得借款收到的现金远小于归还借款支付的现金”，透露银行对企业贷款的意愿降低；
      2. 企业为筹资支付了显然高于正常水平的利息和中间费用，体现在“分配股利、利润或偿付利息支付的现金”和“支付其它与筹资活动有关的现金”两个科目明细里，意味着企业遇到了必须江湖救急的生存危机；

  3. 通过现金流量表寻求优质企业：

    1. 经营活动产生的现金流量净额>净利润>0；
    2. 销售商品、提供劳务收到的现金>=营业收入；
    3. 投资活动现金流出>投资活动现金流入，且用于扩张，并非用于维持是原有的盈利能力；
    4. 排除当年实施的现金分红因素影响后，现金及现金等价物净增加额>0；
    5. 期末现金及现金等价物余额+可迅速变现的金融资产>=有息负债；

企业财务分析步骤

  1. 浏览资产负债表和利润表各科目，关注有无较大比例变动；

    1. 资产端：应收账款，应收票据，其他应收款，长期待摊费用和营业收入的增长对比；
    2. 负债端：应付票据，应付账款，短期借款是否存在大比例增长；
    3. 利润表：营业利润率是否大比例变化；营业利润的公允价值变动收益，投资收益及资产减值损失三者之和在营业利润中的占比有没有发生重大变化；营业利润和净利润的比例是否保持基本稳定；

  2. 进行财务指标的历史分析；

    1. 国内上市企业的业绩，需要看五年以上的才能判断是否可信；
    2. A股一个普遍的规律，公司上市当年的ROE，会比较上市前三年平均水平下降50%，之后也很难恢复上市前水平，可能就是上市时包装过度；

  3. 对比企业利润和现金流量变化的趋势；

    1. 几年中的企业净利润和经营活动现金流量净额应该保持在一个稳定的比例，且最好经营活动现金流量净额大于净利润；

  4. 对比企业与竞争对手；

    1. 比较资产负债结构，看行业内人士对该企业资源的运用方向观点是否一致，对未来发展有共识还是分歧；
    2. 比较利润表指标，通过利润产生的各项收入和费用所占比例判断管理层是否称职；

  1. 销售商品、提供劳务收到的现金>=营业收入：因为“现金流量表”中“销售商品提供劳务收到的现金”的金额是按收付实现制确定的，反映当期因销售商品提供劳务实际收到的现金，包括收回以前各期的应收账款，当期销售已收回的现金，以及预收的货款三部分。 而当期营业收入是按权责发生制确认。 因此两者金额不相同，有可能大于，也有可能小于。 假如企业销售商品回笼的都是现金，没有赊账，那么现金流量表的销售商品及提供劳务收到的现金=营业收入+销项税 。 但是，现实情况却是非常复杂的，在企业实际经营过程中，其真实的销售收入公式如下： 现金流量表的“销售商品及提供劳务收到的现金”=利润表的“营业收入”*（1+增值税税率）+资产负债表的“应收账款、应收票据”（期初-期末）+资产负债表的“预收账款”（期末-期初）。
  2. 权责发生制：也称为应收应付制，以权利和责任的发生来确定“应得的收入”和“应付的费用”，不考虑实际是否收到或者支出的现金；
  3. 经营活动现金流量净额大于净利润：在经营性的应收、应付款金额比例保持稳定的正常情况下，公司的经营活动现金流量净额应基本等于净利润加上固定资产折旧、无形资产摊销等不需要付出现金，但按照权责发生制原则应计入生产成本的费用项目。 也就是说，经营活动现金流量净额一般应大于净利润。 经营活动现金流量净额与净利润经营活动现金流量净额是由在经营活动中产生的现金流入减去与经营活动有关的现金支出构成的；

策略产品经理

智能化时代下的产品经理

  1. AI时代下的市场环境？
  2. 什么是策略产品经理？
  3. 策略产品经理和普通产品经理的差异？

  1. 当下技术和市场环境

    1. 技术快速增长，AI技术突飞猛进；
    2. 增量放缓，存量博弈，企业更关注性价比和利用技术带来的降本增效；
    3. 互联网时代的特性：触达用户成本更低，试错效率更高，面向不同用户的个性化；
    4. 互联网的网络效应，赢家通吃；快鱼战略；

  2. 策略产品经理是什么？

    1. 策略产品经理：策略产品经理是将业务模型建模成业务函数，把握输入，输出，优化目标，推动策略在业务上应用并不断迭代，最终实现业务目标的负责人；

    2. 策略产品经理做什么？

      产品调研> 产品设计> 评审沟通> 模型研究> 开发上线
      数据分析 业务建模 开发量评估 数据准备 项目管理
      竞品调研 逻辑设计 可行性评估 训练模型 测试验收
      用户调研 原型设计 投产比评估 离线验证 线上评估

    3. 相比传统产品经理，策略产品经理应该具备全面的数据分析能力；

      1. 产品上，竞争从可见的感性维度(产品设计，交互，呈现)延伸到不可见的理性维度(背后的算法)；

      2. 能力上，需要懂AI，懂数据；

      3. 思维上，从过去补丁思维(发现一个问题解决一个问题)转变模型思维(系统性理解和建模)

        1. 产品逻辑和要素越来越复杂，例如千人千面无法通过补丁思维解决，每个新要素的增加都会带来测试成本几何指数增长；
        2. 相比于补丁思维的离散性提升，企业更寻求模型思维带来的连续性提升达到收益最大；

    4. 策略产品有个模型研究的阶段，使这份工作成为研究性项目；

      1. 结果不确定性，产品中后期额外的资源投入带来的指标边际提升越来越困难；
      2. 时间的不确定性，开发时间节点无法确定，甚至路径最终验证无法成功；

必备的函数方法论

  1. 如何定义业务函数？
  2. 如何获取业务样本？
  3. 如何选取模型特征？
  4. 分类，回归，搜索问题模型评估方法？

  1. 业务函数的定义和拆解，面临一个具体的业务场景，首先要做的是，对问题建模，将现实问题转化为数学问题；

    1. 定义业务函数，确定业务的输入参数，和输出参数；

    2. 业务函数拆解，这个业务函数f，是我们所需的产品策略；当这个业务函数非常大时，我们通过以下方法拆解：

      1. 横向拆解，从业务输出入手——把业务输出可能的影响因子罗列，例如广告收入；
      2. 纵向拆解，从业务输入入手——依据业务流程，拆解输入的每个过程，例如语音助手；

  2. 函数方法论框架

    1. 当业务函数定义域和值域确定后，如何进行输入和输出？

      1. 人工执行判断工作，例如风控专员，强依赖人工，扮演人肉黑箱角色，人力成本高，管理难度大，执行标准不一，标准泄露风险；

      2. 基于统计样本观察总结经验公式，对真实业务场景抽象和简化参数，对行为或决策定性或定量解释；

      3. 基于单个样本，寻求已知输入和输出和样本，在输入参数寻找相似的特征，在输出定义标签，即完成f(x)=y的函数，以此函数对样本进行预测，评估业务函数对真实业务场景的拟合程度，构建出f(x)的函数模型；

        1. 策略产品在其中承担的工作：确定评估的方法，获得样本数据，确定特征的范围；
        2. 提出基准策略方案（最低成本下可行性系统，项目开始阶段样本少，资源少，可以基于先验知识或数据分析上线简单基准策略，后续根据不断累积的样本去优化模型）；

  3. 函数方法论三要素详解（3+1要素：样本+特征+评估+模型和算法）

    1. 样本

      1. 选取原则：样本的完整性(数据清洗)；与业务目标一致性(选取表现较好的数据)；多元化样本(避免单一获取方法是模型过拟合，模拟更真实环境)
      2. 获取方法：内部数据(埋点，标注，重组，增强)；外部样本(公开数据集，抓取和众包，采购，商务合作)

    2. 特征

      1. 特征时效性，避免事件后信息(未来函数)；

      2. 特征获取成本，避免高频更新的数据；

      3. 注重特征收益，选择高性价比特征可以带来模型的效果大幅提升，精选特征可以稳定模型效果；

      4. 特征选择方法

        1. 基于业务知识，过往经验，选择特征；
        2. 基于模型收益特征选取(封装法)，具有普适性，但是对计算开销较大；
        3. 基于代理指标的特征选择（过滤法），对特征进行指标筛选过滤后构建模型；

    3. 评估：

      1. 离线评估可以分为评价指标，目标函数，二者都是用于衡量模型预测结果和样本的相似度的；产品经理通常更关注评价指标，是一个稳定的参考，也容易与业务指标挂钩；

      2. 常见评估方法：

        1. 分类问题：常用指标：精确率，​准确率，​召回率；

          1. ROC曲线和AUC，利用面积对比区分哪个模型效果更好；
          2. Lift曲线：判断模型在某个样本百分位下与随机先验策略对比的倍数；
          3. K-S曲线：判断模型在哪个样本百分位下效果最好；

        2. 回归问题：常用指标：MAE：平均绝对误差；MSE：均方误差；RMSE：均方根误差；这些误差越小，说明预测结果和真实数据越小

        3. 搜索推荐问题：关心输出顺序，利用MAP和nDCG指标

策略产品的价值，迭代路径与壁垒

  1. 产品价值的主要来源：企业怎么创造价值：劳动，分工，新的技术，制度，交易；除了劳动以外，其它四类本质来源是效用差和效率差；

  2. 效率差是策略类产品主要的价值来源；

    1. 利用机器取代“人肉决策树”，降低边际成本，创造效率差；
    2. 训练一个模型让机器接近或者达到普通人的水准，在此基础上用机器代替部分人工。但需要关注边界，并添加必要的人工干预入口，处理长尾情况，比如机器判断置信度较低的潜在风险内容，突发事件，或者无法融入模型的先验因子；

  3. 利用样本优势构建效率差，利用效率差构建产品的壁垒：高频的业务容易对低频的业务形成竞争优势，高频业务切入低频业务很容易，低频业务难以切入高频业务中；

备注：

  1. 基准策略：Baseline策略，又称基准策略，是指在较低成本条件下使评估指标达到最低要求的策略，目的是为了最低成本快速上线一个可行性的系统；

  2. 封装法：反复尝试不同的特征子集并进行计算评分，最终选取最好模型最好的特征子集，不依赖人的业务能力，具有普适性，但是需要多次迭代，计算开销大；

  3. 过滤法：将特征送入模型和算法之前，可以通过一些代理指标对特征进行筛选，代理指标不决定模型效果，更多通过描述特征分布关系对特征进行侧写；筛选方式可以通过于特征X本身相关的指标，或者度量特征X与标签Y之间关系的指标；

    1. 特征X自身相关指标：

      1. 覆盖率，可以覆盖多少样本；
      2. 方差，不同样本在同一特征值上差异越小，意味特征区分度低，可以替代；

    2. 特征X和标签Y的关系：

      1. Pearson相关系数，等价于经过标准化后余弦相似度，当该值为0时，代表特征和标签线性无关(对非线性无法区分)；
      2. Fisher得分：特征为数值变量，标签为类别变量时，得分越高说明特征在同一类中差异越小，不同类别差异越大；
      3. 互信息：取值为0，代表特征和标签相互独立，取值越大，相关性越大；

  4. 评价指标：从整体上评估模型好坏的相对宏观指标，同一类问题（分类，聚类，回归等）的评价指标基本相同；

  5. 目标函数：用于最优化计算，同一类问题不同模型的目标函数不同；

  6. ROC曲线和AUC：遍历所有预测值为分类阈值时，可以描绘出一条横坐标为假正率，纵坐标为真正率，从（0，0）到（1，1）的曲线，称为ROC曲线；ROC可以用于评估哪个模型更好，谁的面积更大谁更好，用AUC来确定模型在ROC曲线下的面积；

  7. AUC（Area Under ROC Curve），AUC=1，模型效果上限，实际业务中不会存在绝对完美的模型；0.5<AUC<1，正常模型得分，越高越好；AUC=0.5，随机分类模型，也是基准水平；AUC<0.5，还不如随机模型，应该检查是否将正负判断反了；

  8. Lift曲线：Lift=精确率/所有测试样本中正样本的占比，横轴为分类阈值，纵轴为Lift值绘图，利用Lift曲线可以看出模型在某个样本百分位下与随机先验策略对比的倍数；

  9. K-S曲线：纵轴是真正率，假正率，横轴是分类阈值，利用K-S曲线可以看到，当真正率和假正率的差值最大的时候，就是模型在这个分类阈值下效果最好；

  10. 真正率（TPR），和召回率（查全率，灵敏度，Recall）：应该预测为正例的样本中，预测成功的比例（和精确率此消彼长）一致；

  11. 假正率（FPR），表示所有应当预测为负例的样本中，被错误预测为正例的比例；

  12. 准确率（Accuracy）：正确判断了正，负样本的比例；

  13. 精确率（查准率，Precision）：预测为正例的样本中，预测成功的比例；

  14. 召回率（查全率，灵敏度，Recall）：应该预测为正例的样本中，预测成功的比例（和精确率此消彼长）；

  15. MAP：输出结果的精确率两次求平均值；

  16. nDCG：对模型预测的前K个排序结果的评估指标DCG/理想情况得分；

  17. RFM模型：R：最近一次消费时间；F：最近一段时间消费频率；M：最近一段时间消费总金额；

怎么研究用户

  1. 怎么研究用户？
  2. 怎么理解用户行为，用户行为背后的原理？
  3. 怎么理解产品对用户的价值？
  4. 如何提升产品对用户的价值？

  1. 如何理解用户：

    1. 研究用户的四个维度：生物的人；社会的人；认知的人；情境的人；

    2. 用户的五个属性：异质性；情景性；可塑性；自利性；有限理性；

    3. 人脑思考的方式：快思考；慢思考；默认使用快思考考虑问题，大部分决策由快思考作出；

      1. 人类面对食物，性，危险，钱财相关内容时会本能的注意到，用的就是快思考；

  2. 如何理解用户行为：

    1. 背后的原理：用户行为发生之前，会受到一个情景的刺激，调用偏好和认知函数，产生一个主观期望效用，促使用户产生相应的行为；这个行为产生的结果又会成为经验，影响偏好和认知函数，对下一次行为产生影响；

      1. 用户=偏好+认知：偏好是主观的，是用户对事物是什么，和事物对自我的用处的综合认知结果；偏好受到先天遗传和后天教育影响；
      2. 在偏好的影响下，用户仍存在异质性，同样的用户在不同的情景下可能会选择不同的答案；
      3. 不同情景下用户偏好会塑造用户期望效用，这个过程可能会因快思考作出决策，也可能因为认知偏误受到影响；
      4. 这个事物对自我的用处将影响用户选择和行为；
      5. 用户的实际体验将形成经验反馈修正自我偏好；

    2. 如何利用原理：

      1. 选择合适的用户，偏好和认知合适的人群；
      2. 对偏好认知不那么合适的用户，针对情景五步的设计，使用户期望效用最大化的方式是完成某个行为，当用户完成某个行为后才给他想要的，比如拼多多的拉人头红包；

  3. 如何理解用户价值：

    1. 对用户而言，价值由主观效用决定；

    2. 幸福=效用/欲望，通过提高效用或者降低欲望可以满足人的幸福感；

      1. 欲望具有无限性和约束性。是指人的所有行为都是在约束条件下追求价值最大化，理想很丰满，现实很骨感；
      2. 效用特点是主观性，每个人的需求不一样，同一个产品面对每个人的效用也就不一样；

    3. 所以产品的价值属性也就是主观属性，由需求方来决定，用户的主观认知是什么，它就是什么，用户可以感知到的价值才是产品的价值；

    4. 用户价值具备认知依存，情景依存，经验反馈演化三个特性；

    5. 用户价值=产品价值＝新体验－旧体验－替换成本；

      1. 让新体验更大化，让旧体验更小化，降低替换成本；

如何理解产品

  1. 产品是什么？
  2. 如何分析一个产品？
  3. 什么样的产品容易成功？

  1. 产品是一种价值交换的媒介，连接企业和用户；本质上是企业给用户提供的在限定条件下解决某类问题的解决方案；

  2. 分析一个产品的重点：

    1. 利从何来，利往何去；
    2. 创造了什么价值，解决了什么问题或需求，价值怎么分配；
    3. 如果增加了一个功能，为什么增加，解决了用户什么需求；

  3. 怎么做一个产品容易成功：

    1. 创造用户价值，解决用户问题为目的；
    2. 打破一个旧的利益平衡，同时做出增量价值，例如滴滴；
    3. 建立一个对自己有利的新产业链；
    4. 一个好产品的三个属性：解决用户需求，能赚钱，可持续；有效用，有利润，可持续；

如何理解企业

  1. 企业的本质是什么？
  2. 怎么发现市场上的获利机会？
  3. 产品经理对企业的价值？
  4. 如何让效率高于市场？
  5. 怎么理解组织效率？
  6. 如何分析一家企业？
  7. 怎么理解产品对企业的价值？
  8. 企业如何创造价值？
  9. 如何利用产品与用户进行价值交换？

  1. 企业的本质：发现市场获利机会，生产效率高于市场；企业的目的是创造交易，有交易才有用户；

  2. 如何发现市场的获利机会：

    1. 洞察：利用信息不对称获利，知道用户在想什么，知道怎么解决用户的问题，知道哪些约束条件将会变化，知道别人不知道，并对别人有价值的信息；
    2. 试错：由于信息不完备原理，需要通过试错来确定自己的信息价值和市场获利机会；
    3. 偶然性：一分天注定；
    4. 目前这个社会企业获利机会主要源于市场环境和制度变化，关键性技术，组织建设能力；

  3. 产品经理对企业的价值创造：在于发现市场的获利机会，不幸的是这里存在阿罗信息悖论现象，发现后的实现是个漫长过程，成功后无法说明主要原因归于产品经理的洞察；

  4. 怎么让生产的效率高于市场：

    1. 权威性，自上而下的更优决策，具有专业知识的人才在岗位上做出的超过市场水准的更优决策；
    2. 通过制度把个人知识沉淀为企业共同知识，并提供给企业后来者应用；企业各个关键岗位掌握专业知识和共同知识的超过平均水平越多，企业的效率优势越大；
    3. 企业扩张的边界是因企业内部科层增加而增大的组织成本等于外部市场交易成本；

  5. 怎么理解组织效率：

    1. 组织就是有共同目标，共同理念，共同知识，遵循一定规则的一群人；
    2. 共同目标和共同理念不可以轻易改动，也被称为使命，愿景，价值观，可以用来不花钱给成员打鸡血；
    3. 规则也是制度，有效的激励和约束制度可以让组织活得更久；
    4. 一个组织如果目标差，理念落后，运行机制差，没法让成员认同，长期只会逆向筛选人才，劣币驱逐良币；
    5. 决定组织文化的三点：创始人的人格特性，高级管理者的行为，员工受到的企业制度和工作；

  6. 企业自己做产品的产出：相对于投资其它企业做产品，自己做产品时除了财务收益，还具有该领域内技术沉淀，团队，无形资产（利益链的信任，品牌）等；

  7. 从詹姆森生产函数分析企业：Q=Fr（L,K,M,C:T)

    1. 企业产量Q在外部规则r限制下，由技术T约束产出边界；
    2. 劳动力L，资本K，原材料M，企业内部规则C的组合在上述约束下决定生产能力；

  8. 怎么理解产品的价值：

    1. 对产品而言最核心的价值概念是：使用价值和交换价值；
    2. 没有使用价值的东西就不是产品；
    3. 产品不一定有交换价值，具有交换价值的三个条件是有效用，被认知，稀缺性；
    4. 企业创造对用户有使用价值的有交换价值的产品，然后用它跟用户换钱，换时间等企业需要的价值；

  9. 企业怎么创造价值：劳动，分工，新的技术，制度，交易；

  10. 如何更好的用产品进行价值交换：

    1. 理解产品即交易，人做出任何有意识的行为，都是趋向一种对自己更有利的状态，都需要付出代价 ，不限于金钱，时间，体力等成本，产品即交易的交易指把人的任何有意识行为，看作付出预期代价购买预期收益。
    2. 提升对用户主观价值判断水平，以提升对用户需求的理解；
    3. 从用户角度出发：用户选择产品的前提是：效用-成本>0；对用户提高效用，减低成本；
    4. 从企业的角度出发：企业愿意生产产品前提是：收益-成本>0；收益除了金钱收益，还有其它无形收益，企业通过持续维护和为用户创造价值保证交易的“可持续性”，企业的一切行为，都应该围绕“让交易行为更多的发生”;

交易与交易模型

  1. 什么是交易模型？
  2. 怎么发生更多的交易？
  3. 如何理解效用？
  4. 如何理解边际?
  5. 如何理解交易成本？
  6. 如何降低交易成本？
  7. 如何分析一个产品是否可以促成新的交易？

  1. 产品就是交易，每个用户的每次行为都可以看作是和世界进行的一次交易；用户在进行每次行为时，必然会经过快思考或者慢思考，考虑成本和收益；

  2. 世界上不存在等价交换，交换必然存在的额外的收益，每一次交换都给其中参与者创造了新的价值；交换过程中总的物质不增加，但是交换前后对这产品给交易参与者带来的价值完全不同，他们对这个产品的估值增加了，这个世界上人类创造新价值的能力增加了，这个世界上的总价值增加了；

  3. 一个企业的核心目标应该就是创造交易，有交易才有用户，有交易才算完成了价值创造，才有可能赚钱；发现市场机会（用户需求），本质是发现可以促成一个交易，让用户用某种代价来买你的产品；

  4. 什么是交易模型：

    1. 产品经理发现和设计的合理机制，它能促成用户做出某种行为，且可以持续。本质上也属于一种商业模式，用于利益创造和分配；
    2. 重点是利从何来，利往何去；
    3. 几个切入点：损失厌恶，选择增加福利，激励相容；
    4. 产品经理需要做的就是持续发现市场上高于用户支付代价的的用户价值，并设计合适的交易模型把它固化为产品；发现更高效，更低成本的方案解决用户的需求，并让用户愿意买单；

  5. 如何发生更多的交易：

    1. 持续发现和追加可交换的、有利可图的用户价值。
    2. 创造和更高效率地创造这些用户价值。
    3. 持续降低生产成本和所有交易成本。
    4. 企业行为的权重，按影响用户交换行为的ROI排序。企业设计产品时，更应该从这个产品面向的用户在交易时所在意的地方着手；
    5. 维护企业生存能力和可持续发展能力。

  6. 什么是效用：效用：是指满足了用户的什么需求和满足程度，人们默认追求总效用最大化，人们通过…

    1. 边际效用递减原理：一个人对于财富的占有多多益善，即效用函数一阶导数大于零；随着财富的增加，满足程度的增加速度不断下降，效用函数二阶导数小于零。

    2. 最大效用原理：在风险和不确定条件下，个人的决策行为准则是为了获得最大期望效用值而非最大期望金额值。

    3. 幸福=效用/欲望，通过提高效用或者降低欲望可以满足人的幸福感；欲望的无限性是社会进步的推动力；

    4. 效用的类型：马斯洛的需求层次：生理（衣食），安全（生命保障），社交（人际），尊重，自我实现；

    5. 如果用户认可一个产品带来的某种效用，而且这个效用大于用户愿意支付的代价，这个效用就能成立，世界上就多了一笔交易，同时也为双方创造了两组新的价值；

    6. 从需求角度定义企业的用户，不同的产品，不同的使用场景，满足了不同的效用，只要多提供一种效用，多了一种交换内容，就多了一类用户，有交易就有用户；

    7. 产品效用和用户需求的不对等性：底线需求，够用就好，越多越好，惊喜；

      1. 一般来说，我们需要满足用户的“底线需求”，把“够用就好”的需求打磨到边际回报率最划算的那个点就停下，然后把“越多越好”的价值创造做到具有极高的投入产出比，适量提供“惊喜”。
      2. 我们一般说的研究迎合用户需求，其实指的是，在给定条件下，选择创造哪些用户价值、满足哪些用户需求更能促成交换，最适合企业的长期和短期目标，能让企业有最高的边际投入回报比。

    8. 产品是约束条件下的效用组合；

  7. 边际：是指额外的增量；

    1. 边际效用是指消费者对某种物品的消费量每增加一单位所增加的额外满足程度。
    2. 边际成本是指生产者对某种产品的生产了每增加以单位所增加的额外成本；
    3. 边际利润是指在一个给定约束条件下，再多做一笔交易，增加的额外利润。

  8. 什么是交易成本：也称为交易费用；

    1. 从产品角度来看，交易成本的本质是，所有用户（买方）付出的但企业（卖方）没收到的成本，以及所有企业（卖方）付出但用户（买方）没收到的成本。

    2. 考虑产品和交易模型需要重视交易成本，交易成本的变化可能会带来交易方式的变化和生产方式的变革；

    3. 移动互联网通过替代效应和新增效应促使用户交易；

    4. 机会主义的存在使交易费用提高。交易越复杂，交易费用提高的幅度也越大。一切足以提高市场交易费用的其他因素都是通过人的机会主义行为才使交易费用上升的。

    5. 从产品角度，交易成本源自信息不对称，信息不确定，信息不完整，信息有成本；所以，能降低交易成本的手段通常是信息相关；

    6. 质量信息不对称和产品质量的可度量性，严重影响我们设计市场结构，很多互联网产品的设计就是利益交换系统的设计，其实就是市场设计。

    7. 交易成本分为：买卖交易成本，组织管理成本，政治交易成本；

      1. 搜寻（商品和交易对象）成本和度量（交易对象和商品的属性）成本；
      2. 寻价（议价比价）成本和决策（决策和订立契约）成本；
      3. 实施成本和保障（权利、违约、意外、监督等）成本；

    8. 按照质量信息确定性的高低，可以把产品分为搜寻品，体验品，信任品；我们如果可以把产品标准化成搜寻品，用户的决策过程交易成本大幅降低，可以促成更多交易；

    9. 交易对象和商品质量的度量成本常常很高，一旦有新的有效度量方法问世，往往能大幅降低交易费用，促成无数新交易，使市场扩大。例如评价，点评，打分，降低很多交易的度量成本，也促成很多新的交易；

    10. 如何降低交易成本：

      1. 标准化，并且让用户信任这个标准，把产品变为搜寻品，例如麦当劳或链家真房源；
      2. 将用户的交易成本转化为收益，例如智能手机的普及让人们的时间成本可以做到转化；
      3. 线上化，便于进行有效度量，降低决策交易成本；
      4. 复杂过程藏在背后，面向用户的是最简单的，最符合人类直觉的；

  9. 人间第一定律：其它条件不变时，相对价格降低，需求量上升；

    1. 相对价格=（直接成本+交易成本）/效用组合；
    2. 分析一个产品功能是否可以促成新的交易，观察它的成本是否降低，效用是否提升；

决策

  1. 怎么做出理性的决策？
  2. 在做产品时如何定义目标？
  3. 常用的决策方法？
  4. 常用的决策误区？
  5. 什么样的决策才有价值？

  1. 怎么做出理性的决策？

    1. 理性决策的三要素：理性信念，理性目标，理性的行动；如果第三项有所成就，决策的边际收益更依赖前两项。

    2. 决策就是选择，是在多个可行方案中选择最佳方案。所以我们需要训练模拟推演各种可能性的发展过程和结果的能力，以及在多个方案中做复杂价值判断的能力，不同利益主体间权益平衡的能力；

      1. 价值判断不仅仅是遵循“用户至上”就够了，因为“用户至上”天生是行不通的。
      2. 用户和企业的利益也有冲突，单边倾斜做成的产品，是不可持续的。互联网产品做大后都会变成平台，涉及生态链上的利益和冲突；

    3. 希望做大互联网产品，需要用经济学思维，通过权衡取舍以追求整个系统的效率；

    4. 决策的目标：价值最大化；在用户价值=产品价值＝新体验－旧体验－替换成本背景下，追求目标是新体验的最大化，旧体验最小化，替换成本最小化；

      1. 将新要素引入原有生产方式或生活方式，如能有效应用，即可创造大量新价值——这也是创新的本质含义。新体验不限于新要素与旧要素的结合，旧要素与旧要素的新组合，也可以创造新体验。

      2. 旧体验最小化的完美情况是没有同类竞品，这种新场景下的新用户性价比最高，他们往往对公司产品和品牌产生光环效应，较大概率成为忠实用户，是公司品牌价值和长期市场竞争力的重要来源，并且先发者更容易在用户心中梳理体验标准，当用户在面临其它产品时，会存在替换成本；

        1. “完美新用户”概念带给产品发展路径的一个方法是：一种全新种类的产品出现早期，不一定只能做首创发明者，即使是模仿者、后来者也会有机会。一旦产品的用户价值被验证（不要追求大版本和完美功能），应该最快速地提高市场渗透率，把所有“完美新用户”的新体验都变成旧体验，之后竞品再想抢市场就很难了，因为它会面临把“新体验与旧体验之差”做到大于“替换成本”的问题；

      3. 一般来说，产品经理要么降低自己用户的替换成本，要么提升自己用户的流失成本（相当于提高了竞品的替换成本）。

        1. 让用户从其它平台迁移过来的更方便平滑（竞争环境的拉新），或者让用户在自己产品上花费更多成本不便迁移（竞争环境的留存）；
        2. 替换成本：常见的替换成本是认知成本、获取成本、使用成本，而广义来说，价格和所有交易成本也都是替换成本。

    5. 用户价值判断的权衡视角

      1. 对自我的认知判断；对给定目标的批判性思考；用户价值的参照系；成本；尊重和敬畏世界的不确定性；成功的概率；非货币的其它价值；外部性；时间尺度标准；

  2. 常用的决策方法：

    1. 数据决策：数据对于做产品的价值，一般是作为做正确决策和达成共识的最低成本工具。

      1. 宏观经济理论必须建立在微观个体最优决策的基础之上，而不是基于任意的统计观察，否则会导致卢卡斯批判。
      2. 用户模型和交易模型更可能作为一种辅助思考决策的高级工具，与数据驱动一起发挥作用。

    2. 逻辑决策：需要让用户有着一定的逻辑水平，成本较高；

      1. 一个人一天的行为中大约有5%是非习惯性的，而其他95%的行为都源自习惯。这基本上意味着是习惯而不是逻辑，决定了我们的一生。
      2. 所谓理性，是指当条件合适时（理性并非总是优于本能和习惯），优先用逻辑、知识和思维去做出决策和预期收益最大化的行为，而不是任由本能和习惯驱使。

    3. 主观判断决策：是最难的决策，没有数据或逻辑支撑，只能用关键变量和用户模型、交易模型来模拟市场运行，并据此做出判断。此类决策往往还需要强大的个人影响力来支撑落地。

      1. 需要提取特定环境下的关键变量（关键约束条件），事先找到用户模型和交易模型，极高的洞察能力，才能及时，高效，较高水平的构建模型推演，满足决策需要；
      2. 当有一个想法时，把它记录下来，用过往的经验去证伪，如果无法证伪，就假设它成立，存储起来，以后有新的信息或思考时，随时将它提取出来匹配一下，就可能有所得，从而产生新的结论。

  3. 常见的决策误区：

    1. 大脑自有一些处理资讯的规则，为了增进决策与判断效率的“心理捷径”称作捷思，捷思有时会产生一些不好的效应，这些不好的效应称作认知偏误。

    2. 归因偏误：遇事需要归因时，从三个维度快速评估一下行为主体（人或组织）、关联方（如果有，也进行一遍三维归因）、行为情境对结果的影响，往往能减少错误归因。

      1. （情境）特殊性（distinctiveness）：行为人在不同情境中是否表现不同。
      2. （他人）一致性（consensus）：其他行为人在相同情境下是否表现相同。
      3. （个体）一贯性（consistency）：行为人在不同时间是否表现相同。

    3. 锚定效应：锚定效应是指为不熟悉的事物估值时，会把熟悉的类似事物或不久前接触到的无关数值当作“锚”，估出来的数值会大大倾向“锚”。

    4. 选择性注意：人们常常倾向于忽略掉不符合自己期望的东西，只看到自己期望看到的东西，他们期望看到的东西又是基于熟悉的东西、先前的经历，或是个人的欲望、兴趣需求，这就是选择性注意。

    5. 刻板印象：指对某人或某一类人产生的一种比较固定的、类化的看法。人们习惯把人进行机械的归类，把某个具体的人看作某类人的典型代表，把对某类人的评价视为对某个人的评价。

    6. 自我中心偏误：会美化回忆中的自己，往往认为自己比实际情况更好。

    7. 信念偏误：认知与现实的偏差。

    8. 决策偏误：根据结果倒推过程的偏差；

  4. 能落地的决策才有价值：一个管理者职位越高，他的日常工作中一般都会有越高比例的沟通说服工作。

    1. 推动决策落地的方法：规章制度，理性说服，画大饼，共赢，私交，施压，目标对齐；
    2. 推动决策落地的技巧：设置引起讨论的议题；

备注

  1. 经济学公理：理性人是自私人；

  2. 人类知识体系大致分为四类。

    1. 自然科学：研究对象是自然界存在的物质，可证伪，可重复验证，所发现规律一般有普遍适用性，如物理、化学、天文、地球科学等。
    2. 形式科学：研究对象在真实自然界不存在，是人类建构的抽象规则，不涉及经验，不可能证伪，永远正确，如数学、逻辑、概率、编程语言等。
    3. 人文科学：研究对象是人内心的主观世界和精神活动，无法区分对错，无法证实，不适用于证伪，如宗教、哲学、诗歌、文学、美术、音乐、影视等。
    4. 社会科学：研究对象是人作为社会中的存在，或人与人组成的社会，如社会学、政治学、经济学、社会心理学等。社会科学与自然科学的最大区别是研究对象不可重复（人类意识复杂多样，决定人类行为的动机和预期有互相博弈性和不确定性，社会永恒变化，正所谓人不能两次踏进同一条河流），虽然能应用自然科学的研究方法（观察、分析、控制实验等），但必须以大量的经验和数据为依据，且控制实验事实上无法真正重复验证，社会科学的研究结论就总是有不确定性，无法做到自然科学的高确定性（对于自然科学，假如观察到的现象与规律所预言的不同，要么是因为观察中有错误，要么是因为至此为止被认为是正确的规律是错误的）。

  3. 效用：是指满足了用户的什么需求和满足程度，人们默认追求总效用最大化，人们通过消费物品和劳务满足欲望。

  4. 成本，要明白什么是直接成本、交易成本和机会成本。

  5. 边际，要明白基本的边际递减、边际成本、边际效用、边际交易。

  6. 供需定律是指其他条件不变时，价格下降则需求量上升。

  7. 用户模型：用户模型是指产品用户和潜在用户的偏好和行为反应模型。

  8. 用户模型掌握的标志是：预判产品迭代后的用户行为变化，准确率较高。要以异质性、情境性、可塑性、自利性、有限理性的用户行为为基本单位来研究产品的用户模型，并通过大量真实个体行为样本来进行分析和洞察。

  9. 交易模型：研究是指以交易为基本单位来研究产品，目标是建立可持续交易的互惠模型。

  10. 交易模型掌握的标志是：理解一个领域各利益方的价值判断和行为习惯，以及产品实现过程中的各种干扰因素，这样才能自如地设计和优化交易模型。

  11. 产品每增加一个属性，调整一个功能，都是在调整效用组合，给用户提供不同效用，有增加，也有减少。在增加和减少效用的同时也进行了用户选择，选择了哪一类用户，哪一类情境优先，哪一类其次。因为不是所有的效用都能无限叠加，且用户体验是有成本的，有的效用叠加成本很高，一旦增加，企业可能就要亏本，甚至活不下去。企业追求的是效率，简单来说，就是考虑投入和产出。这些，就属于交易模型的范畴。

  12. 可持续：只有产业链上各方在创造价值和利益分配方面公平合理，有竞争性，产品才可持续。

  13. 异质性：用户千差万别，每个人都不一样，千人千面，用户画像不唯一；

  14. 情景性：不同的环境背景下每个用户有着不同的选择，反应和行为，人类受到环境影响；

  15. 可塑性：用户是变化的，认知和偏好是可以改变的；

  16. 自利性：本性是自私的，追求个人的利益最大化；

  17. 有限理性：大部分人的能力有限，会做错，会被欺骗，只能做到一定程度的理性，再深入就容易出错；

  18. 快思考：代表人类的本能和习惯，可以快速且自动启动，耗费资源很少，经常为习惯、经验、刻板印象等认知偏见所支配；

  19. 慢思考：代表人类的逻辑和理性，其启动是被动、缓慢、耗费资源的，存在意识控制；

  20. 价值的认知历史：早的定义出现于经济学萌芽期，指无差别的人类劳动。价值是劳动决定的，这本质上是成本价值论或客观价值论。经过从19世纪70年代持续到20世纪初的边际革命，人们认识到边际效用的概念，认为价值是由效用决定的，国际学术界和社会开始普遍接受价值的主观性，放弃了价值的客观性。由需求决定价值；

  21. 认知依存：用户的认知决定了他的偏好；

  22. 情景依存：有情景才有用户，脱离情景就没有用户，需要环境和氛围；

  23. 经验反馈演化：用户需求是变化的，会随时间环境迭代，演化的结果是相对价格（付出的成本）的变化，并决定不同的资源配置，进而实现价值的最大化；

  24. 阿罗信息悖论：在买方得到信息前，他并不了解信息对他具有价值，但是他一旦了解了信息的价值，他事实上已经无成本的获得了这个信息；就是说在发掘用户新的需求时，他没意识这个需求对他有啥用，但是当他意识到后，他就已经解决了；

  25. 使用价值：产品的各种效用，也就是用户价值；

  26. 交换价值：是一种使用价值同另一种使用价值相交换的量的比例或关系。交换价值是社会属性，一份产品的市场价格反映其交换价值，是市场供需博弈的结果。

  27. 马斯洛的需求层次：生理（衣食），安全（生命保障），社交（人际），尊重，自我实现；

  28. 损失厌恶，指的是人们面对同样数量的收益和损失时，认为损失更加难以忍受。同量的损失带来的负效用为同量收益的正效用的2.5倍。所以平均来说，用户换回的物品或其他价值要大于支付代价的2.5倍，交易才能成立。

  29. 选择增加福利：从用户角度，有选择比没得选感觉更好点（需要考虑成本）；

  30. 激励相容：在市场经济中，每个参与者都会按实现个人利益最大化的策略行动，如果有一种制度安排，使行为人追求个人利益的行为正好与企业实现整体价值最大化的目标相吻合，这一制度安排就是“激励相容”。简单地说，就是让对方干得好要比干得不好时获得更多收入，这样对方才有上进的动力。

  31. 发现更高效，更低成本的方案解决用户的需求，并让用户愿意买单；

  32. 有利可图：这里的利更为广义，可以指从用户身上获得的除金钱以外其他的东西，比如时间，流量等；

  33. 可持续：把一次性博弈改为重复性博弈，在多次博弈的情况下，合作是最稳定的；

  34. 所在意的地方：价格是显性的，可能在不同场景下也存在着其它隐性因素，比如对于电商和外卖行业来说，较少受供给影响，价格比使用体验（以下均指符合市场底线预期的至少及格的使用体验）更影响用户行为。一些视频和内容类产品不太受价格影响，但供给比使用体验更影响用户行为。对于搜索引擎和很多工具类产品，价格和供给都难影响用户行为，使用体验和先发优势的影响权重较高，产品经理在市场竞争早中期的价值权重也就较高。对于打车产品，供给和价格都比App交互体验更影响用户行为。在有些情境下，政府态度和安全感也更影响供给、价格和用户行为。

  35. 交易费用：

    1. 诺贝尔经济学奖获得者科斯认为，交易费用是获得市场信息、谈判、订约、监督等活动的费用，并认为企业的管理费用和市场交易费用决定了企业和市场的边界。
    2. 诺贝尔经济学奖获得者阿罗定义，交易费用是经济系统的运行费用。
    3. 巴泽尔定义交易费用为与权利的转让、获取和保护相关的费用。
    4. 沃利斯和诺斯将经济活动分为交易活动和生产转换活动，并据此将经济活动中的费用分为交易费用和生产转换费用，给出交易费用的范围是衡量交换物之价值的成本、保护权利的成本和监管实施契约的成本。
    5. 张五常认为，交易费用包括所有在鲁滨逊似的一人世界不可能出现的费用，交易费用就是制度费用。
    6. 诺贝尔经济学奖获得者威廉姆森将交易费用分为事前的交易费用和事后的交易费用。他认为，事前的交易费用是指由于将来的情况不确定，需要事先规定交易各方的权利、责任和义务，就要花费成本和代价；事后的交易费用是指交易发生以后的成本，它可以有许多形式。

  36. 交易成本的影响：新制度经济学认为，单位交易费用的下降会带来交易方式和生产方式的变革，从而导致市场规模的扩大和经济显著增长。诺斯则用交易费用来衡量制度的有效性，并提出降低单位交易费用会提高制度的效率，他认为，制度变迁才是经济发展的真正原因。

  37. 机会主义：威廉姆森把人一有机会就会不惜损人而利己的“本性”称为机会主义。人的这种本性直接影响了以私人契约为基础的市场效率。市场上交易的双方不但要追求自己的利益，还要随时提防对方的机会主义行为。每一方都不清楚对方是否诚实，都不敢轻率地以对方提供的信息为基础，而必须以自己直接收集的信息为基础做出交易决策。

  38. 搜寻品是高质量确定性高的产品，例如苹果手机，麦当劳汉堡，在任何地方体验一致；

  39. 体验品是需要使用后才知道质量的产品；

  40. 信任品是指消费者即使在使用后也无法验证其质量的产品，比如有些美容品和保健品。

  41. 股票精选点评的理论化分析：用户选股因为信息的不确定性具有很高的交易成本，我们通过把投顾对个股的点评进行分析统计，度量个股，降低用户决策成本，促成新的交易；

  42. 决策：是指在收集、加工信息之后，借助一定工具和方法，进行分析、计算、判断，并得出结论的过程。

  43. 理性信念：即与真实世界一致的信念，也可以理解成“对自我认知的认知”。我们要对自我认知保持怀疑和反思：关于某个决策所需的完备信息，我的认知程度可以打几分？可能用什么方法，值得付出多少成本来获取哪些信息？我可能存在哪些认知偏差？人类最大的理性，是理性地理解自身认知能力的局限性，除此无他。

  44. 理性目标：应该是约束条件下的价值（总效用）最大化。我们应保持对当前目标的反思：我的目标（或任务）理性吗？为什么不能是其他目标呢？从确定这个目标到现在，有什么关键变量（约束条件）已发生变化或将会发生变化？

  45. 理性的行动：是在给定目标下，寻找最优解决方案。

  46. 用户至上的弊端：很多人，无论是用户自己还是产品经理，对“用户价值”“用户体验”的理解可能不同，不同用户群体的需求和利益也有矛盾和冲突。我们要满足谁，不满足谁？或在不同情境下分别向谁倾斜呢？像《民粹主义》里的人民是谁？

  47. 新要素：历史上，计算机、互联网、智能手机、移动支付，都是重大新要素。产品经理的职责，就是加速新要素的应用与普及。除了新的技术，也包括新人群（如有些产品要下沉到三四线用户）、新渠道（如公众号、小程序、抖音）、新方法（如AB测试、产品驱动、数据中台等）、新工具（例如工作中用到的生产力工具，iPhone、钉钉、维基百科、谷歌等都可创造更好的新体验）等等。

  48. 替换成本：常见的替换成本是认知成本、获取成本、使用成本，而广义来说，价格和所有交易成本也都是替换成本。

    1. 认知成本包括品类认知（如了解触屏手机）成本、品牌认知（如了解苹果、华为、小米）成本、美誉度认知（如口碑评价）成本等等。
    2. 获取成本包括渠道（如公众号、小程序、App、二维码）获取成本、下载成本（如减少安装包大小）等等。一个极端的例子是捆绑安装，此举大幅降低了替换成本，但依然受新旧体验差影响（若产品太烂，相对市场平均水平是负体验，用户还是会走）。
    3. 交互视觉成本就属于使用成本。越简洁易用，使用成本越低；越沿袭原有用户的习惯，使用成本越低；极端情况下，有些产品直接复制行业已有的交互，使用成本几乎为零。又如，某些产品随使用时间的延长，体验变好（如个性化推荐型产品），此时用户切换到其他产品的成本就会升高。

  49. 卢卡斯（凭理性预期理论获1995年诺贝尔经济学奖）认为，利用加总的历史数据来给出政策建议和经济预测，其结论值得怀疑。

  50. 逻辑决策也是“讲理决策”，结论靠逻辑推导得来。但需要对方具备相应的认知水平，否则就会鸡同鸭讲。想让一个人理解和接受一个道理，是要在他的偏好和认知结构内去实现的。如果对方不具备相应的偏好和认知结构，就需要帮他补齐，但这个成本实在太高了。所以，逻辑决策的成本远高于数据决策。

  51. 认知偏误：是指在某些特定情况下的特定思考、行为倾向，会导致理性或判断产生系统性偏误，这些现象被心理学与行为经济学广泛研究。

  52. 归因偏误常见现象：

    1. 自利偏误：将成功归因于自己，而将失败归因于环境。把集体行动的结果归因于自己，多过其他人会归因的程度。
    2. 正面效应与负面效应：评价喜欢的人的行为时，把他们做好事归因于内在本质，而把他们做坏事归因于环境因素。评价不喜欢的人的行为时，把他们做好事归因于环境因素，而把他们做坏事归因于内在本质。
    3. 群体归因偏误：把个体的特质当作其所属群体的特质，或把个体的行为当作其所属群体的偏好。终极归因偏误：把问题归因于整个群体，而非群体中的个体。
    4. 基本归因偏误：解释他人行为时，过度归因于内在特质（一定是他有这样的人格，才做出这样的行为），而非外在情境（也许是情势所迫，或这个场所有特殊的潜规则）。

  53. 锚定效应常见现象：

    1. 幸存者偏差：过度关注在某些经历后幸存下来的人事物，忽略那些没有幸存的（沉默的数据），造成结论偏离实际情况。
    2. 峰终定律：我们对一项事物进行体验之后，印象最深的是在峰值与结束时的体验，而在过程中好与不好体验的比重、好与不好体验的时间长短，对记忆影响很小。高峰之后，终点出现得越迅速，这件事留给我们的印象越深刻。
    3. 框架效应：同一资讯以不同方式呈现会使得人的想法不同，例如告知公众有一个做法将会“导致三分之一的人死亡”和“保护三分之二的人活下来”的效果是不一样的。
    4. 禀赋效应：指人们拥有或即将拥有某物品或资产时，对其价值的评估会比没有时高出许多，因而不愿失去或放弃它。
    5. 心理账户：人们将消费分门别类（住宅、食物、服装、娱乐、投资等），且每一个类别对应各自的心理账户，每个心理账户都有自己的预算和单独的参照点，使得各个账户之间的替代性非常有限。一个人对给定数量金钱赋予的价值，取决于这部分钱属于哪一个心理账户，继而依赖于背景、框架和情境。心理账户能解释同一个人在不同事物上的花钱矛盾现象。
    6. 损失厌恶：指人们面对同样数量的收益和损失时，认为损失更加令人难以忍受。损失带来的负效用约为同量收益的正效用的2.5倍。损失厌恶反映了人们的风险偏好并不客观，当涉及的是收益时，人们表现为风险厌恶；当涉及的是损失时，人们则表现为风险寻求。

  54. 选择性注意常见现象：

    1. 选择性观察偏误：人们在观察时不可避免受到前置条件的限制而筛选了样本，因而得出不适当的结论。例如问卷调查到的人一定是热心、愿意填问卷的人，因而其结果未必能反映不热心、不愿意填问卷的人的想法。
    2. 观察者偏见：是指由于观察者个人的动机和期望导致的偏误。通常，人们看见的和听见的只是他们期望的，而不是事实的本来面目。观察者期望效应：由于研究者期待某种结果，因而下意识中不当操作实验或诠释资料。
    3. 受试者期望效应：由于受试者期待某种结果，因而下意识地扭曲了回报内容。常见现象是安慰剂效应，即被给予无效的药物或治疗，病人却相信或觉得情况改善。

  55. 刻板印象常见现象：

    1. 光环效应：一个人的某个单一特性（比如漂亮、长得帅、富有等）一旦给人以非常好的印象，在这种印象的影响下，人们对这个人的其他品质（与这个单一特性不相关的品质如能力、人品）也会给予较好的评价。
    2. 外群体同质性偏见：认为自己的个性、行为、情绪是多变的，而他人是一成不变且容易预测的。
    3. 首因效应：指个体在社会认知过程中，通过“第一印象”最先输入的信息，对以后的认知产生的影响作用最强，持续的时间也长，影响力远大于以后得到的信息。

  56. 自我中心偏误常见现象：

    1. 自我美化偏误：会美化回忆中的自己，往往认为自己比实际情况更好。例如记忆中自己的考试成绩比实际更好。
    2. 支持选择偏误：回顾自己做过的选择时，倾向认为它是明智的，给出高于实际的评价。如购买后把之前的购买决定合理化，即使买下的商品太过昂贵或有瑕疵。
    3. 错误共识效应：高估别人对自己的认同程度，认为所有人以同一方式思考。
    4. 投射偏误：不自觉地以为他人（或未来的自己）和（现在的）自己有相似的情感、思想与价值观。
    5. 素朴实在论：相信自己所见所闻即是真相，是客观且不带偏见的，认为这是显而易见的事实。控制错觉：高估自己对外在事件的影响力，认为事情是受自己控制或影响的，但实际上它可能与自己毫无关系。
    6. 达克效应：能力过差的人由于无法分辨能力好与能力差的不同，因而会错误地高估自己的真实水准，有一种虚幻的自我优越感。
    7. 逆火效应：遇上与自身信念抵触的观点或证据时，除非它们足以完全摧毁原信念，否则会忽略或反驳它们，原信念反而更加强化。
    8. 证实偏见：人们普遍偏向能够验证己方假设和观点的信息，而不是那些否定假设的信息。关注、寻找、诠释、记忆信息的方向多半是能证实自己假设和成见的方向，而那些可能推翻自己观点的信息往往会被忽视掉。

  57. 信念偏误常见现象：

    1. 可得性偏误：对容易想到的事，会高估其发生概率。然而一件事是否容易想到还受发生多久、激起情绪的程度等因素影响，无法反映实际的发生概率。
    2. 频率错觉：因最近注意到一件原先没注意到的事，就觉得这件事到处都在发生。
    3. 后见之明偏误：又称“我早就知道了”“马后炮”“事后诸葛亮”。在事情发生或发展后，以为自己事前就能预测其发生与发展。
    4. 可获性层叠：一件事越常被公开谈论，人们就越相信其正确性，类似“三人成虎”。
    5. 货币错觉：专注于货币名目上（表面上）的价值，而非其实质购买力。
    6. 一致性偏误：记忆中他人过去的态度与行为会变得像目前的态度与行为。
    7. 巴纳姆效应：人们会把他们认为是为自己量身定做的人格描述评价为高度准确，而这些描述往往十分模糊及普遍，能放诸四海而皆准，适用于很多人。这个效应多出现于占卜星相、算命和个体性格分析等情境。
    8. 基本比率忽视：只关注针对性的信息，忽略一般性的信息（基本比率），导致不恰当的认知。例如因为酒驾较容易肇事，就认为肇事者中很多是酒驾，然而由于酒驾者占所有驾驶者的比例很小，肇事者中酒驾者的比例不会那么高。
    9. 赌徒谬误：认为某事多次发生则未来发生的概率较小，或多次未发生则未来发生的概率较大。就像赌徒认为掷骰子时连续几把结果是“大”，那么下一把是“小”的概率更高，而忽视了每一次掷骰子都是独立事件，有独立概率。
    10. 逆赌徒谬误：认为发生概率很小的事不会发生，如果发生了，一定是做了很多次。

  58. 决策偏误常见现象：

    1. 结果偏误：评价决策好坏时，根据其最终结果，而不是根据做决策的过程。
    2. 道德评价偏误：评价他人行为的道德时，根据结果而不是根据做出行为时的情境。
    3. 资讯偏误：倾向寻求更多资讯以做出决策，即使寻求的资讯对决策没有帮助。
    4. 共有资讯偏误：团队讨论倾向花较多时间与精力讨论所有成员都知道的事（即共有资讯），而花较少时间讨论较少成员知道的事。
    5. 集体错觉：团体在决策过程中，由于成员倾向让自己的观点与团体一致，因而令整个团体缺乏不同的思考角度，不能进行客观分析。
    6. 从众效应：倾向做很多人做的事，或相信很多人相信的事，以和他人保持一致，也就是“随大流”。
    7. 沉没成本谬误：当人们证实先前累积的投入（沉没成本）已经浪费掉了，仍然对这些已经不能回收的成本念念不忘，因而做出不理性的选择。
    8. 塞麦尔维斯反射：条件反射般地否定、拒绝新证据或新知识，因其有悖于现有的常规、信仰或价值观。
    9. 麦纳马拉谬误：过度相信数据，依赖数据评估事情，忽略难以量化的事。此典故源自美国前国防部长麦纳马拉，他相信能用量化数据（如击毙越军的尸体数量和减少己方人员的伤亡数量）衡量美国在越战中的成功，但忽略了其他因素。

八次危机

发表于 2022-01-03 更新于 2024-03-17

八次危机

作者：温铁军

作者团队观点

  1. 人类文明差异派生论：气候周期变化与随之而来的资源环境条件的改变对人类文明差异和演化客观上起决定作用；故人类文明在各个大陆演化进程应该是多元化，而不是遵循产业资本时代西方经典理论学家提出的生产方式展开；

  2. 制度派生极其路径依赖理论：不同地理资源条件，决定了近代全球化之前人类文明及制度的内生性和多元性，决定了近代史上不同现代化的原始积累途径，由此形成不同的制度安排和体系结构，并构成了其后制度变迁的路径依赖；

  3. 成本递次转嫁理论：近现代以来，在全球化所形成的世界体系中，核心国家和居于主导地位的群体既得利益阶级不断向外转嫁制度成本而获取超额收益，以完成资本的原始积累，实现产业扩张和向金融资本跃升，广大发展中国家和底层民众不断被迫承受成本转嫁而深陷“低水平陷阱”难以自拔；

    当代全球化本质上是一个因为不同利益导向而相互竞争的，以金融资本为主导，递次向外转嫁成本以维持金融资本寄生性生存的体系。最终承受代价的是资源和生态环境；

  4. 发展中国家外部性理论：第二次世界大战后，绝大多数发展中国家是通过和宗主国谈判形成主权，实质是一场交易，任何交易都存在信息不对称带来的风险，因转交交易范围之外的经济和社会承载而转为外部问题，任何信息单方垄断都在占有收益的同时，对交易的另一方做成本转嫁，由此发展中国家谈判中形成的主权必有负外部性，导致难以摆脱“依附”地位；

    但，越是一次性博弈则风险爆发造成的谈判双方双输的可能性越大，发达国家在巧取豪夺巨大收益的同时，其风险也在同步深化和加剧；

  5. 乡土社会应对外部性的内部化理论：中国作为原住民人口大国唯一完成工业化的国家，其比较经验恰恰在有着几千年“内部化处理负外部性”的村社基础，其中，村社理性和政府理性构成了中国两大比较优势。

    但，政府同样是人类制造出来但反过来统治人类自身成本高昂的产物，遂有政府与资本结合相结合激进推进现代化之后的经济，社会，文化，资源和环境等负外向性问题，成为中国通往可持续的问题；

  6. 从历史经验看，新中国在其追求“工业化+城市化=现代化”道路上，发生过八次经济危机，凡是能动员广大农村分担危机成本的，就能实现软着陆，否则，只能在城市实现硬着陆；

  7. 自20世纪80年代农村去组织化改革以来，作为经济基础的“三农”日渐衰败，作为上层建筑的“三治”成本不断上升，二者之间的错配乃至哲学意义上的冲突日益深化，导致在生态环境，食品，文化等多方面形成短板；

1958-1960第一次经济危机

1950年，朝鲜战争爆发，中国开始引入苏联外资，接受苏联的重工、军工产业转移；

1958年，中苏关系破裂，苏联撤走专家和外资，中国欠下苏联巨额外债，为了维持城市工业发展，发动农村人民公社化，大跃进、大炼钢铁开始；

1960年，苏联撤资后，中国以农产品还债，同期中国经历三年自然灾害，由于之前城市大量推动无效的生产，产生大量赤字，使城市陷入经济危机，大量青年失业，城市青年“上山下乡”，据不完全统计，前两次上山下乡，农村吸纳无业青年总数达4000万人；

‍

1968-1970第二次经济危机

从1964年开始，中国陷入地缘政治危机，被迫投入大量资本在国家战略纵深进行“三线建设”，1966年， “文化大革命” 爆发，使城市工业停摆；

随后1968年，中国第二次赤字型经济危机爆发，只得再次发动“上山下乡”，向农村输送千万过剩的城市劳动力，使城市危机“软着陆”；

‍

1974-1976第三次经济危机

1971年，中美关系缓和，之后中美建交，中国开始从转向向西方引进资本，1972年中国第二次引进外资，进行43亿美元的生产设备采购，改变原有军工、重工的工业结构，向沿海、轻工业转型；

1974年，中国引进西方设备和国内配套投资都造成了巨大的财政赤字，中央再次启动“上山下乡”，这也导致之后的政治运动—— “批林批孔”和“批邓反击右倾翻案风”

1976年，周恩来、毛泽东相继去世，地方工业化再度起步，中央领导提出更大规模引进外资的82亿美元的“八二方案”，引起地方政府大量追加引进外资，直至1978年，中国经济面临崩溃；

‍

1979-1980第四次经济危机

1978年，中国确定了对内改革，对外开放的政策。1979年第四次经济危机爆发，年度赤字相当于50年代累计赤字总额，1980年，政府退出农村经济，农村集体化解体；

1981年，由于经济危机在城市“硬着陆”，企业“关停并转”引发大规模失业，约4000万“待业青年”滞留城市，随即造成城市社会治安动荡，政府发起两个 “严打” 活动——严厉打击“刑事犯罪”和“经济犯罪”；

1982年，中央一号文件允许农村家庭承包，国家允许“小农经济”回归，使生产力基本要素回到农户和村社，乡镇企业开始萌芽；

1985年，中国强调允许 “先富带动后富” ，沿海各地陆续开放，借加工贸易引入外资生产线，挤占国内上游设备制造业市场，1986年沿海经济进一步开放，实现快速增长，但同时因为价格双轨制，造成官倒囤积和腐败泛滥；

‍

1989-1990第五次经济危机

1988年，为了价格并轨，国家逐渐统一物价，在88年物价闯关失败，诱发抢购挤兑，发生严重通货膨胀和生产停滞，引发严重的社会危机，但政府缺乏改革失败的预案，调控滞后，造成高达500亿的巨额赤字。1989年三角债爆发并生产停滞，并引发更大的政治风波。1990年，因城市经济危机，农产品和乡镇企业产品全面滞销，农民现金收入连续3年下降，中国城市资本危机又一次把代价转嫁给农村；

‍

1993-1994年第六次经济危机

1991年，苏联解体，东欧剧变；

1992年，政府由于财政赤字取消了粮票等票证，从此人民币成为真正意义上的货币，完成了实质上的货币化。随之而来的是证券、期货、房地产三个投机性资本市场开放，但由于当时没有商业银行，财政赤字开始从银行透支，至1993年，引发了由财政、外汇、金融赤字导致的第六次经济危机；

1994年，为应对经济危机，中央政府进行分税制改革，导致地方政府通过圈地生财，引起社会群体事件显著增加。同时进行国有企业“下岗分流，减员增效” ，导致4000万人失业；

1996年，中国财政GDP下降到历史最低，为了甩包袱，教育、医疗“被产业化”；

1997年第七次经济危机

1997年，东南亚金融危机，引发中国第七次经济危机，1998年，中国启动国债投资应对危机；

2008年第八次经济危机

2000年，中国告别百年的短缺状态，进入生产过剩阶段。2001年，中国加入WTO。2002年，中国四大国有银行商业化改制完成，中国有了独立于政府财政的商业银行；

2008年，华尔街金融海啸、次贷危机爆发，中国遭遇“输入型通胀”，2009年，受全球经济危机影响，中国发生第八次经济危机，中国政府4万亿巨资救市。

‍

八次危机的成因

中国的前六次经济危机，是由于当时国家缺乏资本进行工业化的原始积累，源于内生性的资本短缺，被迫先后四次引进国外资本，然后造成财政赤字，最终只能转嫁到国内消化。 其结果就是，城市爆发经济危机，产生大量失业人口。然后国家再将城市经济危机转移给农村，城市经济实现“软着陆”，如果不能转移给农村，城市经济“硬着陆”。

站在这个角度看问题，上山下乡，以及历次政治运动，严打运动，都是经济危机造成的影响。

而中国的历次改革，也都与经济危机有关，又比如人民公社运动、大跃进、大炼钢铁，邓的三上三下，以及家庭联产承包、工业学大庆、教育医疗产业化……完整地解释了中国的历史事件、社会结构与经济环境的关系。

对外开放一方面作为中国政府在资本短缺时期作为推动经济发展和政策调整的重要工具，另一方面，又加剧国家对外负债导致数年经济困境，又导致领导人对出口创汇的重视和相关产业连续发展的惯性。

到2000年以后，中国不再贫穷，开始出现生产过剩，劳动力过剩，之后又出现了资本过剩。在不缺衣少食的环境里，我们又迎来输入型经济危机，1997年东南亚经济危机，以及2008年金融风暴，都波及中国经济。

伊曼纽尔·沃勒斯坦的世界系统理论：自人类迈入马克思所说的“资本主义文明阶段”，几乎所有的民族都渐次被核心国家整合到同一个世界经济体系中，形成“核心-半边缘-边缘”结构。通过不平等的交换，半边缘和边缘区域的剩余不均等地位往核心区域输送，这一过程中核心国家地位不断强化，边缘国家地位不断弱化。

在全世界政治家都不得不以经济增长来表现权力合法性的庸俗政治压力下，当代中国60年的客观过程，也几乎是一种不得不的选择；

中国发展的可持续性分析

三农对经济周期波动和经济社会系统运行压力的承载和调节意义重大，但，进入工业化中期阶段后，三农内部和外部宏观环境发生了根本性变化：

  1. 农村青壮年劳动力的流失，劳动资本收益被内外资和发达地区占有，制度代价甩给了中西部不发达地区；

    1. 剩余的老龄化和女性化人口极大弱化人口池和劳动力蓄水池的危机承载能力和社会条件功能，农村剩余劳动力因为“残值偏低”必然难以“被资本化”

  2. 维持农户理性——人力资源的自我资本化以保留收益用于维持风险内部化机制的主要条件正在衰减趋零；

  3. 中央在资本过剩的背景下有条件向“亲民生”转变，但是地方政府仍然面临短期资本压力，不得不在发展主义的导向下延续着“高负债高增长高推进”反科学发展模式；

  4. 小农家庭本质是追求短期现金收入的有产流动打工者，而不是经典意义的工人阶级。但，2003年新修订的《农村集体土地承包法》宣布，新增人口不再分地，农民无法无偿获得村内土地及无风险收益预期，“村社理性”被极大削弱，促使农民工行为取向发生实质性改变，生活压力超出务农收入，驱动农民工定居城市，演变为城市工人阶级，并开始形成对中国社会结构性矛盾和其变迁的主导地位；

作者建议 ：

  1. 转变执政思路，以中央政府掌握的垄断金融为基础，自上而下的组建能统筹县以下新农村建设和城镇化（双轮驱动）的基本建设投资机构，赋予其承担维护乡村稳定，创造就业和拉动内需的政策职能；
  2. 继续深化以“组织创新，制度创新”为内涵的农村综合体制改革。在构建国家新农村和城镇化基本建设投资机构的同时，把国家对三农的综合投资职能，主要对接到乡村综合性的合作组织载体上；

备注：

  1. 负外向性：也称外部成本或外部不经济，是指一个人的行为或企业的行为影响了其他人或企业，使之支付了额外的成本费用，但后者又无法获得相应补偿的现象。或是对交易双方之外的第三者所带来的未在价格中得以反映的成本费用。
  2. 比较优势：比较优势是指一个生产者以低于另一个生产者的机会成本生产一种物品的行为。如果一个国家在本国生产一种产品的机会成本（用其他产品来衡量）低于在其他国家生产该产品的机会成本的话，则这个国家在生产该种产品上就拥有比较优势。也可以说，当某一个生产者以比另一个生产者更低的机会成本来生产产品时，我们称这个生产者在这种产品和服务上具有比较优势。
  3. 三治：县治，乡治，村治；
  4. 工业化：按照产业资本的发展规律，大致划为三个阶段——原始积累，产业扩张，结构调整；
  5. 价格双规制（剪刀差）的思考：价值是怎么衡量，不取决于供给端，而取决于需求端和其中的交易成本，双轨制中，政府通过调节交易成本来调整价格，获得额外不平等收益；价值=需求+交易成本；
  6. 核心-半边缘-边缘：双输博弈，边缘国家和半边缘国家（新兴国家和发展中国家）以不成比例的方式向核心国家作双重输送，既输送实物产品，形成对西方的贸易顺差来推动其货币的增发，又反过来以不断累计的贸易盈余购买西方增发的国债，输送贸易盈余回到西方资本市场来促进金融资本的愈益泡沫化；
  7. 战后发展中国家的工业化路径：对内剥夺+对外负债；大多数发展中国家在工业化进程中陷入“外资依赖”，主权债务在追求发展的过程中对发达国家形成；
  8. 资本的思考：这里的资本不仅仅是美元之类的货币，应该指的是那个时候国内缺少的技术，设备等工业化基础设施；
  9. 传统的乡土社会是容易稳定的，而现代城市社会不容易稳定，经济危机总是率先爆发于资本积聚的城市；
  10. 中央资本过剩：1994年分税制改革，财权上移，事权下移，中央的财政盈余和地方财政赤字同步攀升；2002年完成银行商业化改革，中央政府进入以经营垄断性金融收益为主的“白领政府”阶段；
  11. 高负债高增长高推进：中央垄断控制的金融部分，在利益导向的市场化改革中异化于产业经济，也使地方政府失去对金融机构的控制权陷入普遍的严重资本短缺。这恰在本质上决定了欠发达地区政府的实质政策导向，只会是愈发单一的“亲资本”的发展主义，与中央倡导的“亲民生”政策相对立；
  12. 小农家庭：这个因农村土地集体所有，按家庭人口数量为基数承包到户特殊体制为无风险财产基础的农民流动打工者群体特征，是中国与其他人口大国在20年内维持低价劳动力的比较优势的真实内因；
  13. 合作组织载体的思考：应该指的是类似人民公社的集体组织；
  14. 村社理性：村社理性是乡土中国异于西方典型理论模式的一个核心机制。与农业生产和农村生活密切相关的村社理性，主要体现在单家独户之外、需集结众家之力才能办成的领域，如治安团防和水利灌溉。
