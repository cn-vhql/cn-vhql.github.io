工作，消费主义和新穷人

发表于 2022-03-13 更新于 2024-03-17

工作，消费主义和新穷人

作者：齐格蒙特·鲍曼

‍

工作的意义：创造工作伦理——生产者社会

工作伦理的起源：吸引穷人到工厂工作，消灭贫困，保证社会安宁；作用是训练和约束人们，向他们灌输新的工厂制度发挥作用所必须的服从性；

  1. 工作伦理本质是一条戒律，即时你看不到任何收益，也应该继续工作，工作是正义的，不工作是罪恶的；

    1. 两个前提：人们想维持生活并获得快乐，需要做一些别人认同的有价值的事情；安于现状且不思进取是可耻的；
    2. 两个假设：所有人都有能力工作，通过工作获得回报和维持生活；只有公认有价值的工作，可以要求薪水回报的，才会被工作伦理认同；

  2. 驱使人们去工作；

    1. 工作伦理改革运动是一场关于控制和服从的战争，是权力斗争，以崇高道德为名，迫使劳动者接受既不高尚，也不符合他们道德标准的生活；

      1. 通过训练出机械化的工人，成为复杂机器中如同没有灵魂的小齿轮一样工作的部分；工作伦理的本质是对自由的摈弃；
      2. 工作伦理把生产活动和人们的需求分离，人们优先考虑能做什么，而不是需要做什么；形成了新时代悖论，为增长而增长；
      3. 工厂体系的诞生瓦解了工匠与他们事业间的亲密关系，后面演变的绩效管理等方式只是试图让工人再度全身心进入工作；

    2. 历史不同流派的思想家，都提到人类的终极目标是征服自然；任何有利这个目标的，都成了合乎道德的，穷人对工厂化的抵制，都成了进步的阻碍，需要被消灭；

      1. 怜悯，同情和关怀被摒弃，它们会拖累人们变革的步伐；
      2. 工作伦理否定伦理改革运动目标群体的生活习惯，喜好和欲望，质疑他们的秉性，认为他们病态的，不受控制的冲动都是新兴工业必须抵御，对抗并消灭的一部分；

  3. 要么工作，要么死亡

    1. 把工作定义为是唯一体面的，道德的，可行的生存方式，让不适合工作的人生活的很痛苦，以减少不工作的人口；
    2. 工作的道德优越性，可以让政府毫无心理负担的向不工作的人以最小化原则援助；
    3. 救济院环境极差，令工厂工作与救济院相比也就显得不那么糟糕了，甚至显得像一种幸福，让工人缺少选择，只能工作；
    4. 引入“仅供生存”概念，把薪水保持在一个够低的水平，工人只能靠它活到第二天的黎明，这样除了继续辛苦工作，他们别无选择；

  4. 制造生产者，将工作穿插人们的一生；

    1. 给所有人工作，让所有人都去工作，通常被认为是治愈因社会暂时的不完美，不成熟导致的一切弊病和麻烦的良方；
    2. 工作类型决定了个人在生活，社会里的合理地位，决定了人们应该匹配的生活标准，标记了人生旅程，是回溯人生成败的重要记录；
    3. 工作成为了人们的主要基准，一旦确定了职业规划，其它方方面面都水到渠成，需要做什么也基本确定了下来；
    4. 工作使人们接受训练，培养服从规范，遵守纪律，形成自己的社会性格，成为顺从现代国家的公民，是现代社会的“圆形监狱”；
    5. 让大部分男性进入工厂工作是作为生成和维持社会秩序的主要手段，以男性为绝对权威建立父权家庭，使秩序生成和服务网络的规训压力传导到工厂无法触及的人群；
    6. 工作伦理要求人们选择一种献身于劳动的生活，但这也意味着没有选择，无法选择和禁止选择；

  5. 从“更好”到“更多”，工作伦理进化至消费主义

    1. 在美国，工人们认为工作是一种手段而不是价值，是获取更多财富，从而更加独立自由的手段，无需任何道德包装；
    2. 美国梦劝导人们，所有工厂遭受的苦难只是暂时的烦恼，屈从老板的虚妄只是成为老板的必经之路；
    3. 通过物质激励而不是道德宣教获得服从纪律的工人，告诉大家工作是赚取更多金钱的手段，人们认为，从工匠变成工人时失去人的尊严，需要通过赚取更多盈余才可以恢复，衡量人们声望和社会地位的是工资的差别，而不再是勤于道德或者惰于工作的罪恶；
    4. 社会生存质量的权力斗争转变为获取更多金钱的斗争，经济收益成为了自治自主的唯一体现，这也把人的动机和对自由的渴望牢牢的，不可逆转的转向消费领域；

‍

从工作伦理到消费美学——消费者社会

现代社会渐进到消费主义社会，大规模生产不再需要大规模的劳动力，曾经作为劳动者储备军的穷人被重铸为有缺陷的消费者，使他们失去了有用的社会功能；

  1. 技术进步在就业萎缩的前提下确保了生产力的增长，工厂员工越来越少，生产者社会进化至消费者社会；

    1. 工作伦理与消费者角色要求背道而驰，消费者应该不固守任何一样东西，没有永久的承诺，没有可以被完全满足的需求，也没有终极欲望；

    2. 理想情况下，消费者得到立刻满足，没有时延，不需要持久的准备和技能学习；如果消费者无法对任何目标保持长期关注和欲望，如果他们没有耐心，焦躁和冲动，尤其是易于激动，又同样容易失去兴趣，“即时满足”就达到了最佳效果，类似短半衰期收益的事；

      1. 如果欲望不用等待就可以充分满足，消费者的消费能力可能会远远超越所有先天，后天的需求和限制，超越消费品物理耐久性的限制，一直以来的消费与满足的关系或许会反转，对满足感的期待或期望会先于需求，并总是大于现存需求，但又不至于大到让人失去欲望，事实上，越是陌生的需求，越具有吸引力，能获取未知生活体验将会带来大量的乐趣；前所未有的新鲜的兴奋感，就是消费者游戏的秘密；
      2. 要想提高消费者的消费能力就不能让他们休息，他们需要不断接受新的诱惑，持续处于永不枯竭的兴奋之中，持续处于怀疑和不满之中，诱使他们转移注意力的诱饵需要肯定这种怀疑，同时提供一个宣泄的出口：“你以为这是全部？好戏”
      3. 在正常运转的消费社会中，消费者会主动寻求被诱惑，每一个新的诱惑都不尽相同，似乎比之前的更诱人，消费者生活在这个轮回中；
      4. 本来是市场选择了消费者，剥夺了他们不受诱惑的自由，但每次来市场，消费者都觉得自己掌控了一切。寻求自我认同，获取社会地位，以他人认为有意义的方式活着，这些都需要日复一日消费；

    3. 在工业阶段，每个人首先是生产者，然后才拥有其它身份；在消费时代，每个人首先是消费者，然后才拥有其它特别身份；

      1. 消费逻辑通过意象性的符码关系让人们进入一种他们欲望深处企盼的消费游戏，这种游戏通过一种“你追我赶”的竞争性购买，自发的生成一体化的无意识的“纪律”，由此，消费主义在阴暗处形成了自己的统治，进入消费社会；

  2. 制造消费者

    1. 不同政治派别都开始鼓吹通过消费主导经济复苏；消费取代了工作，把个人动机，社会整合和系统再生产链接在一起；
    2. 现代终身工作基本被灵活工作取代，寄希望通过工作构建终身身份对大部分人而言不可能；人们的身份变得短暂又灵活，在不同的身份中转换变化；
    3. 目前全球的趋势是通过大幅减少产品和服务的寿命，提供不稳定的工作，将经济导向短周期和不确定的生产；

  3. 由美学评判的工作

    1. 消费是彻头彻尾的一种个人的，孤独的活动；这种活动通过不断满足和唤醒，缓释和激发某种欲望实现；聚集只是凸显了消费行为的私密性，增强其乐趣；
    2. 选择的自由设定了消费者社会的阶层，设定了消费者社会成员绘制生活理想的框架，为他们定义了自我改善的方向，一个人选择的自由度越大，自由行使的权力越多，他在社会阶层中的地位就越高，获得的尊重和自尊就越多，距离美好生活的理想就越近；
    3. 这是信用卡社会，而不是存折社会，信贷的膨胀成了事情朝正确方向发展的可靠信号，受到欢迎；
    4. 消费愈发多样化，对消费者是一种享受，消费者以审美趣味为导向，美学把崇高的体验放在首位；
    5. 对于合格的消费者而言，世界是一个充满了可能性的巨型矩阵，包含着强烈的感受和更深刻的体验，评判世间万物的标准是激发人们的感受和体验的能力——唤起欲望的能力，这是消费过程中最愉悦的阶段，比欲望得到满足更令人陶醉；
    6. 工作也受到了美学的审视，工作的价值取决于产生愉悦体验的能力，不能使人获得内在满足的工作没有价值；

  4. 使命是一种特权

    1. 工作伦理传达出一种工作平等的信息，淡化工作间的差异；但引入美学审视后，工作被强调了区别，放大了差异，某些职业被提升到引人入胜程度，被赋予精致的美学内涵和艺术性体验，其它那些为生计获取报酬的工作则没有任何价值；
    2. 消费时代下，理想的工作必须是有趣的，多样化的，令人兴奋的，具有挑战，存在适度的风险，并不断带来崭新的体验；
    3. 工作的美学价值成为消费者社会有力的层级化因素，把工作本身变成最令人愉悦的体验，具有使命感，娱乐式的工作成为了一种最令人羡慕的特权；

  5. 消费社会的穷人

    1. 消费者社会中，正常生活是作为消费者的生活，他们专注从各种公开的机会中选择，以获取愉悦的感受和生动的体验；一个人如果无法正常生活，就意味着他是失败的消费者，或者是有缺陷的消费者；
    2. 在消费者社会中，作为消费者的不合格是被个人社会降级和内部流放的首要原因，正式这种不合格，无法履行消费者义务的无能，转化为痛苦，他们被抛弃在正常人共同享用的社会盛宴之外；
    3. 有一种境况对失业者特别痛苦，他们似乎永无止境的空闲时间，无法有效利用这些时间，无聊不知如何打发时间；消费主义不允许无聊存在，在消费社会语境下，幸福的生活是绝缘于无聊的，而金钱是进入治疗无聊场所的通行证；
    4. 无聊是消费者社会特有的社会分层因素产生的心理学结果，可能也是社会底层人民最痛苦的感受，也是他们最恼怒，最抗拒的感受；
    5. 消费社会通过大众传媒设定了得体的标准，使未达标的不合格的消费者自卑，制造人为的，主观的不满足感；
    6. 人们满足于自己拥有的东西被消费社会认为是最可怕的威胁，富人被人崇拜因为财富本身，而非工作伦理；富人拥有选择自己生活的能力，随心所欲的改变生活，对他们而言，未来总比过去要精彩，诱人；富人对消费美学的掌握，也让他们高人一等，令他们获得普遍的崇拜；

工作伦理和新穷人

技术进步取代工人，生产力的发展推动失业率的提升，消费社会使每个人都有消费欲望，但并不是所有人都有能力满足，无力满足之人成为新穷人；

  1. 现代无论穷人是否真的不愿意参加生产，都不会影响生产力的增长；人力成本的提升使得资本对失业人口数量的增长无动于衷，甚至还会感到振奋；

  2. 市场上的四类职业：

    1. 引导者：发明家，广告商，推广者和商人；
    2. 教育工作者：积极从事于可雇佣劳动力的再生产，把劳动力塑造成可供购买和消费的商品；
    3. 从事消费市场服务的人：工作目的是激活他人的消费能力；
    4. 常规劳动者：被配置于传统流水线；

    第四类劳动者（常规劳动者）在市场上缺少核心竞争力，很容易被弃用和替换；

  3. 在由工厂，办公室，工作间等构成的世界崇尚灵活性，工作伦理失去的市场，不需要那么多劳动力，在这样的世界存活需要寻找其它的意义，消费主义适时而出，并愿意提供这种意义；

  4. 全球化资本流动使得工人面对资本失去议价权，劳动者的生计取决于老板的仁慈；

  5. 从失业到过剩

    1. 失业者虽然暂时没有工作，但环境一旦好转，他们就有望回到生产者行列。一切也将回到正轨。过剩的人不同，他们是多余的，编外的，不被需要的，如果他们不存在，经济将会更好；
    2. 在如今精简型，小型化，资本和知识密集化产业，劳动力成为提升生产力的制约要素；
    3. 经济增长和就业率提升成了相互矛盾的，技术进步和利润增长是以劳动力的替代和淘汰为标准的，这种背景下，工作伦理显得愈发空洞；
    4. 工作伦理在新时代下的作用是调和社会众生和永恒存在的穷人之间的关系，将穷人贫困的原因归结于他们自身；

  6. 底层阶级的浮现

    1. 底层阶级被认为是那些被排除在阶级体系之外的人，他们没有任何机会，也无法被重新接纳，他们没有容身之所，对社会没有任何有益的贡献，原则上不值得救赎；他们的共同点是没有存在的必要，特征是危险和无用，如果他们消失，所有人会获益；
    2. 底层阶级也是经济逻辑的产物，因社会发展而失去了工作，他们也被认为是消费社会下错误选择者的集合，证明其成员的选择无能；
    3. 工作伦理在消费社会下另一作用，洗涤社会中正常人的双手和良知，帮他们在抛弃同胞致使其永久的失业的罪责中摆脱出来，在对穷人的道德进行谴责的同时，对非穷人的道德实施赦免，通过这种双管齐下的方法，达到双手和良心的洁净；

  7. 贫穷即犯罪

    1. 消费者社会的逻辑是把穷人塑造成未被满足的消费者，消费者的生活方式对于那些低收入者而言越来越触不可及，这也是最令人痛苦的剥夺；
    2. 一个社会如果对自己的生存方式缺乏自信，就会产生围城心态，塑造出一个攻墙的恶魔，一个有形的敌人，人们可以与之不断斗争并有望征服它；
    3. 在消费社会中，消费需求越大，这个社会就越安全，越繁荣；但是，不是所有人都有资源去消费，没有消费能力的人在这个社会中成为了废弃品，只要这个消费游戏不停止，参与者源源不断，废弃品也会源源不断的产生，他们有消费欲望却没有消费能力，当他们屈服于欲望行动时，他们被定义为犯罪；
    4. 在消费时代下，依赖救济的人是犯罪集团天然的温床，他们有着消费意愿却又无力满足；

  2. 美学：应该指愉悦的感受和体验；
